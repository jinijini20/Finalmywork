인증된 사용자의 도메인 디렉터리에 있는 도메인 프로필 및 도메인 연락처 목록을 제공합니다.
syncToken [https://developers.google.com/people/api/rest/v1/people/listDirectoryPeople?hl=ko#body.QUERY_PARAMETERS.sync_token]가 지정되면 마지막 동기화 이후 삭제된 리소스가 PersonMetadata.deleted [https://developers.google.com/people/api/rest/v1/people?hl=ko#Person.PersonMetadata.FIELDS.deleted]가 true로 설정된 사용자로 반환됩니다.
pageToken [https://developers.google.com/people/api/rest/v1/people/listDirectoryPeople?hl=ko#body.QUERY_PARAMETERS.page_token] 또는 syncToken [https://developers.google.com/people/api/rest/v1/people/listDirectoryPeople?hl=ko#body.QUERY_PARAMETERS.sync_token]가 지정되면 다른 모든 요청 매개변수가 첫 번째 호출과 일치해야 합니다.
동기화 요청의 경우 쓰기에 몇 분의 적용 지연이 있을 수 있습니다. 증분 동기화는 쓰기 후 읽기 사용 사례에 적합하지 않습니다.
사용 예시는 변경된 디렉터리 사용자 목록 [https://developers.google.com/people/v1/directory?hl=ko#list_the_directory_people_that_have_changed]을 참고하세요.

pageToken
string
선택사항입니다. 이전 응답 nextPageToken [https://developers.google.com/people/api/rest/v1/people/listDirectoryPeople?hl=ko#body.ListDirectoryPeopleResponse.FIELDS.next_page_token]에서 수신된 페이지 토큰입니다. 후속 페이지를 검색하려면 이를 입력합니다.
페이지로 나눌 때 people.listDirectoryPeople [https://developers.google.com/people/api/rest/v1/people/listDirectoryPeople?hl=ko#google.people.v1.PeopleService.ListDirectoryPeople]에 제공된 다른 모든 매개변수는 페이지 토큰을 제공한 첫 번째 호출과 일치해야 합니다.
requestSyncToken
boolean
선택사항입니다. 응답이 nextSyncToken [https://developers.google.com/people/api/rest/v1/people/listDirectoryPeople?hl=ko#body.ListDirectoryPeopleResponse.FIELDS.next_sync_token]를 반환해야 하는지 여부입니다. 요청 syncToken [https://developers.google.com/people/api/rest/v1/people/listDirectoryPeople?hl=ko#body.QUERY_PARAMETERS.sync_token]에서 설정하여 마지막 요청 이후의 증분 변경사항을 가져오는 데 사용할 수 있습니다.
동기화 동작에 관한 자세한 내용은 people.listDirectoryPeople [https://developers.google.com/people/api/rest/v1/people/listDirectoryPeople?hl=ko#google.people.v1.PeopleService.ListDirectoryPeople]를 참고하세요.
syncToken
string
선택사항입니다. 이전 응답 nextSyncToken [https://developers.google.com/people/api/rest/v1/people/listDirectoryPeople?hl=ko#body.ListDirectoryPeopleResponse.FIELDS.next_sync_token]에서 수신된 동기화 토큰입니다. 이 토큰을 제공하면 마지막 요청 이후 변경된 리소스만 가져올 수 있습니다.
동기화할 때 people.listDirectoryPeople [https://developers.google.com/people/api/rest/v1/people/listDirectoryPeople?hl=ko#google.people.v1.PeopleService.ListDirectoryPeople]에 제공된 다른 모든 매개변수는 동기화 토큰을 제공한 첫 번째 호출과 일치해야 합니다.
people.listDirectoryPeople [https://developers.google.com/people/api/rest/v1/people/listDirectoryPeople?hl=ko#google.people.v1.PeopleService.ListDirectoryPeople]에서 동기화 동작에 관해 자세히 알아보세요.
