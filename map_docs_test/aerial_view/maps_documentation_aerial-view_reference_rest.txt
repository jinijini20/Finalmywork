Aerial View API
Maps Aerial View API
서비스: aerialview.googleapis.com
이 서비스를 호출하려면 Google에서 제공하는 클라이언트 라이브러리 [https://cloud.google.com/apis/docs/client-libraries-explained?hl=ko]를 사용하는 것이 좋습니다. 애플리케이션에서 자체 라이브러리를 사용하여 이 서비스를 호출해야 한다면 API 요청을 실행할 때 다음 정보를 사용해야 합니다.
검색 문서
검색 문서 [https://developers.google.com/discovery/v1/reference/apis?hl=ko]는 머신에서 읽을 수 있는 사양이며 REST API를 설명하고 사용하는 데 참조할 수 있습니다. 또한 클라이언트 라이브러리, IDE 플러그인, 기타 Google API와 상호작용하는 도구를 빌드하는 데 사용됩니다. 하나의 서비스는 여러 개의 검색 문서를 제공할 수 있습니다. 이 서비스는 다음 탐색 문서를 제공합니다.
https://aerialview.googleapis.com/$discovery/rest?version=v1 [https://aerialview.googleapis.com/$discovery/rest?version=v1]
서비스 엔드포인트
서비스 엔드포인트 [https://cloud.google.com/apis/design/glossary?hl=ko#api_service_endpoint]는 API 서비스의 네트워크 주소를 지정하는 기준 URL입니다. 하나의 서비스는 여러 서비스 엔드포인트를 포함할 수 있습니다. 이 서비스에는 다음 서비스 엔드포인트가 포함되고, 그 아래의 모든 URI가 이 서비스 엔드포인트와 관련됩니다.
https://aerialview.googleapis.com
REST 리소스: v1.videos [https://developers.google.com/maps/documentation/aerial-view/reference/rest/v1/videos?hl=ko]
방법
lookupVideo [https://developers.google.com/maps/documentation/aerial-view/reference/rest/v1/videos/lookupVideo?hl=ko] GET /v1/videos:lookupVideo
주소 또는 videoId가 지정된 동영상을 가져옵니다.
lookupVideoMetadata [https://developers.google.com/maps/documentation/aerial-view/reference/rest/v1/videos/lookupVideoMetadata?hl=ko] GET /v1/videos:lookupVideoMetadata
주소 또는 videoId가 주어진 동영상의 메타데이터를 가져옵니다.
renderVideo [https://developers.google.com/maps/documentation/aerial-view/reference/rest/v1/videos/renderVideo?hl=ko] POST /v1/videos:renderVideo
동영상이 아직 렌더링되지 않은 경우 렌더기의 대기열에 주소를 추가합니다.
