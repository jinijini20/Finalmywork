지정된 위치의 현재 날씨를 반환합니다.

HTTP 요청
GET https://weather.googleapis.com/v1/currentConditions:lookupURL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 구문을 사용합니다.

쿼리 매개변수
| 매개변수 |
| --- |
| location | object (LatLng) 필수 항목입니다. 현재 날씨를 확인할 위치입니다. |
| unitsSystem | enum (UnitsSystem) 선택사항입니다. 반환된 날씨 조건에 사용할 단위 시스템입니다. 제공하지 않으면 반환된 날씨 상태가 미터법 (기본값 = METRIC)으로 표시됩니다. |
| languageCode | string 선택사항입니다. 클라이언트가 응답의 언어를 선택할 수 있습니다. 해당 언어에 데이터를 제공할 수 없는 경우 API는 가장 근접한 언어를 사용합니다. 허용되는 값은 IETF BCP-47 표준을 따릅니다. 기본값은 'en'입니다. |

요청 본문
요청 본문은 비어 있어야 합니다.

응답 본문

currentConditions.lookup RPC의 응답 - 요청된 위치의 현재 날씨를 나타냅니다.
성공한 경우 응답 본문은 다음과 같은 구조의 데이터를 포함합니다.

| JSON 표현 |
| --- |
| {   "currentTime": string,   "timeZone": {     object (TimeZone)   },   "weatherCondition": {     object (WeatherCondition)   },   "temperature": {     object (Temperature)   },   "feelsLikeTemperature": {     object (Temperature)   },   "dewPoint": {     object (Temperature)   },   "heatIndex": {     object (Temperature)   },   "windChill": {     object (Temperature)   },   "precipitation": {     object (Precipitation)   },   "airPressure": {     object (AirPressure)   },   "wind": {     object (Wind)   },   "visibility": {     object (Visibility)   },   "currentConditionsHistory": {     object (CurrentConditionsHistory)   },   "isDaytime": boolean,   "relativeHumidity": integer,   "uvIndex": integer,   "thunderstormProbability": integer,   "cloudCover": integer } |

| 필드 |
| --- |
| currentTime | string (Timestamp format) 반환된 데이터와 연결된 현재 시간 (UTC)입니다. 생성된 출력은 항상 Z-정규화되고 소수점 이하 자릿수가 0, 3, 6 또는 9인 RFC 3339를 사용합니다. 'Z' 이외의 오프셋도 허용됩니다. 예를 들면 "2014-10-02T15:01:23Z", "2014-10-02T15:01:23.045123456Z" 또는 "2014-10-02T15:01:23+05:30"입니다. |
| timeZone | object (TimeZone) 요청된 위치의 시간대입니다. |
| weatherCondition | object (WeatherCondition) 현재 날씨 상태입니다. |
| temperature | object (Temperature) 현재 온도 |
| feelsLikeTemperature | object (Temperature) 요청된 위치의 현재 체감 온도입니다. |
| dewPoint | object (Temperature) 현재 이슬점 온도입니다. |
| heatIndex | object (Temperature) 현재 체감 온도입니다. |
| windChill | object (Temperature) 피부에 노출된 현재의 체감온도입니다. |
| precipitation | object (Precipitation) 현재 강수 확률과 지난 1시간 동안 누적된 강수량입니다. |
| airPressure | object (AirPressure) 현재 기압 상태입니다. |
| wind | object (Wind) 현재 바람 상태입니다. |
| visibility | object (Visibility) 현재 공개 상태입니다. |
| currentConditionsHistory | object (CurrentConditionsHistory) 지난 24시간 동안의 현재 상태 변화입니다. |
| isDaytime | boolean 요청된 위치의 현재 시간이 현지 일출 시간 (양 끝값 포함)과 일몰 시간 (양 끝값 제외) 사이에 있는 경우 true입니다. 그렇지 않으면 야간 (일몰과 다음 일출 사이)입니다. |
| relativeHumidity | integer 현재 상대 습도 (0~100 사이의 값)입니다. |
| uvIndex | integer 현재 자외선 (UV) 지수입니다. |
| thunderstormProbability | integer 현재 뇌우 발생 확률 (0~100 사이의 값)입니다. |
| cloudCover | integer 현재 하늘에서 구름이 차지하는 비율입니다 (0~100). |

CurrentConditionsHistory

지난 24시간 동안의 현재 상태의 일련의 변경사항을 나타냅니다.

| JSON 표현 |
| --- |
| {   "temperatureChange": {     object (Temperature)   },   "maxTemperature": {     object (Temperature)   },   "minTemperature": {     object (Temperature)   },   "qpf": {     object (QuantitativePrecipitationForecast)   } } |

| 필드 |
| --- |
| temperatureChange | object (Temperature) 현재 온도에서 24시간 전의 온도를 뺀 값입니다. |
| maxTemperature | object (Temperature) 지난 24시간 동안의 최고 (높은) 기온입니다. |
| minTemperature | object (Temperature) 지난 24시간 동안의 최저 (낮은) 온도입니다. |
| qpf | object (QuantitativePrecipitationForecast) 지난 24시간 동안 누적된 강수량 (비 또는 눈)으로, 액체 물 등가량으로 측정됩니다. 참고: QPF는 양적 강수 예보의 약어입니다 (자세한 내용은 QuantitativePrecipitationForecast 정의를 참고하세요). |
