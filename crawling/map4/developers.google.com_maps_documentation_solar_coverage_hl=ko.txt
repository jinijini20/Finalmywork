Solar API 적용 범위

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

Solar API는 전 세계 수억 개의 건물에 대한 태양광 데이터를 제공합니다. 이미지 품질 [https://developers.google.com/maps/documentation/solar/reference/rest/v1/dataLayers?hl=ko#imageryquality]은 HIGH, MEDIUM 또는 LOW일 수 있습니다.

HIGH: 태양 데이터는 고해상도 (예: 10cm) DSM 데이터(일반적으로 저고도 항공 이미지에서 가져옴)
MEDIUM: 태양 데이터는 중간 해상도 (예: 25cm) DSM 데이터(일반적으로 고고도 항공 이미지에서 가져옴)
LOW: 태양 데이터가 저해상도 (예: 30cm 이하) DSM 데이터(일반적으로 위성 이미지에서 가져옴)를 사용합니다.

Google Maps Platform팀에서는 Google API 서비스의 서비스 지역을 확대하기 위해 지속적으로 노력하고 있습니다. 이 대화형 지도는 HIGH 및 MEDIUM 품질의 태양광 데이터를 사용할 수 있는 위치를 보여줍니다. LOW 품질의 태양광 데이터를 사용할 수 있는 위치는 표시되지 않습니다.

신뢰할 수 있는 테스터는 Solar API에서 추가 지역의 실험용 데이터를 사용할 수 있습니다. 데이터를 미리 보고 의견을 제공하려면 신뢰할 수 있는 테스터 액세스 권한을 요청 [https://forms.gle/a7QqYfDTsQc38Tm3A]하세요.

위도 및 경도 좌표는 Geocoding API [https://developers.google.com/maps/documentation/geocoding?hl=ko] 또는 Places API [https://developers.google.com/maps/documentation/places/web-service?hl=ko]와 같은 다른 Google Maps Platform API를 사용하여 가져올 수 있습니다. Maps JavaScript API [https://developers.google.com/maps/documentation/javascript?hl=ko]를 사용하여 만든 지도에서 좌표를 추출할 수도 있습니다. 예를 보려면 클릭 이벤트에서 위도/경도 가져오기 [https://developers.google.com/maps/documentation/javascript/examples/event-click-latlng?hl=ko]를 참고하세요.

지도를 드래그하여 특정 지역을 표시한 다음 확대하여 세부적으로 확인합니다.
참고: 여기에 표시된 데이터는 정확한 노출 범위의 근사치입니다. 특정 지역 (예: 국가 국경 근처)에서는 지도에 해당 지역이 표시되는데도 404 오류가 발생할 수 있습니다. 노출 범위 업데이트에 관한 자세한 내용은 Solar API 출시 노트 [https://developers.google.com/maps/documentation/solar/release-notes?hl=ko]를 참고하세요.

이 지도를 만드는 데 사용된 셰이프 파일을 ZIP 파일로 다운로드 [https://developers.google.com/static/maps/documentation/solar/images/solar-api-coverage.zip?hl=ko]합니다.
