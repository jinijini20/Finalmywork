Air Quality API

서비스: airquality.googleapis.com
RPC 클라이언트 스터브를 만들려면 서비스 이름 airquality.googleapis.com이 필요합니다.

google.maps.airquality.v1.AirQuality [https://developers.google.com/maps/documentation/air-quality/reference/rpc/google.maps.airquality.v1?hl=ko#google.maps.airquality.v1.AirQuality]

| 메서드 |
| --- |
| LookupCurrentConditions | 현재 조건 엔드포인트는 100개가 넘는 국가에서 최대 500x500m 해상도로 시간당 공기질 정보를 제공합니다. |
| LookupForecast | 지정된 기간 동안 특정 위치의 공기질 예보를 반환합니다. |
| LookupHeatmapTile | 타일 PNG 이미지의 데이터를 포함하는 바이트 배열을 반환합니다. |
| LookupHistory | 지정된 기간 동안 특정 위치의 공기질 기록을 반환합니다. |
