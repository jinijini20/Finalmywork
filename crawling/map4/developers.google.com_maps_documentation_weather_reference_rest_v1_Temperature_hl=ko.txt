| JSON 표현 |
| --- |
| {   "unit": enum (TemperatureUnit),   "degrees": number } |

| 필드 |
| --- |
| unit | enum (TemperatureUnit) 온도 값을 측정하는 데 사용되는 단위의 코드입니다. |
| degrees | number 지정된 단위의 온도 값 (도)입니다. |

TemperatureUnit

온도를 측정하는 데 사용되는 단위를 나타냅니다.

| 열거형 |
| --- |
| TEMPERATURE_UNIT_UNSPECIFIED | 온도 단위가 지정되지 않았습니다. |
| CELSIUS | 온도는 섭씨로 측정됩니다. |
| FAHRENHEIT | 온도는 화씨로 측정됩니다. |
