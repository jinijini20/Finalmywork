weather.googleapis.com API

서비스: weather.googleapis.com
RPC 클라이언트 스터브를 만들려면 서비스 이름 weather.googleapis.com이 필요합니다.

google.maps.weather.v1.Weather [https://developers.google.com/maps/documentation/weather/reference/rpc/google.maps.weather.v1?hl=ko#google.maps.weather.v1.Weather]

| 메서드 |
| --- |
| LookupCurrentConditions | 지정된 위치의 현재 날씨를 반환합니다. |
| LookupForecastDays | 현재 날짜부터 지정된 위치의 일일 예측을 최대 10일 동안 반환합니다. |
| LookupForecastHours | 현재 시간부터 시작하여 지정된 위치의 시간별 예측을 최대 240시간까지 반환합니다. |
| LookupHistoryHours | 지정된 위치의 지난 1시간부터 최대 24시간 동안의 시간별 과거 날씨 데이터를 반환합니다. |
