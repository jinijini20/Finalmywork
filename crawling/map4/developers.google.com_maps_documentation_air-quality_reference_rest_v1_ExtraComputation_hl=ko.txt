ExtraComputation

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

선택적으로 사용 설정할 수 있는 추가 기능입니다. 추가 계산을 지정하면 관련 요소 및 필드가 응답에 반환됩니다.

| 열거형 |
| --- |
| EXTRA_COMPUTATION_UNSPECIFIED | 기본값입니다. 매개변수로 전달되면 서버에서 이를 무시합니다. |
| LOCAL_AQI | 응답에 요청된 위치 (국가)의 현지 (국가) AQI를 포함할지 여부를 결정합니다. 지정된 경우 응답에는 위치의 로컬 AQI에 있는 모든 관련 데이터가 있는 'air_quality_index' 데이터 구조가 포함됩니다. |
| HEALTH_RECOMMENDATIONS | 응답에 현재 AQI 상태에 대한 건강 관련 조언과 권장 조치를 포함할지 결정합니다. 이 권장사항은 일반 대중보다 오염 민감도가 높은 위험 집단의 6개 집단에 대한 맞춤형 모집단입니다. 지정된 경우 관련 데이터를 사용할 수 있을 때 응답에 healthRecommendations 필드가 채워집니다. |
| POLLUTANT_ADDITIONAL_INFO | 응답에 각 오염 물질의 추가 정보를 포함할지 여부를 결정합니다. 이 플래그를 지정하면 데이터를 사용할 수 있을 때 '색인' 필드 응답에 포함된 각 공기질 지수 객체에 additionalInfo 필드가 포함됩니다. |
| DOMINANT_POLLUTANT_CONCENTRATION | 글로벌 및/또는 현지 지표에 따라 측정된 주요 오염 물질의 농도를 응답에 포함할지 여부를 결정합니다. 요청이 전역 AQI와 로컬 AQI를 모두 지정한 경우 최대 2개의 오염 코드가 반환될 수 있습니다. 지정하면 '오염 물질' 목록에 포함된 주요 오염 물질 객체에 데이터가 있을 때 concentration 필드가 포함됩니다. |
| POLLUTANT_CONCENTRATION | 글로벌 및/또는 현지 지표에 따라 사용 가능한 측정값으로 모든 오염 물질의 농도를 응답에 포함할지 결정합니다. 지정하면 데이터의 '오염 물질' 필드에 포함된 각 오염 물질은 데이터가 있을 때 concentration 필드를 포함합니다. |
