2023년 7월, Google에서는 데이터 영역 요청 [https://developers.google.com/maps/documentation/solar/requests?hl=ko#make_a_data_layers_request]에 대한 호출로 반환되는 URL 형식에 관한 변경사항을 출시할 예정입니다. 이 버전을 준비하려면 두 URL 형식을 모두 지원하도록 앱을 작성하세요. 지금 새 URL 형식을 준비해도 앱은 이 업데이트의 영향을 받지 않습니다.

2023-05-09
미리보기: Google Cloud에서 Google Maps Platform의 일부로 Solar API의 제한된 미리보기가 출시되었습니다.

Google Cloud에서 Google Maps Platform의 일부로 호스팅되는 새로운 Solar API의 제한된 미리보기가 출시되었습니다. 이 새로운 API는 현재 지원 중단된 EarthEngineSolar API를 대체하며, 서비스 중단을 최소화하기 위해 가능한 한 빨리 이전하는 것이 좋습니다.

프로젝트 수준 액세스 [https://developers.google.com/maps/documentation/solar/quickstart?hl=ko] 개선

액세스하려면 Solar API팀의 담당자에게 문의하세요.
액세스 권한은 프로젝트 단위로 수동으로 부여되며 EarthEngineSolar API에 대한 액세스 권한과 무관합니다.

새로운 문서 [https://developers.google.com/maps/documentation/solar?hl=ko]가 추가되었습니다.
새 서비스 약관 [https://developers.google.com/maps/documentation/solar/terms?hl=ko] 설정
Cloud Monitoring [https://cloud.google.com/monitoring?hl=ko] 및 Cloud Billing [https://cloud.google.com/billing?hl=ko]과 통합
Solar API 엔드포인트에 대한 호출이 프로젝트의 결제 대시보드에 표시되기 시작합니다 (미리보기 기간 동안 100% 할인).

엔드포인트 및 메서드 [https://developers.google.com/maps/documentation/solar/reference?hl=ko] 이름 변경
earthenginesolar.googleapis.com -> solar.googleapis.com
solarInfo:get -> dataLayers:get
buildings:findClosest -> buildingInsights:findClosest

새 할당량 한도 [https://developers.google.com/maps/documentation/solar/limits?hl=ko] 구성

일일 및 분기별 요청 할당량이 분당 할당량으로 대체되었습니다.

할당량 한도는 프로젝트 수준에서 설정되며 각 엔드포인트에서 분당 쿼리 60개로 시작합니다.
https://console.cloud.google.com/google/maps-apis/quotas [https://console.cloud.google.com/google/maps-apis/quotas?hl=ko]에서 프로젝트 할당량을 찾으세요.

기존 EarthEngineSolar API를 지원 중단했습니다.

지원 중단된 EarthEngineSolar API는 새 Solar API가 미리보기 상태인 동안에도 계속 작동합니다.

중간 품질 (imagery_quality: 'MEDIUM') 데이터 범위 확장

독일, 영국, 프랑스, 스페인, 이탈리아, 플로리다에서 중급 데이터 품질 지원 확대
플로리다주에서 훨씬 더 많은 지역에 중간 품질 수준의 재무 데이터 가용성이 확대되었습니다 (BuildingInsights 사용).

2020-09-08

분류되지 않은 지붕 크기에 대한 정보를 추가했습니다.
패널별 프로덕션 값을 추가했습니다.
