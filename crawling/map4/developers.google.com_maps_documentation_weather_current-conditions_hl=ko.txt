내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

currentConditions [https://developers.google.com/maps/documentation/weather/reference/rest/v1/currentConditions/lookup?hl=ko] 엔드포인트는 특정 위치의 현재 날씨에 관한 정보를 제공합니다. API는 다음을 반환합니다.

해당 아이콘 [https://developers.google.com/maps/documentation/weather/weather-condition-icons?hl=ko]과 함께 현재 날씨 조건 [https://developers.google.com/maps/documentation/weather/reference/rest/v1/WeatherCondition?hl=ko#type]을 설명합니다.
현재 기온
체감 온도
이슬점
열 지수
체감 온도
상대 습도
자외선 지수
강수 확률, 백분율, 유형 [https://developers.google.com/maps/documentation/weather/reference/rest/v1/Precipitation?hl=ko#precipitationtype]
뇌우 확률
해면 기압
바람
방향 [https://developers.google.com/maps/documentation/weather/reference/rest/v1/Wind?hl=ko#cardinaldirection], 속도, 돌풍
가시성 및 구름
온도 변화, 최고 및 최저 온도, 강수량 등 지난 24시간의 기록 데이터

API 탐색기를 사용하면 API와 API 옵션을 숙지할 수 있도록 라이브 요청을 할 수 있습니다.

사용해 보기 [https://developers.google.com/maps/documentation/weather/current-conditions?hl=ko#try_it]

현재 조건 요청 정보

현재 상태를 요청하려면 다음 주소로 HTTP GET 요청을 보냅니다.

https://weather.googleapis.com/v1/currentConditions:lookup?key=YOUR_API_KEY&location.latitude=LATITUDE&location.longitude=LONGITUDE

요청 URL 매개변수 [https://developers.google.com/maps/documentation/weather/reference/rest/v1/currentConditions/lookup?hl=ko#query-parameters]에 위치의 위도 및 경도 좌표를 포함합니다.

현재 상태 요청 예

다음 예에서는 캘리포니아주 마운틴뷰의 현재 상태 정보를 요청합니다.

curl -X GET "https://weather.googleapis.com/v1/currentConditions:lookup?key=YOUR_API_KEY&location.latitude=37.4220&location.longitude=-122.0841"

응답은 다음 형식입니다.

"currentTime": "2025-01-28T22:04:12.025273178Z",
"timeZone": {
"id": "America/Los_Angeles"

페이지 오른쪽에서 API 아이콘 api을 선택합니다.
원하는 경우 요청 매개변수를 수정합니다.
실행 버튼을 선택합니다. 대화상자에서 요청을 보낼 때 사용할 계정을 선택합니다.
API 탐색기 패널에서 전체 화면 아이콘 전체 화면을 선택하여 API 탐색기 창을 펼칩니다.
