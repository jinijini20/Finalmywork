특정 기간 동안 특정 위치의 날씨 상태를 나타냅니다.면책 조항: 날씨 아이콘 및 상태 코드는 변경될 수 있습니다. Google은 필요에 따라 새로운 코드와 아이콘을 도입하거나 기존 코드와 아이콘을 업데이트할 수 있습니다. 최신 정보는 이 문서를 정기적으로 참고하시기 바랍니다.

| JSON 표현 |
| --- |
| {   "iconBaseUri": string,   "description": {     object (LocalizedText)   },   "type": enum (Type) } |

| 필드 |
| --- |
| iconBaseUri | string 파일 형식 확장자를 포함하지 않는 아이콘의 기본 URI입니다. 아이콘을 표시하려면 원하는 경우 테마와 파일 형식 확장자 (.png 또는 .svg)를 이 URI에 추가합니다. 기본적으로 아이콘은 밝은 테마이지만 어두운 모드의 경우 _dark를 추가할 수 있습니다. 예: iconBaseUri이 'https://maps.gstatic.com/weather/v1/dust'인 경우 'https://maps.gstatic.com/weather/v1/dust.svg' 또는 'https://maps.gstatic.com/weather/v1/dust_dark.svg' |
| description | object (LocalizedText) 이 날씨 상태에 대한 텍스트 설명 (현지화됨)입니다. |
| type | enum (Type) 날씨 유형입니다. |

LocalizedText

특정 언어로 현지화된 텍스트의 변형입니다.

| JSON 표현 |
| --- |
| {   "text": string,   "languageCode": string } |

| 필드 |
| --- |
| text | string 아래 languageCode에 해당하는 언어로 현지화된 문자열입니다. |
| languageCode | string 텍스트의 BCP-47 언어 코드입니다(예: 'en-US' 또는 'sr-Latn'). 자세한 내용은 http://www.unicode.org/reports/tr35/#Unicode_locale_identifier를 참조하세요. |

유형

일기예보 요소의 컨텍스트에서 날씨 상태 유형을 표시합니다.

| 열거형 |
| --- |
| TYPE_UNSPECIFIED | 날씨 상태가 지정되지 않았습니다. |
| CLEAR | 구름이 없습니다. |
| MOSTLY_CLEAR | 가끔 구름이 끼겠습니다. |
| PARTLY_CLOUDY | 부분적으로 흐림 (구름 일부) |
| MOSTLY_CLOUDY | 대체로 흐림 (구름이 태양보다 많음). |
| CLOUDY | 흐림 (구름만 있고 태양이 없음) |
| WINDY | 강풍 |
| WIND_AND_RAIN | 강한 바람과 강우 |
| LIGHT_RAIN_SHOWERS | 간헐적으로 약한 비가 내리고 있습니다. |
| CHANCE_OF_SHOWERS | 간헐적으로 비가 올 수 있습니다. |
| SCATTERED_SHOWERS | 간헐적인 비가 내리고 있습니다. |
| RAIN_SHOWERS | 소나기는 비보다 지속 시간이 짧은 강우로 간주되며 시작과 종료 시간이 갑작스럽고 강도가 급격하게 변하는 것이 특징입니다. |
| HEAVY_RAIN_SHOWERS | 강한 소나기 |
| LIGHT_TO_MODERATE_RAIN | 비 (약하거나 중간 정도의 양) |
| MODERATE_TO_HEAVY_RAIN | 비 (중간에서 심한 정도) |
| RAIN | 중간 정도의 비가 내립니다. |
| LIGHT_RAIN | 약한 비가 내리고 있습니다. |
| HEAVY_RAIN | 폭우 |
| RAIN_PERIODICALLY_HEAVY | 비가 내리고 가끔 폭우가 예상됩니다. |
| LIGHT_SNOW_SHOWERS | 짧은 시간 동안 다양한 강도로 내리는 가벼운 눈입니다. |
| CHANCE_OF_SNOW_SHOWERS | 소낙눈이 올 가능성이 있습니다. |
| SCATTERED_SNOW_SHOWERS | 눈이 잠시 동안 다양한 강도로 내리고 있습니다. |
| SNOW_SHOWERS | 소낙눈 |
| HEAVY_SNOW_SHOWERS | 소낙눈이 세차게 내리고 있습니다. |
| LIGHT_TO_MODERATE_SNOW | 약하거나 중간 정도의 눈이 내립니다. |
| MODERATE_TO_HEAVY_SNOW | 중간에서 강한 눈 |
| SNOW | 중간 정도의 눈 |
| LIGHT_SNOW | 약한 눈 |
| HEAVY_SNOW | 폭설 |
| SNOWSTORM | 천둥과 번개가 동반된 눈이 내리고 있습니다. |
| SNOW_PERIODICALLY_HEAVY | 눈이 내리고 때로는 폭설이 예상됩니다. |
| HEAVY_SNOW_STORM | 천둥과 번개가 동반된 폭설이 내릴 수 있습니다. |
| BLOWING_SNOW | 눈이 내리고 바람이 강하게 불고 있습니다. |
| RAIN_AND_SNOW | 비와 눈이 섞여 내립니다. |
| HAIL | 우박. |
| HAIL_SHOWERS | 짧은 시간 동안 다양한 강도로 내리는 우박 |
| THUNDERSTORM | 뇌우 |
| THUNDERSHOWER | 천둥과 번개를 동반한 소나기 |
| LIGHT_THUNDERSTORM_RAIN | 약한 뇌우와 비 |
| SCATTERED_THUNDERSTORMS | 잠시 동안 다양한 강도의 비가 내리는 뇌우 |
