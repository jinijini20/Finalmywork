iOS용 Navigation SDK 코드 샘플
iOS용 Navigation SDK GitHub 샘플 저장소 [https://github.com/googlemaps-samples/maps-sdk-for-ios-samples]에는 SDK 사용 방법을 보여주는 샘플 앱이 포함되어 있습니다. 앱을 가져와 빌드하고, API 키를 추가하고, 데모를 보고, 앱의 시작점으로 제공된 샘플 코드를 사용할 수 있습니다. 저장소에는 샘플 앱의 두 가지 버전이 포함되어 있습니다.
Swift: googlemaps-samples/maps-sdk-for-ios-samples
Objective-C: googlemaps-samples/maps-sdk-for-ios-samples
샘플 앱 실행에 대한 자세한 내용은 데모 사용해 보기 [https://developers.google.com/maps/documentation/navigation/ios-sdk/try-demo?hl=ko]를 참고하세요.
iOS용 Navigation SDK Codelab
간단한 iOS 앱을 빌드하여 Navigation SDK를 사용하는 방법을 알아보려면 iOS용 Navigation SDK Codelab [https://codelabs.developers.google.com/codelabs/maps-platform/navigation-sdk-101-ios?hl=ko]을 참고하세요. 이 Codelab에서는 Swift를 사용하여 SDK를 구현하는 단계별 가이드를 제공합니다.
