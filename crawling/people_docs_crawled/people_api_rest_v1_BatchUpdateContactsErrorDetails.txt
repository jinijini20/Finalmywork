BatchUpdateContactsErrorDetails
bookmark_border
people.batchUpdateContacts 작업에서 요청이 실패할 때 오류 세부정보를 기록하는 메시지입니다.
JSON 표현
{
  "contactErrors": {
    string: {
      object (Status [https://developers.google.com/people/api/rest/v1/Status?hl=ko])
    },
    ...
  }
}
필드
contactErrors
map (key: string, value: object (Status [https://developers.google.com/people/api/rest/v1/Status?hl=ko]))
일괄 업데이트 실패의 원인이 된 연락처를 클라이언트가 식별할 수 있도록 세부 정보를 제공합니다. 연락처 ID가 표시되지 않으면 연락처가 성공적으로 업데이트된 것입니다.
아래 지도에서:
키는 BatchUpdateContactsRequest.contacts [https://developers.google.com/people/api/rest/v1/people/batchUpdateContacts?hl=ko#body.request_body.FIELDS.contacts]의 리소스 이름 / 연락처 ID입니다.
값은 상태입니다. google.rpc.Status.details [https://developers.google.com/people/api/rest/v1/Status?hl=ko#FIELDS.details]에는 연락처의 오류 목록이 포함됩니다.
"key": value 쌍 목록을 포함하는 객체입니다. 예: { "name": "wrench", "mass": "1.3kg", "count": "3" }
