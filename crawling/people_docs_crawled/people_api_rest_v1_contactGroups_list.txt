인증된 사용자가 소유한 모든 연락처 그룹을 나열합니다. 연락처 그룹의 구성원은 채워지지 않습니다.
HTTP 요청
GET https://people.googleapis.com/v1/contactGroups
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 구문을 사용합니다.
쿼리 매개변수
매개변수
pageToken
string
선택사항입니다. 이전 contactGroups.list [https://developers.google.com/people/api/rest/v1/contactgroups/list?hl=ko] 호출에서 반환된 nextPageToken 값입니다. 리소스의 다음 페이지를 요청합니다.
pageSize
integer
선택사항입니다. 반환할 최대 리소스 수입니다. 유효한 값은 1 이상 1000 이하입니다. 설정하지 않을 경우 기본값은 30이며, 0으로 설정됩니다.
syncToken
string
선택사항입니다. 이전 contactgroups.list 호출에서 반환된 동기화 토큰입니다. 동기화 토큰이 생성된 이후에 변경된 리소스만 반환됩니다.
groupFields
string (FieldMask [https://developers.google.com/protocol-buffers/docs/reference/google.protobuf?hl=ko#google.protobuf.FieldMask] format)
선택사항입니다. 반환되는 그룹의 필드를 제한하는 필드 마스크입니다. 설정하지 않거나 비워두면 기본값은 metadata, groupType, memberCount, name입니다. 유효한 필드는 다음과 같습니다.
clientData
groupType
