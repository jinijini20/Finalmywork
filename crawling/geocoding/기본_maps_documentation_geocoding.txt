Maps JavaScript API의 지오코딩
Maps JavaScript API 지오코더 서비스를 사용하여 웹페이지에 지오코딩 기능을 추가합니다.
기능
Maps Geocoding API의 핵심 기능에 대해 알아봅니다.
좌표 지오코딩
주소의 지리적 좌표를 가져옵니다.
역 주소 검색
위도/경도 좌표 집합의 주소를 가져옵니다.
장소 ID 지오코딩
장소 ID의 주소를 가져옵니다.
표시 영역 상세 검색
지정된 표시 영역 내의 결과를 선호하도록 지오코딩 요청을 제한합니다.
지역 상세 검색
특정 지역 내에서 결과를 반환하도록 지오코딩 요청을 제한합니다.
구성요소 필터링
지오코딩 결과를 특정 국가 또는 우편번호로 제한합니다.
예시 앱
Maps JavaScript API를 사용하여 로컬 컴퓨터와 즐겨 사용하는 코드 플레이그라운드에서 라이브 코드 샘플을 실행합니다.
travel_explore

역 지오코딩
