개요
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/advanced-markers/overview?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/advanced-markers/overview?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/advanced-markers/overview?hl=ko]
고급 마커를 사용하면 성능이 뛰어난 마커를 만들고 맞춤설정할 수 있습니다. 기본 마커의 배경, 테두리와 글리프 색상을 맞춤설정하고 기본 마커 아이콘을 맞춤 그래픽 이미지로 대체할 수 있습니다.
고급 마커는 AdvancedMarker [https://developers.google.com/android/reference/com/google/android/gms/maps/model/AdvancedMarker?hl=ko]유형의 객체이며, 이는 Marker [https://developers.google.com/android/reference/com/google/android/gms/maps/model/Marker?hl=ko]의 서브클래스입니다. 고급 마커를 사용하면 다음을 비롯한 기존 Marker 속성을 모두 맞춤설정할 수 있습니다.
제목 [https://developers.google.com/maps/documentation/android-sdk/marker?hl=ko#customize_a_marker] 및 스니펫 [https://developers.google.com/maps/documentation/android-sdk/marker?hl=ko#customize_a_marker] 필드
클릭 이벤트 [https://developers.google.com/maps/documentation/android-sdk/marker?hl=ko#marker_click_events]
드래그 이벤트 [https://developers.google.com/maps/documentation/android-sdk/marker?hl=ko#marker_drag_events]
고급 마커에는 다음과 같은 기능이 추가되었습니다.
색상 핀 맞춤설정
Android 뷰 [https://developer.android.com/reference/android/view/View?hl=ko]를 마커로 설정
충돌 동작 제어
마커 속성 맞춤설정
기본 마커의 배경, 글리프, 테두리 색상을 맞춤설정합니다.
Android 뷰를 마커로 사용
기본 마커 아이콘을 완전히 맞춤설정된 마커가 포함된 Android View [https://developer.android.com/reference/android/view/View?hl=ko]로 대체합니다.
마커 충돌 동작 설정
마커가 다른 마커 또는 지도 라벨과 충돌할 때의 동작을 지정합니다. 마커 고도는 벡터 지도에서만 설정할 수 있습니다.
