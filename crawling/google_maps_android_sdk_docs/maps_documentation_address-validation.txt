Address Validation API
주소와 해당 구성요소를 확인하고 우편 주소를 표준화한 다음 가장 잘 알려진 지오코드를 결정합니다.
탐색
Maps Address Validation API의 작동 방식을 알아보세요.
terminal [https://developers.google.com/maps/documentation/address-validation/demo?hl=ko]
데모 보기 [https://developers.google.com/maps/documentation/address-validation/demo?hl=ko]
Address Validation API 데모 사용해 보기 지원되는 지역의 주소를 사용하여 응답을 확인합니다.
travel_explore [https://developers.google.com/maps/documentation/address-validation/coverage?hl=ko]
지원되는 리전 보기 [https://developers.google.com/maps/documentation/address-validation/coverage?hl=ko]
Address Validation API가 해당 지역 또는 국가에서 지원되는지 알아보려면 최신 사용 범위 세부정보를 확인하세요.
시작하기
Maps Address Validation API를 이용해 지도 작성을 시작합니다.
explore [https://developers.google.com/maps/documentation/address-validation/cloud-setup?hl=ko]
Google Maps Platform 시작하기 [https://developers.google.com/maps/documentation/address-validation/cloud-setup?hl=ko]
계정을 만들고, API 키를 생성하고, 지도 작성을 시작합니다.
done_all [https://developers.google.com/maps/documentation/address-validation/requests-validate-address?hl=ko]
첫 번째 주소 확인 요청 [https://developers.google.com/maps/documentation/address-validation/requests-validate-address?hl=ko]
주소를 확인합니다.
code [https://developers.google.com/maps/documentation/address-validation/client_libraries?hl=ko]
클라이언트 라이브러리 [https://developers.google.com/maps/documentation/address-validation/client_libraries?hl=ko]
C#, Java, Python, Go 또는 Node.js 클라이언트 라이브러리를 사용하여 서버에서 Google 지도 서비스를 사용하세요.
code [https://developers.google.com/maps/documentation/address-validation/reference/rest?hl=ko]
API 참조 보기 [https://developers.google.com/maps/documentation/address-validation/reference/rest?hl=ko]
Address Validation REST 및 gRPC API 참조를 확인합니다.
기능
Maps Address Validation API의 핵심 기능에 대해 알아보세요.
주소 확인 [https://developers.google.com/maps/documentation/address-validation/requests-validate-address?hl=ko]
주소를 검증하고 주소의 지오코드 정보를 반환합니다.
업데이트된 주소 처리 [https://developers.google.com/maps/documentation/address-validation/handle-updated-address?hl=ko]
업데이트된 주소에 대한 의견을 보냅니다.
CASSTM 지원 [https://developers.google.com/maps/documentation/address-validation/overview?hl=ko#about-cass]
Address Validation API는 주소 확인 프로세스의 일부로 CASSTM를 선택적으로 사용할 수 있습니다.
