확대/축소 수준 스타일 지정
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/cloud-customization/zoom-levels-leg?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/cloud-customization/zoom-levels-leg?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/cloud-customization/zoom-levels-leg?hl=ko] 웹 서비스 [https://developers.google.com/maps/documentation/maps-static/cloud-customization/zoom-levels-leg?hl=ko]
Android에서 클라우드 기반 지도 스타일 지정을 사용하려면 Android용 Maps SDK를 위한 최신 렌더기를 사용해야 합니다. 다음 안내를 참고하세요 [https://developers.google.com/maps/documentation/android-sdk/renderer?hl=ko].
지원 대상: 이 기능은 Android, iOS, JavaScript, Maps Static에서 이용할 수 있습니다.
지도가 표시되는 확대/축소 수준에서 지형지물을 강조 표시하여 지도를 사용자의 필요에 더 적합하게 조정하세요. 예를 들어 사용자가 특정 관심 위치를 확대하여 보는 경우 관심 장소 또는 도로를 강조 표시할 수 있습니다.
지도 지형지물 요소가 모든 확대/축소 수준에서 지도에 동일하게 표시되도록 스타일을 지정하거나, 일부 요소의 경우 확대/축소 수준에 따라 다른 스타일을 지정할 수 있습니다.
모든 지형지물에 대해 확대/축소 수준을 맞춤설정할 수 있는 것은 아닙니다. 확대/축소 수준 스타일 지정을 설정할 수 있는 지도 지형지물 요소의 오른쪽에는 다이아몬드 아이콘이 표시됩니다.
확대/축소 수준 스타일 지정 이해하기
확대/축소 수준의 범위는 가장 축소된 수준인 0(세계를 보여줌)에서 가장 확대된 수준인 22(지도의 작은 상세 부분을 보여줌)까지입니다. 키 확대/축소는 새로운 스타일이 시작되는 확대/축소 수준입니다.
여러 확대/축소 수준에서 지도 지형지물의 스타일을 지정하면 해당 스타일은 다른 스타일이 지정된 확대/축소 수준에 도달할 때까지 더 높은(더 확대된) 확대/축소 수준에 걸쳐 적용됩니다. 확대/축소 수준 0에는 항상 현재 스타일이 적용됩니다.
예를 들어 확대/축소 수준 5와 10에서 스타일을 설정하는 경우,
확대/축소 수준 5 스타일은 확대/축소 수준 5~9에 적용됩니다.
확대/축소 수준 10 스타일은 확대/축소 수준 10~22에 적용됩니다.
확대/축소 수준 0~4는 현재 스타일을 유지합니다.
지도 지형지물의 확대/축소 수준 스타일을 맞춤설정하면 전체 지도 지형지물의 모든 맞춤 스타일이 재정의됩니다.
참고: 모든 확대/축소 수준에 표시되는 지도 지형지물은 매우 적습니다. 내가 정의하는 스타일은 지도 지형지물이 표시되는 확대/축소 수준에만 적용됩니다.
여러 확대/축소 수준 맞춤설정
Cloud 콘솔에서 지도 스타일 [https://console.cloud.google.com/project/_/google/maps-apis/studio/styles?hl=ko]로 이동하고, 필요한 경우 프로젝트를 선택합니다.
지도 스타일을 만들거나 열고 스타일 편집기에서 맞춤설정을 선택합니다. 자세한 내용은 지도 스타일 만들기 [https://developers.google.com/maps/documentation/android-sdk/cloud-customization/map-styles-leg?hl=ko#create-style]를 참고하세요.
지도 지형지물 창에서 맞춤설정할 지도 지형지물을 선택합니다. 창이 열리고 맞춤설정할 수 있는 요소가 표시됩니다.
확대/축소 수준 스타일 지정을 추가할 요소 옆에서 오른쪽의 확대/축소 다이아몬드를 선택하여 확대/축소 수준 스타일 지정 창을 엽니다.
현재 확대/축소 옆에서 맞춤설정할 확대/축소 수준을 선택합니다. 미리보기 지도가 해당 확대/축소 수준으로 조정됩니다.
키 확대/축소 추가를 선택합니다.
확대/축소 수준 스타일 지정 창이 현재 스타일(이 경우 검은색)로 확대/축소 수준 0을 표시하도록 변경되고, 선택된 확대/축소 수준에 대한 선이 추가됩니다. 이 확대/축소 수준의 스타일을 지정하고, 필요한 경우 확대/축소 수준 0의 스타일을 지정합니다.
참고: 확대/축소 스타일 지정은 선택된 확대/축소 수준 이상(더 확대된 수준)에 적용됩니다. 따라서 이 경우 확대/축소 수준 0(세계)에서 11까지 채우기 색상은 검은색(이 요소에 설정된 현재 스타일)이고 확대/축소 수준 12에서 22까지는 파란색입니다.
다른 확대/축소 수준을 설정하려면 현재 확대/축소를 변경하고 키 확대/축소 추가를 다시 선택하여 목록에 다른 확대/축소 수준을 추가합니다. 완료할 때까지 확대/축소 수준 스타일을 계속 추가하고 설정합니다.
완료되면 확대/축소 수준 스타일 지정 창을 닫습니다.
확대/축소 수준 스타일 지정 저장 및 게시
지도 스타일 지정 변경사항을 저장하려면 저장을 선택합니다.
내 지도 스타일과 연결된 지도 ID에 변경사항을 적용하려면 게시를 선택합니다.
확대/축소 수준 스타일 삭제
지도 지형지물 창에서 맞춤설정할 지도 지형지물을 선택합니다.
확대/축소 수준 스타일 지정을 삭제할 요소 옆에서 오른쪽의 작은 확대/축소 다이아몬드를 선택합니다. 확대/축소 수준 스타일이 지정된 요소에는 속이 채워진 파란색 다이아몬드가 표시됩니다.
삭제할 확대/축소 수준 옆에서 휴지통 아이콘을 선택하여 확대/축소 수준을 삭제합니다.
X를 선택하여 확대/축소 수준 스타일 지정 창을 닫습니다.
모든 확대/축소 수준 스타일 삭제
지도 지형지물의 요소 창에서 확대/축소 수준 스타일 지정 창을 엽니다.
모두 삭제될 때까지 각 확대/축소 수준 옆의 휴지통 아이콘을 선택합니다.
해당 지도 지형지물 요소에서 모든 맞춤 스타일 지정을 삭제하려면, 요소 창에서 요소 이름 옆의 빼기 아이콘을 선택하세요.
