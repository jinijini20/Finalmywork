장소 데이터 필드 (신규)
플랫폼 선택: Android iOS JavaScript 웹 서비스
장소 데이터 필드는 장소 세부정보 (신규), 텍스트 검색 (신규), 주변 지역 검색 (신규)의 응답 Place 객체에서 반환할 장소 데이터를 정의합니다. 요청에서 Place.Field에서 반환할 필드 목록(필드 마스크 또는 필드 목록이라고 함)을 지정합니다. 필드 마스크에 필드를 하나 이상 지정하지 않거나 필드 마스크를 생략하면 호출에서 오류가 반환됩니다.
이 페이지에는 가격 등급 (SKU)별로 모든 장소 데이터 필드가 나와 있습니다. 요청에 대한 요금 청구 방식에 대한 자세한 내용은 사용량 및 결제를 참고하세요.
중요: 장소 데이터 결과는 비워둘 수 없으므로 데이터가 있는 장소 결과만 반환됩니다. 예를 들어 요청된 장소에 사진이 없으면 결과에 photos 필드가 표시되지 않습니다.
참고:
Android용 Places SDK 4.0 버전에서는 여러 장소 데이터 필드의 지원 중단과 새 필드의 추가가 도입되었습니다. 자세한 내용은 Android용 Places SDK 4.0의 새로운 장소 데이터 필드로 이전 을 참고하세요.
RESOURCE_NAME 필드에는 places/PLACE_ID 형식의 장소 리소스 이름이 포함됩니다. DISPLAY_NAME을 사용하여 장소의 텍스트 이름에 액세스합니다.
