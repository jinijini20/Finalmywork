개요
이 제품 또는 기능은 미리보기(GA 이전) 버전입니다. GA 이전 제품과 기능은 지원이 제한적일 수 있으며, GA 이전 제품과 기능에 대한 변경사항은 다른 GA 이전 버전과 호환되지 않을 수 있습니다. GA 이전 버전 제품 및 서비스에는 Google Maps Platform 서비스별 약관 [https://cloud.google.com/maps-platform/terms/maps-service-terms?hl=ko]이 적용됩니다. 자세한 내용은 출시 단계 설명 [https://developers.google.com/maps/launch-stages?hl=ko]을 참고하세요.
지도 JavaScript의 포토리얼리스틱 3D 지도 클래스를 통해 애플리케이션에서 Google의 3D 이미지를 사용할 수 있습니다. Google의 기본 제공되는 3D 렌더링 환경으로 3D 지도를 빠르고 원활하게 통합하거나 이를 사용하여 실험할 수 있습니다.
Maps JavaScript API의 3D 지도로 할 수 있는 작업의 예시는 다음과 같습니다.
핵심 기능 및 역량
통합된 데이터 및 렌더기 솔루션: 지도 JavaScript의 포토리얼리스틱 3D 지도는 생생한 지도 환경을 생성하기 위한 단일 터치 포인트가 될 수 있습니다. 기본 제공되는 렌더링 기능과 다양한 장소가 포함된 내장 기본 지도로 개발 단계를 간소화하고 3D 지도 시각화를 빠르게 빌드할 수 있습니다.
성능: Google의 기본 제공 렌더링 기술로 표시 속도가 빠르며 원활하게 작동하는 사용자 환경을 구축할 수 있습니다.
스타일 지정: 지도 JavaScript의 포토리얼리스틱 3D 지도에서는 다중선, 다각형, 압출 도형을 통한 스타일 지정이 지원되므로 사용자의 시선을 끄는 실제 환경의 주요 지리적 요소를 구축할 수 있습니다.
카메라 경로 사전 설정: 사전 정의된 카메라 애니메이션을 사용하여 사용자를 3D 지도에서 손쉽게 안내하고, 관심 장소로 매끄럽게 전환하고, 다양한 각도에서 주요 위치를 보여줍니다.
맞춤 마커: 완전히 맞춤설정 가능한 마커로 지도의 시각적 매력과 기능을 향상할 수 있습니다. 이를 통해 3D 환경과 원활하게 조화를 이루거나 강조되는 위치에 대해 고유한 식별자를 만들 수 있습니다.
3D 모델: 건물, 명소 또는 맞춤 객체의 상세한 3D 모델을 통합하여 지도 작성 환경을 개선하고 사용자에게 지도에 표시된 지역을 훨씬 더 몰입도 높고 유용한 정보를 제공합니다.
광범위한 데이터: Google은 계속 최고의 3D 이미지를 제공하고 실제 환경에 대한 깊은 이해를 돕기 위해 항상 지도 플랫폼에 고품질 데이터를 많이 가져오고 있습니다.
