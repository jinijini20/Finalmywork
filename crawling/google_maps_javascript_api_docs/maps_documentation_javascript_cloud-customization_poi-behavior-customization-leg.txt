관심 장소 밀도 조정
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/cloud-customization/poi-behavior-customization-leg?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/cloud-customization/poi-behavior-customization-leg?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/cloud-customization/poi-behavior-customization-leg?hl=ko] 웹 서비스 [https://developers.google.com/maps/documentation/maps-static/cloud-customization/poi-behavior-customization-leg?hl=ko]
지원 대상: 이 기능은 Android, iOS, Maps JavaScript API, Maps Static에서 이용할 수 있습니다.
지도에서 관심 장소(POI)의 밀도를 제어할 수 있습니다. 공원, 학교 및 레크리에이션 공간 등의 관심 장소는 기본적으로 해당 아이콘과 함께 지도에 나타납니다.
관심 장소를 포함한 특정 지도 지형지물의 공개 상태를 관리하려면 표시할 지도 지형지물 필터링 [https://developers.google.com/maps/documentation/javascript/cloud-customization/viz-leg?hl=ko]을 참고하세요.
실험용: 이 기능은 밝은 지도 스타일에만 설정할 수 있습니다. 관심 장소 밀도 설정이 있는 밝은 지도 스타일을 지도 ID에 연결하면 이러한 설정이 어두운 지도 스타일에도 적용됩니다.
관심 장소 밀도 제어하기
밀도를 조정하여 지도에 표시할 관심 장소의 수를 관리할 수 있습니다.
관심 장소가 보일 정도로 지도를 확대합니다.
지도 지형지물에서 톱니바퀴 아이콘 을 선택하여 설정 창을 엽니다.
원하는 밀도를 선택합니다. 설정 창이 닫히고 지도에 관심 장소의 밀도가 조정되는 것을 확인할 수 있습니다.
