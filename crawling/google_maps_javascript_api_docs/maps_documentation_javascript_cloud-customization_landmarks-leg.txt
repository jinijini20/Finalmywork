명소 스타일 변경
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/cloud-customization/landmarks-leg?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/cloud-customization/landmarks-leg?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/cloud-customization/landmarks-leg?hl=ko] 웹 서비스 [https://developers.google.com/maps/documentation/maps-static/cloud-customization/landmarks-leg?hl=ko]
지원 대상: 이 기능은 Android, iOS, Maps JavaScript API, Maps Static에서 이용할 수 있습니다.
명소와 유명 장소에는 자체 아이콘이 있을 수 있습니다. 두 가지 아이콘 유형 중 하나를 선택하여 이러한 유형의 장소를 표현할 수 있습니다. 표준 관심 장소 아이콘은 지도의 다른 장소 마커와 모양이 유사하고, 일러스트레이션 관심 장소 아이콘은 표준 아이콘과 구별되는 독특한 느낌을 줍니다.
실험용: 이 기능은 밝은 지도 스타일에만 설정할 수 있습니다. 맞춤 랜드마크 스타일이 있는 밝은 지도 스타일을 지도 ID에 연결하면 이러한 스타일이 어두운 지도 스타일에도 적용됩니다.
랜드마크 아이콘 스타일을 변경하려면 다음 단계를 따르세요.
랜드마크가 보일 정도로 지도를 확대합니다. 예를 들어 확대/축소 수준 13으로 파리를 확대합니다.
지도 지형지물에서 톱니바퀴 아이콘 을 선택하여 설정 창을 엽니다.
표준 랜드마크 스타일을 선택하여 표준 관심 장소 아이콘을 표시합니다.
삽화 랜드마크 스타일을 선택하여 독특한 관심 장소 아이콘을 표시합니다.
참고: 일러스트레이션 아이콘을 사용 설정하면 명소 관심 장소 아이콘이 연결된 일러스트레이션 아이콘이 있는 관심 장소를 대체합니다. 일러스트 아이콘을 사용 중지하면 일러스트 아이콘이 해당 관심 장소의 원래 기능 유형으로 되돌아갑니다.
필터 또는 지도 검사기를 사용하여 랜드마크 요소 창을 엽니다. 자세한 내용은 지도 지형지물 찾기 [https://developers.google.com/maps/documentation/javascript/cloud-customization/map-styles-leg?hl=ko]를 참고하세요.
랜드마크 요소 창에서 공개 상태, 아이콘 색상, 텍스트 채우기 색상, 텍스트 획 색상을 맞춤설정할 수 있습니다.
