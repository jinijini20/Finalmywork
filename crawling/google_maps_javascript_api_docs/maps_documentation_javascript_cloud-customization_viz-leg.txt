표시할 지도 지형지물 필터링
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/cloud-customization/viz-leg?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/cloud-customization/viz-leg?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/cloud-customization/viz-leg?hl=ko] 웹 서비스 [https://developers.google.com/maps/documentation/maps-static/cloud-customization/viz-leg?hl=ko]
공개 상태를 조정하여 표시할 특정 지도 지형지물을 필터링할 수 있습니다. 지도에서 모든 관심 장소(POI)의 밀도를 조정하려면 관심 장소의 밀도 조정 [https://developers.google.com/maps/documentation/javascript/cloud-customization/poi-behavior-customization-leg?hl=ko]을 참조하세요.
지도 지형지물에 대한 공개 상태를 설정하여 지도에 표시할 관심 장소를 맞춤설정합니다.
표시하거나 숨기려는 관심 장소 지도 지형지물을 스타일 편집기에서 선택합니다. 예를 들어 관심 장소 > 숙박 시설을 선택합니다.
공개 상태에서 공개 상태는 눈 아이콘 visibility 전환으로 관리됩니다. 눈 모양 아이콘을 선택하여 공개 상태 선택을 사용 설정하면 아이콘이 파란색으로 바뀝니다. 기본 베이스맵 공개 상태는 회색으로 표시됩니다.
아이콘을 다시 선택하면 표시 여부가 사용 중과 사용 안 함 간에 전환됩니다.
켜짐: 확대/축소 수준에서 허용되는 경우 항상 이 지도 지형지물을 표시합니다.
꺼짐: 어떤 경우에도 해당 지도 지형지물을 표시하지 않습니다.
참고: 지도 기능을 숨기면 경계 부정확성이 드러날 수 있습니다. 지도 기능 경계는 항상 정확하지는 않지만 모든 레이어가 표시될 때 올바르게 보이도록 조정됩니다. 지도 레이어가 서로에게 미치는 영향을 자세히 알아보려면 겹치는 스타일 관리 [https://developers.google.com/maps/documentation/javascript/cloud-customization/overlap-leg?hl=ko]를 참고하세요.
