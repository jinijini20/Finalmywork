장소 데이터 필드는 장소 세부정보 (신규), 텍스트 검색 (신규), 주변 지역 검색 (신규)의 응답으로 반환할 장소 데이터를 정의합니다. 요청에서 속성 필드 열에서 반환할 필드 목록(필드 마스크 또는 필드 목록이라고 함)을 지정합니다. 필드 마스크에 필드를 하나 이상 지정하지 않거나 필드 마스크를 생략하면 호출에서 오류가 반환됩니다.
