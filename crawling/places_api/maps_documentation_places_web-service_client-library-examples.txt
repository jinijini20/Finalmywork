클라이언트 라이브러리 설치
설치 안내는 Places API (신규) 클라이언트 라이브러리를 참고하세요.
인증
클라이언트 라이브러리를 사용할 때는 애플리케이션 기본 사용자 인증 정보(ADC)를 사용하여 인증합니다. ADC 설정은 애플리케이션 기본 사용자 인증 정보에 대한 사용자 인증 정보 제공을 참고하세요. 클라이언트 라이브러리에서 ADC를 사용하는 방법은 클라이언트 라이브러리를 사용하여 인증을 참고하세요.
API 키를 사용하여 클라이언트 라이브러리에 인증할 수도 있습니다. 예를 들면 다음과 같습니다.
애플리케이션에서 API 키를 사용하는 경우 저장 및 전송 중에 안전하게 보호되도록 해야 합니다. API 키를 공개적으로 노출하면 계정에 예상치 않은 비용이 부과될 수 있습니다. 자세한 내용은 Places API와 함께 API 키 사용을 참고하세요.
이 페이지의 예에서는 애플리케이션 기본 사용자 인증 정보를 사용합니다.

다음은 클라이언트 라이브러리를 사용하여 Nearby Search를 호출하는 방법의 예입니다.
위도 및 경도 좌표와 반경을 사용하여 원 위치 제한을 정의합니다. 예시 요청을 빌드할 때는 위치 제한과 함께 포함된 유형 필터(레스토랑)를 사용합니다. FieldMask는 요청 시 전달됩니다.

위도 및 경도 좌표와 반경을 사용하여 Circle 객체를 빌드하여 Location Bias를 정의합니다. 다른 API 매개변수와 함께 Circle 객체를 요청 객체에 전달합니다.

장소 사진 (신규)
다음은 클라이언트 라이브러리를 사용하여 Place Photo를 호출하는 방법의 예입니다.
먼저 장소 세부정보 (신규)를 호출하여 장소의 사진을 요청합니다. 장소 세부정보 (신규)는 사진 이름으로 응답합니다. 그런 다음 장소 세부정보 (신규)에서 반환된 첫 번째 사진 이름이 장소 사진 (신규)을 호출하는 데 사용됩니다. 반환된 사진의 최대 너비도 설정합니다.
