Map Tiles API 개요
유럽 경제 지역 (EEA) 개발자
Map Tiles API를 사용하면 Google의 2D 타일과 포토리얼리스틱 3D 타일에 직접 액세스할 수 있습니다. 2D 타일은 색인이 지정된 그리드로 나뉜 세계를 묘사하는 이미지 파일입니다. 2D 타일과 마찬가지로 3D 타일은 이미지 파일이지만 건물, 기념물, 기타 관심 장소의 실사 이미지가 포함되어 있어 확대해야 볼 수 있습니다. 2D 및 3D 타일을 사용하여 Google 지도 기반 몰입형 환경을 자체 애플리케이션에 구축할 수 있습니다.
Map Tiles API를 사용해야 하는 이유
Map Tiles API는 다음을 충족하는 Google 고객을 위한 것입니다.
JavaScript 또는 모바일 SDK를 사용하지 않고 지도 환경을 빌드하고 싶습니다.
데스크톱에서 웹 기반이 아닌 맞춤 앱을 사용합니다.
맞춤 하드웨어 및 지도가 필요한 기타 도구
자동차 및 항공용 내비게이션 시스템
Street View 서비스 또는 Static Street View API에서 제공하는 것보다 더 많은 스트리트 뷰 메타데이터가 필요합니다.
2D 및 3D로 고도로 맞춤화된 몰입형 시각화를 만들고 싶습니다.
사용 가능한 지도 타일
다음 테마 지도 타일은 Map Tiles API를 통해 제공됩니다. 사용 범위 정보는 Google Maps Platform 사용 범위 세부정보 [https://developers.google.com/maps/coverage?hl=ko]를 참고하세요.
지도 테마 설명
로드맵 [https://developers.google.com/maps/documentation/tile/roadmap?hl=ko] 표준 Google 지도 기본 지도 타일 + 오버레이 데이터 (도로, 건물, 관심 장소, 정치적 경계 포함)
위성 지도 [https://developers.google.com/maps/documentation/tile/satellite?hl=ko] 위성 및 항공 이미지
지형 [https://developers.google.com/maps/documentation/tile/terrain?hl=ko] 초목과 같은 자연 지형지물을 보여주는 음영 및 등고선 지도
스트리트 뷰 [https://developers.google.com/maps/documentation/tile/streetview?hl=ko] 메타데이터가 포함된 스트리트 뷰 파노라마
실사 3D [https://developers.google.com/maps/documentation/tile/3d-tiles-overview?hl=ko] 위성 및 항공 이미지가 포함된 3D 텍스처 메쉬
팁: Map Tiles API를 사용하여 오버레이 정보에 액세스할 수도 있습니다. 자세한 내용은 선택사항인 overlay [https://developers.google.com/maps/documentation/tile/session_tokens?hl=ko#overlay] 필드를 참고하세요.
Map Tiles API 사용 방법
2D 타일
1 Google Maps Platform 서비스 약관 검토하기 자세한 내용은 Google Maps Platform 서비스 약관 [https://cloud.google.com/maps-platform/terms?hl=ko] 및 지도 타일 API 정책 [https://developers.google.com/maps/documentation/tile/policies?hl=ko]을 참고하세요.
2 설정하기 Google Cloud 프로젝트 설정하기 [https://developers.google.com/maps/documentation/tile/cloud-setup?hl=ko]부터 시작하여 다음 안내를 완료합니다.
3 API 키 가져오기 API 키를 만들고 보호합니다. 자세한 내용은 API 키 사용 [https://developers.google.com/maps/documentation/tile/get-api-key?hl=ko]을 참고하세요.
4 세션 토큰 가져오기 mapType [https://developers.google.com/maps/documentation/tile/session_tokens?hl=ko#required_fields] 및 세션의 기타 옵션을 지정하여 세션 토큰 [https://developers.google.com/maps/documentation/tile/session_tokens?hl=ko]을 가져옵니다.
5 2D 지도 타일 요청 API 키와 세션 토큰을 사용하여 지도 타일을 요청 [https://developers.google.com/maps/documentation/tile/2d-tiles-overview?hl=ko#tile-request]하거나 파노라마 ID를 가져오거나 [https://developers.google.com/maps/documentation/tile/streetview?hl=ko#panoid-search] 스트리트 뷰 메타데이터를 가져옵니다 [https://developers.google.com/maps/documentation/tile/streetview?hl=ko#street_view_metadata]. 스트리트 뷰 파노라마를 요청하려면 스트리트 뷰 타일 [https://developers.google.com/maps/documentation/tile/streetview?hl=ko]을 참고하세요.
6 뷰포트 정보 가져오기 기본 지도, 위성, 지형 타일의 경우 뷰포트 정보를 검색 [https://developers.google.com/maps/documentation/tile/2d-tiles-overview?hl=ko#viewport_info_request]하여 표시된 타일에 관한 세부정보를 확인할 수 있습니다. 뷰포트 요청은 이미지가 있는 지역과 확대/축소 수준을 알려줍니다. 모든 지역이 최대 확대/축소 수준 22를 지원하지는 않기 때문입니다.
포토리얼리스틱 3D 타일
1 Google Maps Platform 서비스 약관 검토하기 자세한 내용은 Google Maps Platform 서비스 약관 [https://cloud.google.com/maps-platform/terms?hl=ko] 및 지도 타일 API 정책 [https://developers.google.com/maps/documentation/tile/policies?hl=ko]을 참고하세요.
2 설정하기 Google Cloud 프로젝트 설정하기 [https://developers.google.com/maps/documentation/tile/cloud-setup?hl=ko]부터 시작하여 다음 안내를 완료합니다.
3 API 키 가져오기 API 키를 만들고 보호합니다. 자세한 내용은 API 키 사용 [https://developers.google.com/maps/documentation/tile/get-api-key?hl=ko]을 참고하세요.
4 3D 포토리얼리스틱 타일 요청 API 키가 있으면 원하는 3D 타일 렌더러에 루트 타일 세트 URL을 지정하여 포토리얼리스틱 타일에 액세스할 수 있습니다. 자세한 내용은 3D 타일 가져오기 [https://developers.google.com/maps/documentation/tile/3d-tiles?hl=ko#getting_3d_tiles]를 참고하세요.
