표시할 지도 지형지물 필터링
공개 상태를 조정하여 표시할 특정 지도 지형지물을 필터링할 수 있습니다. 지도에서 모든 관심 장소(POI)의 밀도를 조정하려면 관심 장소의 밀도 조정을 참조하세요.
지도 지형지물에 대한 공개 상태를 설정하여 지도에 표시할 관심 장소를 맞춤설정합니다.
표시하거나 숨기려는 POI 지도 지형지물을 스타일 편집기에서 선택합니다. 예를 들어, 관심 장소>숙박시설을 선택합니다.
공개 상태에서 다음 옵션 중 하나를 선택합니다.
켜짐: 확대/축소 수준이 허용하는 경우 항상 이 지도 지형지물을 표시합니다.
꺼짐: 해당 지도 지형지물을 표시하지 않습니다.
