컨텍스트에 따라 지도 스타일 지정
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/cloud-customization/contextual-styling?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/cloud-customization/contextual-styling?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/cloud-customization/contextual-styling?hl=ko] 웹 서비스 [https://developers.google.com/maps/documentation/maps-static/cloud-customization/contextual-styling?hl=ko]
어두운 스타일은 실험용 [https://developers.google.com/maps/launch-stages?hl=ko#experimental]입니다.
이제 컨텍스트를 사용하여 앱에서 지도 스타일을 변경할 수 있습니다. 지도 ID를 변경하지 않고도 앱에서 스타일을 자동으로 전환할 수 있도록 밝은 스타일, 어두운 스타일, 흑백 지도 스타일을 만드세요. 지도 스타일을 만들 때 밝은 스타일, 어두운 스타일, 밝은 흑백 스타일, 어두운 흑백 스타일 등 여러 기본 스타일 중에서 선택할 수 있습니다. 타겟 스타일에 가장 가까운 기본 스타일을 선택한 다음 필요에 따라 맞춤설정합니다.
밝은 지도 스타일과 어두운 지도 스타일을 맞춤설정하면 두 스타일을 하나의 지도 ID와 연결할 수 있습니다. 애플리케이션에서 지도 ID를 사용하면 지도가 밝은 모드 또는 어두운 모드로 전환될 때 지도 스타일이 지도 스타일 간에 자동으로 전환됩니다.
지도 ID로 밝은 지도 스타일과 어두운 지도 스타일을 사용하려면 새 스타일 만들기 [https://developers.google.com/maps/documentation/javascript/cloud-customization/map-styles-leg?hl=ko#create-new-style] 및 스타일을 지도 ID에 연결 [https://developers.google.com/maps/documentation/javascript/cloud-customization/map-styles-leg?hl=ko#associate-style-with-map-id]로 이동하세요.
