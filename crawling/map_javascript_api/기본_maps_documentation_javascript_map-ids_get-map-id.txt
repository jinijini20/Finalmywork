지도 ID를 만드는 방법
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/map-ids/get-map-id?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/map-ids/get-map-id?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/map-ids/get-map-id?hl=ko] 웹 서비스 [https://developers.google.com/maps/documentation/maps-static/map-ids/get-map-id?hl=ko]
유료 기능: 지도 ID [https://developers.google.com/maps/documentation/get-map-id?hl=ko]를 추가하여 액세스하는 기능의 경우, Android 및 iOS용 동적 지도 SKU에 대한 요금이 부과되는 지도 로드가 트리거됩니다. 자세한 내용은 Google 지도 결제 [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#dynamic-maps-ess-sku]를 참고하세요.
지도 ID는 Google Cloud에 저장된 Google 지도 스타일 지정 및 구성 설정을 나타내는 고유 식별자입니다. 지도 ID를 사용하여 웹사이트와 애플리케이션에서 기능을 사용 설정하거나 지도를 관리하거나 스타일을 지정합니다. Google Cloud 콘솔 프로젝트의 지도 관리 페이지에서 필요한 각 플랫폼(JavaScript, Android, iOS 또는 정적 지도)의 지도 ID를 만들 수 있습니다.
지도 ID를 사용하는 기능 및 자세한 내용은 지도 ID 개요 [https://developers.google.com/maps/documentation/javascript/map-ids/mapid-over?hl=ko]를 참고하세요.
필수 권한
프로젝트에서 지도 ID를 만들거나 관리하려면 프로젝트의 Cloud 콘솔 IAM 페이지에서 적절한 역할 수준 권한(편집자 또는 소유자)이 있는 주 구성원을 사용해야 합니다. 자세한 내용은 IAM 기본 및 사전 정의된 역할 참조 [https://cloud.google.com/iam/docs/understanding-roles?hl=ko#role_types]를 확인하세요.
지도 ID 만들기
다음 단계에 따라 Cloud 콘솔에서 지도 ID를 만듭니다.
필요한 권한 [https://developers.google.com/maps/documentation/javascript/map-ids/get-map-id?hl=ko#permissions]이 있는 Cloud 콘솔 프로젝트에 로그인하여 엽니다.
Cloud Console에서 지도 관리 페이지 [https://console.cloud.google.com/google/maps-apis/studio/maps?hl=ko]로 이동합니다.
지도 ID 만들기를 클릭합니다.
새 지도 ID 만들기 페이지에서 다음을 수행합니다.
이름에 지도 ID의 이름을 지정합니다.
선택사항: 설명에 지도 ID의 용도를 설명합니다.
지도 유형에서 지도 ID를 사용할 플랫폼을 선택합니다. JavaScript를 선택한 경우 래스터 (기본값) 또는 벡터 지도 유형도 선택합니다. 벡터 지도에 관한 자세한 내용은 벡터 지도 [https://developers.google.com/maps/documentation/javascript/vector-map?hl=ko]를 참고하세요.
저장을 클릭하여 새 지도 ID를 표시합니다.
지도 ID를 지도 스타일에 연결
클라우드 기반 지도 스타일 지정을 사용하는 경우 지도 스타일을 지도 ID와 연결합니다. 자세한 내용은 스타일을 지도 ID에 연결 [https://developers.google.com/maps/documentation/javascript/cloud-customization/map-styles-leg?hl=ko#associate-style-with-map-id]을 참고하세요.
