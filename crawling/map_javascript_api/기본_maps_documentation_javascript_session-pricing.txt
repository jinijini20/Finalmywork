유럽 경제 지역 (EEA) 개발자
Google에서는 자동 완성 (신규) [https://developers.google.com/maps/documentation/javascript/place-autocomplete?hl=ko]과 함께 세션을 사용하는 것이 좋습니다. 세션을 사용하면 가격을 간소화하고 자동 완성 (신규)과 함께 사용할 수 있는 일관된 가격 책정 모델을 제공할 수 있습니다.
세션은 세션 토큰을 포함하는 첫 번째 Autocomplete (New) 요청으로 시작하여 후속 Autocomplete (New) 요청을 통해 계속되고 세션 토큰을 사용하는 Place Details (New) 또는 Address Validation 요청에 의해 종료됩니다.
세션이 종료된 후 만료된 세션 토큰을 사용하는 Autocomplete (New), Place Details (New) 또는 Address Validation에 대한 호출은 세션 토큰이 없는 것처럼 청구됩니다.
Place Details (New) 또는 Address Validation 요청으로 종료되지 않은 세션과 같은 불완전한 세션의 경우 SKU: Autocomplete Requests [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#places_autocomplete-request-new-ess-sku]를 사용하는 Autocomplete (New) 요청에 대해서만 요금이 청구됩니다.
세션 결제
Place Details (New) Essentials 요청으로 종료되는 세션
SKU: Places API Place Details Essentials [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#place-details-ess-sku]의 필드를 사용하는 요청으로 종료되는 Autocomplete (New) 세션의 경우:
처음 12개의 Autocomplete (신규) 요청: SKU: Autocomplete Requests [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#places_autocomplete-request-new-ess-sku]를 사용하여 최대 12개의 요청까지 각 Autocomplete(신규) 요청에 대해 요금이 청구됩니다.
동일한 세션에서 13 이상의 자동 완성 (신규) 요청: SKU: Autocomplete Session Usage [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#autocomplete-session-new-ess-sku]에 따라 요금이 청구되므로 이러한 요청에는 요금이 청구되지 않습니다.
Place Details (신규) Essentials: 종료 요청에 대해서도 SKU: Places API Place Details Essentials [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#place-details-ess-sku]로 요금이 청구됩니다.
참고: SKU: Places API Place Details Essentials (IDs Only) [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#place-details-id-only-ess-sku]를 사용하여 Place Details (New)에 요청을 전송하여 세션을 종료하면 세션을 사용하지 않은 것처럼 Autocomplete (New)에 대한 모든 요청에 SKU: Autocomplete Requests [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#places_autocomplete-request-new-ess-sku]를 사용하여 요금이 청구됩니다. SKU: Places API Place Details Essentials (IDs Only) [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#place-details-id-only-ess-sku]를 사용하는 요청에는 요금이 청구되지 않으므로 모든 Autocomplete (신규) 요청이 요청별 가격으로 되돌아갑니다.
Address Validation 또는 Place Details (New) (비필수) 요청으로 종료되는 세션
Place Details Pro [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#place-details-pro-sku], Enterprise [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#place-details-ent-sku] 또는 Enterprise + Atmosphere [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#place-details-ent-plus-sku]의 필드를 사용하여 주소 유효성 검사 요청 또는 Place Details (New) 요청으로 종료되는 Autocomplete (New) 세션의 경우:
Autocomplete (New) 요청: 모든 Autocomplete (New) 요청은 SKU: Autocomplete Session Usage [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#autocomplete-session-new-ess-sku]에 따라 요금이 청구되므로 이러한 요청에는 요금이 청구되지 않습니다.
Place Details (신규) 또는 Address Validation 요청: 적절한 SKU로 종료 요청에 대한 요금도 청구됩니다.
Place Details (New) 종료 요청은 요청된 필드와 관계없이 SKU: Place Details Enterprise + Atmosphere [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#place-details-ent-plus-sku]로 요금이 청구됩니다.
주소 확인 종료 요청은 SKU: Address Validation Enterprise [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#address-validation-ent-sku]로 청구됩니다.

이 시나리오에서는 Place Details (New) Pro [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#place-details-pro-sku], Enterprise [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#place-details-ent-sku] 또는 Enterprise + Atmosphere [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#place-details-ent-plus-sku] SKU에 포함된 필드를 요청하는 Place Details (New)에 대한 단일 요청으로 세션이 종료됩니다.

모든 Autocomplete (New) 요청에는 SKU: Autocomplete Session Usage [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#autocomplete-session-new-ess-sku]에 따라 요금이 청구되므로 이러한 요청에는 요금이 청구되지 않습니다.
종료되는 Place Details (New) 요청은 SKU: Places API Place Details Enterprise + Atmosphere [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#place-details-ent-plus-sku]로 청구됩니다.
결제 및 배송 자동 완성
Address Validation API [https://developers.google.com/maps/documentation/address-validation/overview?hl=ko]는 주소를 허용하는 서비스입니다. 주소 구성요소를 식별하고 유효성을 검사합니다. 또한 우편 주소를 표준화하고 가장 잘 알려진 위도 및 경도 좌표를 찾습니다.
결제 및 배송 시 자동 완성 시나리오에서는 선택한 주소를 검증하기 위해 Address Validation API [https://developers.google.com/maps/documentation/address-validation/requests-validate-address?hl=ko]에 대한 요청으로 세션을 종료합니다.
다음과 같이 요금이 청구됩니다.
모든 Autocomplete (New) 요청에는 SKU: Autocomplete Session Usage [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#autocomplete-session-new-ess-sku]에 따라 요금이 청구되므로 이러한 요청에는 요금이 청구되지 않습니다.
선택사항인 Place Details Essentials 요청은 종료되는 주소 유효성 검사 요청 전에 요청된 경우 무료로 청구됩니다.
종료되는 Address Validation 요청은 SKU: Address Validation Enterprise [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#address-validation-ent-sku]로 청구됩니다.
세션이 없는 가격
세션을 사용하지 않으면 SKU: Autocomplete Requests [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#places_autocomplete-request-new-ess-sku]를 사용하여 Autocomplete (New)에 대한 요청별로 요금이 청구됩니다.
불완전하거나 포기한 세션의 가격
세션이 포기된 경우(Place Details(New) 또는 Address Validation 호출로 종료되지 않음) Autocomplete(New) 요청은 요청별 가격 모델로 되돌아가며 SKU: Autocomplete Requests [https://developers.google.com/maps/billing-and-pricing/sku-details?hl=ko#places_autocomplete-request-new-ess-sku]에 따라 요금이 청구됩니다.
