지원
이 제품 또는 기능은 미리보기(GA 이전) 버전입니다. GA 이전 제품과 기능은 지원이 제한적일 수 있으며, GA 이전 제품과 기능에 대한 변경사항은 다른 GA 이전 버전과 호환되지 않을 수 있습니다. GA 이전 버전 제품 및 서비스에는 Google Maps Platform 서비스별 약관 [https://cloud.google.com/maps-platform/terms/maps-service-terms?hl=ko]이 적용됩니다. 자세한 내용은 출시 단계 설명 [https://developers.google.com/maps/launch-stages?hl=ko]을 참고하세요.
이 페이지에는 지도 JavaScript 기능의 포토리얼리스틱 3D 지도에 대한 알려진 문제와 해결 방법이 나와 있습니다.
하드웨어 가속 사용 설정
3D 지도를 지원하려면 대부분의 브라우저에서 하드웨어 가속을 사용 설정해야 합니다. Chrome 및 Microsoft Edge에서 하드웨어 가속을 사용 설정하려면 설정을 열고, 시스템을 선택한 후, 가능한 경우 하드웨어 가속 사용을 사용 설정하세요.
Safari에서 웹사이트 환경설정을 변경 [https://support.apple.com/guide/safari/websites-ibrwe2159f50/14.0/mac/11.0]하는 방법에 대해 알아보세요.
Firefox의 성능 설정 [https://support.mozilla.org/en-US/kb/performance-settings]에 대해 알아보세요.
2025년 2월 20일 지도 초기화 변경사항
2025년 2월 20일 (버전 3.60.1 이전) 전에 실험용 v=alpha Maps JavaScript API 3D 지도를 만들었고 지도 오류나 검은색 또는 빈 지도가 표시되는 경우 Maps JavaScript API 3D 지도 기능을 계속 사용하려면 코드를 업데이트해야 합니다.
이제 3D 지도가 렌더링되려면 Map3DElement의 새 mode 옵션을 설정해야 합니다(새 값 HYBRID 및 SATELLITE 사용 가능). 기존 defaultLabelsDisabled 옵션이 삭제되고 모드가 HYBRID 또는 SATELLITE로 설정됩니다.
