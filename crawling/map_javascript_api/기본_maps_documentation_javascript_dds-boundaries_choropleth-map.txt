등치 지역도 만들기
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/dds-boundaries/choropleth-map?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/dds-boundaries/choropleth-map?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/dds-boundaries/choropleth-map?hl=ko]
등치 지역도는 데이터 값에 따라 행정 구역에 색상이 지정되거나 음영 처리되는 주제별 지도입니다. google.maps.FeatureStyleFunction [https://developers.google.com/maps/documentation/javascript/reference/data-driven-styling?hl=ko#FeatureStyleFunction]을 사용하여 각 행정 구역이 숫자 값의 범위와 연결되는 데이터 세트를 기반으로 지도의 스타일을 지정할 수 있습니다. 다음 예는 미국의 주 인구 데이터가 표시된 등치 지역도를 보여줍니다.
이 예에서 데이터는 키-값 쌍의 배열로 구성되며 각 주의 표시 이름에 인구를 나타내는 숫자 값이 연결됩니다. maps.FeatureStyleFunction은 배열의 값에 따라 조건부로 각 지역의 색상을 지정합니다.
중요: 지형지물 스타일 지정 함수는 예측 가능한 출력을 생성해야 합니다. 예를 들어 임의의 색상을 적용하는 예를 만들려면 코드가 FeatureStyleFunction 이외의 부분에서 해당하는 임의의 부분을 실행해야 합니다.
