장소 UI 키트
bookmark_border
장소 UI 키트 구성요소 라이브러리를 사용하면 Places API를 지원하는 동일한 데이터를 사용하여 앱과 웹페이지에 장소에 대한 익숙한 Google 지도 사용자 경험을 구현할 수 있습니다. 독립적으로, 함께 또는 다른 Google Maps Platform API와 함께 사용하여 최소한의 비용과 코드로 장소 관련 풍부한 경험을 제공할 수 있는 개별 UI 구성요소가 포함되어 있습니다.
장소 UI 키트
참고: 장소 UI 키트 구성요소 라이브러리는 PlacesSwift SDK에서만 사용할 수 있습니다.
도움말: 장소 UI 키트를 사용한 맞춤설정에는 추가 요금이 부과되지 않습니다.
Places UI Kit은 장소 데이터를 렌더링하기 위한 다음 구성요소를 제공합니다.
장소 세부정보는 선택한 장소의 세부정보를 렌더링합니다.
Basic Place Autocomplete는 요청에 대한 응답으로 장소 ID를 반환합니다.
장소 검색은 맞춤설정 가능한 장소 관련 검색 결과를 장소 카드의 가로 또는 세로 목록에 표시합니다.
