경로 토큰 가져오기
유럽 경제 지역 (EEA) 개발자
Compute Routes가 생성된 경로의 경로 토큰을 반환하도록 요청하려면 다음 단계를 따르세요.
경로 토큰을 반환하는 데 필요한 다음 매개변수를 설정합니다.
travelMode를 DRIVE로 설정합니다.
routingPreference를 TRAFFIC_AWARE 또는 TRAFFIC_AWARE_OPTIMAL으로 설정합니다.
경로의 중간 지점이 중간 지점을 경유하지 않는지 확인합니다.
경로 토큰을 반환하도록 routes.routeToken 필드 마스크를 지정합니다.
X-Goog-FieldMask: routes.routeToken
참고: 동적 도로 상태로 인해 특정 토큰으로 생성된 경로가 계획된 경로와 다를 수 있습니다. 차이를 최소화하려면 토큰 생성 후 몇 분 이내에 토큰을 사용하세요.
Navigation SDK에서 계획된 경로의 경로 토큰을 사용할 수 있습니다. 자세한 내용은 다음을 참고하세요.
경로 계획하기 (Android) [https://developers.google.com/maps/documentation/navigation/android-sdk/customize-route?hl=ko#use-token]
경로 계획 (iOS) [https://developers.google.com/maps/documentation/navigation/ios-sdk/route-customizations?hl=ko#use-token]
경로 토큰 예시
다음은 출발지와 도착지가 각각 1개인 경로의 curl 요청 본문 예시입니다. 필드 마스크를 사용하여 경로 시간, 거리, 경로 다중선과 함께 경로 토큰을 요청합니다.
curl -X POST -d
{"origin":{
    "location": {
        "latLng":{
            "latitude":  -37.8167,
            "longitude": 144.9619
        }
    }
},
"destination":{
    "location": {
        "latLng":{
            "latitude":-37.8155,
            "longitude": 144.9663
        }
    }
},
"routingPreference":"TRAFFIC_AWARE",
"travelMode":"DRIVE"
}
-H 'X-Goog-Api-Key: YOUR_API_KEY' \
-H X-Goog-FieldMask: routes.routeToken,routes.duration,routes.distanceMeters,routes.polyline.encodedPolyline
'https://routes.googleapis.com/directions/v2:computeRoutes'
자세한 내용은 Compute Routes API 참조 [https://developers.google.com/maps/documentation/routes/reference/rest/v2/TopLevel/computeRoutes?hl=ko]를 참고하세요.
