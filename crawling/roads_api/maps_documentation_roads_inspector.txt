Roads API Inspector
Roads API 검사기는 Roads API를 사용해 볼 수 있는 대화형 도구입니다. 다음은 이 도구를 시작하는 데 도움이 되는 몇 가지 추천사항입니다.
Roads API 요청 URL을 텍스트 필드에 복사하고 경로 표시를 선택하여 요청 결과를 살펴봅니다. API 키를 포함할 필요는 없습니다.
추천 예시를 사용하여 API를 실험해 보세요.
특정 지점의 마커를 선택하면 해당 지점에 관한 세부정보가 포함된 정보 창이 표시됩니다.
예시 중 하나를 로드한 후 스트리트 뷰를 열어 스트리트 뷰 파노라마에서 마커를 확인합니다.
보간 설정을 전환하여 결과에 미치는 영향을 확인합니다.
거리 전환을 선택하여 두 원래 지점 간의 거리를 표시하거나 숨깁니다. 스냅 해제된 경로는 지점 간에 직선 녹색 선으로 표시됩니다. 선을 선택하여 거리를 확인합니다.
추천 예시는 다음과 같습니다.
예시 1: 시드니 파이어몬트의 도로에 있는 지점들.
예시 2: 호주 수도 캔버라의 도로에 있는 지점들.
예 3: 캔버라의 도로에 있는 점과 도로에 스냅할 수 없는 점이 있습니다.
예 4: 노스캐롤라이나주 엘킨의 경로로, 보간 설정을 전환한 결과를 잘 보여주는 불규칙한 경로가 있습니다.
이 예시를 전체 화면 [https://developers.google.com/maps/documentation/roads/full/inspector_with_api_key?hl=ko]으로 확인하세요.
