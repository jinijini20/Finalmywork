google.maps.Settings 클래스

Maps JavaScript API의 전체 동작을 제어하는 설정입니다.
const {Settings} = await google.maps.importLibrary("core")를 호출하여 액세스합니다. Maps JavaScript API의 라이브러리 [https://developers.google.com/maps/documentation/javascript/libraries?hl=ko]를 참고하세요.
| 정적 메소드 |
| --- |
| getInstance | getInstance() 매개변수: 없음 반환 값:  Settings google.maps.Settings의 싱글톤 인스턴스를 반환합니다. |
| 속성 |
| --- |
| experienceIds | 참고: v=beta 채널에서만 사용할 수 있습니다. 유형:  Iterable<string> 지도 JS API 호출에 기여도를 부여할 고유한 환경 ID 모음입니다. 반환된 값은 Settings 클래스 싱글톤 인스턴스에 저장된 내부 값의 사본입니다. 따라서 google.maps.Settings.getInstance().experienceIds에 대한 작업은 사본만 수정하고 내부 값은 수정하지 않습니다.  내부 값을 업데이트하려면 속성을 싱글톤 인스턴스의 새 값과 동일하게 설정합니다 (예: google.maps.Settings.getInstance().experienceIds = [experienceId];). |
| fetchAppCheckToken | 유형:  function(): Promise<MapsAppCheckTokenResult> Firebase 앱 체크 토큰 결과로 확인되는 프라미스를 반환합니다. 토큰은 'X-Firebase-AppCheck' HTTP 헤더로 gRPC 요청에 연결됩니다. 토큰이 비어 있으면 헤더가 연결되지 않습니다. |
