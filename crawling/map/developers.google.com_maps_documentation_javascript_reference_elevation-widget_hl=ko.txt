ElevationElement [https://developers.google.com/maps/documentation/javascript/reference/elevation-widget?hl=ko#ElevationElement] 클래스

google.maps.elevation.ElevationElement 클래스

참고: v=alpha 채널 [https://developers.google.com/maps/documentation/javascript/versions?hl=ko#choosing-the-alpha-channel]에서만 사용할 수 있습니다.
고도 데이터를 시각적으로 표시하는 HTML 요소입니다. 경로를 따라 고도의 그래프를 표시하도록 요소의 path 속성을 설정합니다. 고도 요소를 사용하려면 Google Cloud 콘솔에서 프로젝트에 Places UI Kit API [https://console.cloud.google.com/marketplace/product/google/placewidgets.googleapis.com?hl=ko]를 사용 설정합니다.
맞춤 요소:
<gmp-elevation unit-system="imperial"></gmp-elevation>
이 클래스는 HTMLElement를 확장합니다.

이 클래스는 ElevationElementOptions [https://developers.google.com/maps/documentation/javascript/reference/elevation-widget?hl=ko#ElevationElementOptions]를 구현합니다.

const {ElevationElement} = await google.maps.importLibrary("elevation")를 호출하여 액세스합니다. Maps JavaScript API의 라이브러리 [https://developers.google.com/maps/documentation/javascript/libraries?hl=ko]를 참고하세요.
| 생성자 |
| --- |
| ElevationElement | ElevationElement([options]) 매개변수:  options:  ElevationElementOptions optional |
| 속성 |
| --- |
| path | 유형:  Array<LatLng\|LatLngLiteral\|LatLngAltitude> optional 고도 데이터를 표시할 경로입니다. 선분은 배열의 포인트 간에 보간됩니다. 소수의 포인트로도 세부적인 고도 그래프를 생성할 수 있습니다. 점이 하나만 제공되면 위젯에 해당 지점의 고도 숫자가 표시됩니다. |
| unitSystem | 유형:  UnitSystem optional 고도를 피트/마일 또는 미터/km로 표시할지 결정합니다. 정의되지 않으면 위젯은 API 로더에 지정된 지역의 기본 단위 시스템으로 기본 설정됩니다. HTML 속성: <gmp-elevation unit-system="imperial"></gmp-elevation> <gmp-elevation unit-system="metric"></gmp-elevation> |
| 메서드 |

| addEventListener | 참고: v=beta 채널에서만 사용할 수 있습니다. addEventListener(type, listener[, options]) 매개변수:  type:  string 리슨할 이벤트 유형을 나타내는 대소문자 구분 문자열입니다. listener:  EventListener\|EventListenerObject 알림을 수신하는 객체입니다. handleEvent 메서드가 있는 함수 또는 객체여야 합니다. options:  boolean\|AddEventListenerOptions optional 옵션을 참고하세요. 맞춤 이벤트는 capture 및 passive만 지원합니다. 반환 값:  void 지정된 이벤트가 타겟에 전달될 때마다 호출될 함수를 설정합니다. addEventListener를 참고하세요. |
| removeEventListener | 참고: v=beta 채널에서만 사용할 수 있습니다. removeEventListener(type, listener[, options]) 매개변수:  type:  string 이벤트 리스너를 삭제할 이벤트 유형을 지정하는 문자열입니다. listener:  EventListener\|EventListenerObject 이벤트 타겟에서 삭제할 이벤트 핸들러의 이벤트 리스너입니다. options:  boolean\|EventListenerOptions optional 옵션을 참고하세요. 반환 값:  void 이전에 addEventListener에 등록된 이벤트 리스너를 타겟에서 삭제합니다. removeEventListener를 참고하세요. |
