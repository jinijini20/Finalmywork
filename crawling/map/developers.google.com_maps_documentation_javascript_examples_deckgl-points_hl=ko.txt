Visualizing thousands of points

This example visualizes recent earthquakes using the deck.gl GeoJsonLayer and a
ScatterPlot of earthquakes.

For more information, see https://deck.gl/ [https://deck.gl/].

Git and Node.js are required to run this sample locally. Follow these instructions [https://docs.npmjs.com/downloading-and-installing-node-js-and-npm] to install Node.js and NPM. The following commands clone, install dependencies and start the sample application.

git clone -b sample-deckgl-points https://github.com/googlemaps/js-samples.git
cd js-samples
npm i
npm start
