FeatureLayer [https://developers.google.com/maps/documentation/javascript/reference/data-driven-styling?hl=ko#FeatureLayer] 인터페이스

google.maps.FeatureLayer 인터페이스

스타일을 클라이언트 측에서 재정의하거나 이벤트를 연결할 수 있는 특정 FeatureType [https://developers.google.com/maps/documentation/javascript/reference/data-driven-styling?hl=ko#FeatureType]의 지형지물이 포함된 지도 레이어를 나타내는 인터페이스입니다.
| 속성 |
| --- |
| featureType | 유형:  FeatureType 이 FeatureLayer와 연결된 FeatureType입니다. |
| isAvailable | 유형:  boolean 이 FeatureLayer를 사용할 수 있는지 여부, 즉 이 지도에 데이터 기반 스타일 지정을 사용할 수 있는지 여부입니다. Google Cloud 콘솔 지도 스타일에서 이 FeatureLayer가 사용 설정된 벡터 타일을 사용하는 지도 ID가 있습니다. 이 값이 false이거나 false가 되면 이 FeatureLayer의 스타일이 기본값으로 돌아가고 이벤트가 트리거되지 않습니다. |
| datasetId optional | 참고: v=beta 채널에서만 사용할 수 있습니다. 유형:  string optional 이 FeatureLayer의 데이터 세트 ID입니다. featureType가 FeatureType.DATASET인 경우에만 표시됩니다. |
| style optional | 유형:  FeatureStyleOptions\|FeatureStyleFunction optional FeatureLayer의 Feature 스타일입니다. 스타일은 스타일이 설정될 때 적용됩니다. 스타일 함수가 업데이트되면 스타일 속성을 다시 설정해야 합니다. FeatureStyleFunction는 지도 타일에 적용될 때 일관된 결과를 반환해야 하며 성능에 맞게 최적화되어야 합니다. 비동기 함수는 지원되지 않습니다. FeatureStyleOptions를 사용하면 해당 레이어의 모든 지형지물에 동일한 FeatureStyleOptions 스타일이 적용됩니다. 스타일을 null로 설정하여 이전에 설정된 스타일을 삭제합니다. 이 FeatureLayer를 사용할 수 없는 경우 스타일 설정은 아무것도 하지 않고 오류를 로깅합니다. |
| 메서드 |
| --- |
| addListener | addListener(eventName, handler) 매개변수:  eventName:  string 관찰된 이벤트입니다. handler:  Function 이벤트를 처리하는 함수입니다. 반환 값:  MapsEventListener 결과 이벤트 리스너입니다. 지정된 리스너 함수를 지정된 이벤트 이름에 추가합니다. event.removeListener와 함께 사용할 수 있는 이 리스너의 식별자를 반환합니다. |
| 이벤트 |
| --- |
| click | function(event) 인수:  event:  FeatureMouseEvent 이 이벤트는 FeatureLayer가 클릭될 때 실행됩니다. |
| mousemove | function(event) 인수:  event:  FeatureMouseEvent 이 이벤트는 사용자의 마우스가 FeatureLayer 위로 이동할 때 실행됩니다. |

FeatureType [https://developers.google.com/maps/documentation/javascript/reference/data-driven-styling?hl=ko#FeatureType] 상수

const {FeatureType} = await google.maps.importLibrary("maps")를 호출하여 액세스합니다. Maps JavaScript API의 라이브러리 [https://developers.google.com/maps/documentation/javascript/libraries?hl=ko]를 참고하세요.
| 상수 |
| --- |
| ADMINISTRATIVE_AREA_LEVEL_1 | 국가 수준 아래 첫 번째 행정 독립체를 나타냅니다. |
| ADMINISTRATIVE_AREA_LEVEL_2 | 국가 수준 아래 두 번째 행정 독립체를 나타냅니다. |
| COUNTRY | 국가의 정치적 독립체를 나타냅니다. |
| DATASET | 참고: v=beta 채널에서만 사용할 수 있습니다. 서드 파티 데이터 세트를 나타냅니다. |
| LOCALITY | 도시 또는 마을로 통합된 정치적 독립체를 나타냅니다. |
| POSTAL_CODE | 국가 내에서 우편물을 보낼 때 사용되는 우편번호를 나타냅니다. 우편번호를 포함합니다. |
| SCHOOL_DISTRICT | 교육구를 나타냅니다. |

FeatureStyleOptions [https://developers.google.com/maps/documentation/javascript/reference/data-driven-styling?hl=ko#FeatureStyleOptions] 인터페이스

| 속성 |
| --- |

| fillOpacity optional | 유형:  number optional 0.0과 1.0 사이의 채우기 불투명도입니다. 다각형 도형에만 적용됩니다. |

| strokeOpacity optional | 유형:  number optional 0.0과 1.0 사이의 획 불투명도입니다. 선 및 다각형 도형에만 적용됩니다. |
| strokeWeight optional | 유형:  number optional 픽셀 단위의 스트로크 너비입니다. 선 및 다각형 도형에만 적용됩니다. |

장소 ID가 있는 지형지물을 나타내는 인터페이스로, FeatureType.ADMINISTRATIVE_AREA_LEVEL_1 [https://developers.google.com/maps/documentation/javascript/reference/data-driven-styling?hl=ko#FeatureType.ADMINISTRATIVE_AREA_LEVEL_1], FeatureType.ADMINISTRATIVE_AREA_LEVEL_2 [https://developers.google.com/maps/documentation/javascript/reference/data-driven-styling?hl=ko#FeatureType.ADMINISTRATIVE_AREA_LEVEL_2], FeatureType.COUNTRY [https://developers.google.com/maps/documentation/javascript/reference/data-driven-styling?hl=ko#FeatureType.COUNTRY], FeatureType.LOCALITY [https://developers.google.com/maps/documentation/javascript/reference/data-driven-styling?hl=ko#FeatureType.LOCALITY], FeatureType.POSTAL_CODE [https://developers.google.com/maps/documentation/javascript/reference/data-driven-styling?hl=ko#FeatureType.POSTAL_CODE], FeatureType.SCHOOL_DISTRICT [https://developers.google.com/maps/documentation/javascript/reference/data-driven-styling?hl=ko#FeatureType.SCHOOL_DISTRICT] 유형의 지형지물이 포함됩니다.
이 인터페이스는 Feature [https://developers.google.com/maps/documentation/javascript/reference/data-driven-styling?hl=ko#Feature]를 확장합니다.

| 속성 |
| --- |
| placeId | 유형:  string PlaceResult.place_id |
| 상속: featureType |
| 메서드 |
| --- |
| fetchPlace | fetchPlace() 매개변수: 없음 반환 값:  Promise<Place> 이 PlaceFeature의 Place를 가져옵니다. 결과 Place 객체에서 id 및 displayName 속성이 채워집니다. 표시 이름은 최종 사용자가 지도에 보는 언어로 표시됩니다. 일반 Places API 사용 설정 및 결제에 따라 Place.fetchFields()를 통해 추가 필드를 요청할 수 있습니다. 동기식 FeatureStyleFunctions만 지원되므로 FeatureStyleFunction에서 호출하지 마세요. Place 가져오기에 오류가 있으면 프로미스가 거부됩니다. |
