시작하기 전에:
Time Zone API를 사용하려면 결제 계정이 있고

Time Zone API가

사용 설정된 프로젝트가 필요합니다. 프로젝트 소유자 및 결제 관리자를 여러 명 만들어
이러한 역할을 맡은 담당자가 팀에 상주하도록 하는 것이 좋습니다. 자세한 내용은 Cloud 콘솔에서 설정하기 [https://developers.google.com/maps/documentation/timezone/cloud-setup?hl=ko]를 참고하세요.

Time Zone API는 지구 표면의 위치에 대한 시간 오프셋 데이터를 제공합니다. 특정 위도/경도 쌍 및 날짜의 시간대 정보를 요청합니다.
API는 해당 시간대의 이름, UTC와의 시간 오프셋, 일광 절약 시간 오프셋을 반환합니다.

샘플 요청 및 응답

요청을 URL 문자열로 구성하고, 위도/경도 좌표를 사용하여 location를 지정하고, timestamp를 사용하여 날짜를 지정하고, API 키를 사용하여 HTTPS 인터페이스를 통해 Time Zone API에 액세스합니다.

아래 쿼리는 미국 네바다에 대해 시간대 요청을 수행합니다. 타임스탬프는 2012년 3월 8일로 설정되어 있습니다.

URL

https://maps.googleapis.com/maps/api/timezone/json
?location=39.6034810%2C-119.6822510
&timestamp=1733428634
&key=YOUR_API_KEY

cURL

curl -L -X GET 'https://maps.googleapis.com/maps/api/timezone/json?location=39.6034810%2C-119.6822510&timestamp=1733428634&key=YOUR_API_KEY'

웹브라우저에 URL을 입력하여 테스트할 수 있습니다 (YOUR_API_KEY를 실제 API 키 [https://developers.google.com/maps/documentation/timezone/get-started?hl=ko#get-a-key]로 바꾸어야 함).
응답에는 지정된 위치 및 날짜((timestamp))의 시간대 데이터가 포함됩니다.
요청 URL 및 사용 가능한 매개변수 빌드 [https://developers.google.com/maps/documentation/timezone/overview?hl=ko#requests] 및 응답 이해 [https://developers.google.com/maps/documentation/timezone/overview?hl=ko#TimeZoneResponse]에 관한 자세한 내용은 개발자 가이드 [https://developers.google.com/maps/documentation/timezone/overview?hl=ko]를 참고하세요.

클라이언트 라이브러리로 코딩 시작하기

클라이언트 라이브러리는 인증, 요청 제한, 자동 재시도와 같은 일반적인 작업을 위한 간단한 네이티브 구현을 제공하여 Google 지도 웹 서비스 API로 개발하는 것을 더 쉽게 만듭니다. Time Zone API는 Google 지도 서비스용 Java 클라이언트, Python 클라이언트, Go 클라이언트, Node.js 클라이언트 [https://developers.google.com/maps/documentation/timezone/client-library?hl=ko]에서 사용할 수 있습니다.

인증, 할당량, 가격 책정, 정책

인증

Time Zone API를 사용하려면 먼저 API를 사용 설정하고 적절한 인증 사용자 인증 정보를 가져와야 합니다. 자세한 내용은 Google Maps Platform 시작하기 [https://developers.google.com/maps/get-started?hl=ko]를 참고하세요.

할당량 및 가격 책정

Time Zone API에 설정된 할당량 및 가격에 관한 자세한 내용은 사용량 및 결제 [https://developers.google.com/maps/documentation/timezone/usage-limits?hl=ko] 페이지를 참고하세요.

정책

Time Zone API는 API 정책 [https://developers.google.com/maps/documentation/timezone/policies?hl=ko]에 따라 사용해야 합니다.

자세히 알아보기

Time Zone API에서 언어 매개변수를 설정하여 기본값인 영어가 아닌 언어로 결과를 반환할 수도 있습니다. 자세한 예시와 기타 세부정보는 Time Zone API 개발자 가이드 [https://developers.google.com/maps/documentation/timezone/overview?hl=ko]를 참고하세요.

Time Zone API 개발자 가이드 [https://developers.google.com/maps/documentation/timezone/overview?hl=ko]는 Google Maps Platform API 중 하나에서 제공하는 지도에 시간 데이터를 포함하려는 웹사이트 및 모바일 개발자를 대상으로 합니다. API 사용에 관한 소개와 사용 가능한 매개변수에 관한 참조 자료를 제공합니다.
