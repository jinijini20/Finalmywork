Android용 Navigation SDK는 Android용 Maps SDK에 탐색 기능을 추가합니다. Google 지도 기반 Android 애플리케이션에 탐색 기능이 필요한 경우 Android용 Navigation SDK를 사용해야 합니다.
하지만 Android용 Maps SDK의 대부분의 함수는 Android용 Navigation SDK에서 동일하게 작동합니다. 어떤 SDK 종속 항목을 사용하든 동일한 com.google.android.gms.maps 패키지를 사용하여 액세스합니다. 즉, 앱이 이전에 Android용 Maps SDK에 종속된 경우 앱의 기존 기능에 영향을 주지 않고 Android용 Navigation SDK로 종속 항목을 변경할 수 있습니다. 단, 다음 두 가지 예외가 있습니다.
예외를 발생시키는 API
NavSDK에서 작동하지 않고 호출 시 영향을 미치지 않는 API
이러한 예외는 아래에 설명되어 있습니다.
예외를 발생시키는 API
애플리케이션에서 다음 함수를 사용 설정하면 예외가 발생합니다.
LiteMode [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/GoogleMapOptions?hl=ko#liteMode(boolean)]
Navigation SDK에 기능이 없는 API
다음 API는 Navigation SDK에 기능이 없지만 코드가 이를 호출해도 영향을 미치지 않습니다. 다음 카테고리로 분류됩니다.
웨어러블 활동 API
Android용 Navigation SDK에서 이 기능을 사용하려면 TBT 피드 [https://developers.google.com/maps/documentation/navigation/android-sdk/tbt-feed?hl=ko]를 사용 설정하세요.
기타 API
웨어러블용 API
GoogleMapOptions.ambientEnabled(boolean enabled) [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/GoogleMapOptions?hl=ko#ambientEnabled(boolean)]를 사용하면 일반적으로 웨어러블 앱의 대기 모드 [https://developers.google.com/maps/documentation/android-sdk/wear?hl=ko#ambient]를 사용 중지하거나 사용 설정할 수 있습니다. 또한 MapView [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/MapView?hl=ko] 및 MapFragment [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/MapFragment?hl=ko]에는 onEnterAmbient [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/MapView?hl=ko#onEnterAmbient(android.os.Bundle)] 및 onExitAmbient [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/MapView?hl=ko#onExitAmbient()] 메서드가 있지만 호출해도 영향을 미치지 않습니다. 웨어러블 기기의 경우 TBT 피드 [https://developers.google.com/maps/documentation/navigation/android-sdk/tbt-feed?hl=ko]를 대신 사용하세요.
기타 API
NavSDK에서는 맞춤 LocationSource 설정 [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/GoogleMap?hl=ko#setLocationSource(com.google.android.gms.maps.LocationSource)]이 아무런 영향을 미치지 않습니다.
모든 MapsInitializer [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/MapsInitializer?hl=ko] API
RuntimeRemoteException [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/RuntimeRemoteException?hl=ko]을 발생시킵니다.
모빌리티 서비스 고객만 사용하도록 설계된 API
Navigation SDK에는 Google에서 거래당 청구하는 모빌리티 서비스 [https://developers.google.com/maps/documentation/transportation-logistics/mobility?hl=ko] 고객만 사용할 수 있는 API가 여러 개 있습니다. 모빌리티 서비스 고객이 아닌 경우 다음 메서드는 no-op입니다.
