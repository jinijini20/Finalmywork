Polygon [https://developers.google.com/maps/documentation/javascript/reference/3.58/polygon?hl=ko#Polygon] 클래스

google.maps.Polygon 클래스

다각형 (다중선과 마찬가지로)은 순서가 지정된 일련의 연결된 좌표를 정의합니다. 또한 다각형은 닫힌 루프를 형성하고 채워진 영역을 정의합니다. 개발자 가이드의 샘플을 참고하세요. 단순 다각형 [https://developers.google.com/maps/documentation/javascript/examples/polygon-simple?hl=ko], 구멍이 있는 다각형 [https://developers.google.com/maps/documentation/javascript/examples/polygon-hole?hl=ko] 등 다양한 샘플을 확인할 수 있습니다. 데이터 영역 [https://developers.google.com/maps/documentation/javascript/reference/3.58/polygon?hl=ko#Data.Polygon]을 사용하여 다각형을 만들 수도 있습니다. 데이터 영역은 안쪽과 바깥쪽 경로의 순서를 자동으로 처리하므로 구멍을 더 간단하게 만들 수 있습니다.
이 클래스는 MVCObject [https://developers.google.com/maps/documentation/javascript/reference/3.58/event?hl=ko#MVCObject]를 확장합니다.

const {Polygon} = await google.maps.importLibrary("maps")를 호출하여 액세스합니다. Maps JavaScript API의 라이브러리 [https://developers.google.com/maps/documentation/javascript/libraries?hl=ko]를 참고하세요.
