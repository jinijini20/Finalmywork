WebGLOverlayView [https://developers.google.com/maps/documentation/javascript/reference/3.58/webgl?hl=ko#WebGLOverlayView] 클래스

google.maps.WebGLOverlayView 클래스

WebGL 오버레이 뷰를 사용하면 Google Maps Platform에서 벡터 기본 지도를 렌더링하는 데 사용하는 동일한 WebGL 렌더링 컨텍스트에 직접 액세스할 수 있습니다. 이와 같이 공유된 렌더링 컨텍스트를 사용하면 3D 빌딩 도형의 깊이 오클루전 및 2D/3D 콘텐츠를 기본 지도 렌더링과 동기화할 수 있는 기능과 같은 이점이 있습니다. WebGL 오버레이 뷰를 사용하면 WebGL을 직접 사용하거나 Three.js 또는 deck.gl과 같은 인기 그래픽 라이브러리를 사용하여 지도에 콘텐츠를 추가할 수 있습니다. 오버레이를 사용하려면 google.maps.WebGLOverlayView를 확장하고 다음 수명 주기 후크(WebGLOverlayView.onAdd [https://developers.google.com/maps/documentation/javascript/reference/3.58/webgl?hl=ko#WebGLOverlayView.onAdd], WebGLOverlayView.onContextRestored [https://developers.google.com/maps/documentation/javascript/reference/3.58/webgl?hl=ko#WebGLOverlayView.onContextRestored], WebGLOverlayView.onDraw [https://developers.google.com/maps/documentation/javascript/reference/3.58/webgl?hl=ko#WebGLOverlayView.onDraw], WebGLOverlayView.onContextLost [https://developers.google.com/maps/documentation/javascript/reference/3.58/webgl?hl=ko#WebGLOverlayView.onContextLost], WebGLOverlayView.onRemove [https://developers.google.com/maps/documentation/javascript/reference/3.58/webgl?hl=ko#WebGLOverlayView.onRemove]) 각각에 대한 구현을 제공하면 됩니다. 유효한 Map [https://developers.google.com/maps/documentation/javascript/reference/3.58/map?hl=ko#Map] 객체를 사용하여 WebGLOverlayView.setMap [https://developers.google.com/maps/documentation/javascript/reference/3.58/webgl?hl=ko#WebGLOverlayView.setMap]를 호출하여 onAdd() 메서드 호출을 트리거하고 setMap(null)를 호출하여 onRemove() 메서드를 트리거해야 합니다. setMap() 메서드는 생성 시 또는 덮기를 삭제한 후 다시 표시해야 하는 시점에 호출할 수 있습니다. 그러면 확대/축소, 중심, 지도 유형과 같이 요소의 위치를 변경할 수 있는 지도 속성이 변경될 때마다 onDraw() 메서드가 호출됩니다. WebGLOverlayView는 MapOptions.mapId [https://developers.google.com/maps/documentation/javascript/reference/3.58/map?hl=ko#MapOptions.mapId]가 있는 벡터 지도에만 추가할 수 있습니다 (RenderingType.VECTOR [https://developers.google.com/maps/documentation/javascript/reference/3.58/map?hl=ko#RenderingType.VECTOR] MapOptions.renderingType [https://developers.google.com/maps/documentation/javascript/reference/3.58/map?hl=ko#MapOptions.renderingType]로 설정되고 Map.DEMO_MAP_ID [https://developers.google.com/maps/documentation/javascript/reference/3.58/map?hl=ko#Map.DEMO_MAP_ID]를 MapOptions.mapId [https://developers.google.com/maps/documentation/javascript/reference/3.58/map?hl=ko#MapOptions.mapId]로 사용하는 지도 포함).
이 클래스는 MVCObject [https://developers.google.com/maps/documentation/javascript/reference/3.58/event?hl=ko#MVCObject]를 확장합니다.

const {WebGLOverlayView} = await google.maps.importLibrary("maps")를 호출하여 액세스합니다. Maps JavaScript API의 라이브러리 [https://developers.google.com/maps/documentation/javascript/libraries?hl=ko]를 참고하세요.
