PlaceContentConfigElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceContentConfigElement]
class

google.maps.places.PlaceContentConfigElement
class

Configures a PlaceDetailsCompactElement [https://developers.google.com/maps/documentation/javascript/reference/places-widget#PlaceDetailsCompactElement], PlaceDetailsElement [https://developers.google.com/maps/documentation/javascript/reference/places-widget#PlaceDetailsElement] or PlaceSearchElement [https://developers.google.com/maps/documentation/javascript/reference/places-widget#PlaceSearchElement] to show a custom set of content. Append this element as a child to use it.   For PlaceDetailsCompactElement [https://developers.google.com/maps/documentation/javascript/reference/places-widget#PlaceDetailsCompactElement], PlaceDetailsElement [https://developers.google.com/maps/documentation/javascript/reference/places-widget#PlaceDetailsElement] or PlaceSearchElement [https://developers.google.com/maps/documentation/javascript/reference/places-widget#PlaceSearchElement], append any of the following elements to the PlaceContentConfigElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceContentConfigElement] to show the corresponding content:  PlaceAddressElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceAddressElement], PlaceAccessibleEntranceIconElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceAccessibleEntranceIconElement], PlaceAttributionElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceAttributionElement] PlaceMediaElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceMediaElement], PlaceOpenNowStatusElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceOpenNowStatusElement], PlacePriceElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlacePriceElement], PlaceRatingElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceRatingElement], PlaceTypeElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceTypeElement].   Specific to PlaceDetailsElement [https://developers.google.com/maps/documentation/javascript/reference/places-widget#PlaceDetailsElement], you may also append any of the following elements:  PlaceFeatureListElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceFeatureListElement] PlaceOpeningHoursElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceOpeningHoursElement], PlacePhoneNumberElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlacePhoneNumberElement], PlacePlusCodeElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlacePlusCodeElement], PlaceReviewsElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceReviewsElement], PlaceSummaryElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceSummaryElement], PlaceTypeSpecificHighlightsElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceTypeSpecificHighlightsElement], PlaceWebsiteElement [https://developers.google.com/maps/documentation/javascript/reference/place-widget-child-elements#PlaceWebsiteElement].   The order of the children does not matter; the element renders content in a standard order which is not customizable. Example:  <gmp-place-details>   <gmp-place-content-config>     <gmp-place-media lightbox-preferred></gmp-place-media>     <gmp-place-address></gmp-place-address>   </gmp-place-content-config> </gmp-place-details>
Custom element:
<gmp-place-content-config></gmp-place-content-config>
