This example map shows the boundaries for counties near Denver, CO
(Administrative Area Level 2), and has an event handler that styles the clicked
polygon and displays an info window with the event data.

Read the
documentation [https://developers.google.com/maps/documentation/javascript/dds-boundaries/handle-events].

Git and Node.js are required to run this sample locally. Follow these instructions [https://docs.npmjs.com/downloading-and-installing-node-js-and-npm] to install Node.js and NPM. The following commands clone, install dependencies and start the sample application.

git clone -b sample-boundaries-click https://github.com/googlemaps/js-samples.git
cd js-samples
npm i
npm start
