StreetViewService [https://developers.google.com/maps/documentation/javascript/reference/3.58/street-view-service?hl=ko#StreetViewService] 클래스

google.maps.StreetViewService 클래스

StreetViewService 객체는 스트리트 뷰 데이터 검색을 실행합니다.
const {StreetViewService} = await google.maps.importLibrary("streetView")를 호출하여 액세스합니다. Maps JavaScript API의 라이브러리 [https://developers.google.com/maps/documentation/javascript/libraries?hl=ko]를 참고하세요.
| 생성자 |
| --- |
| StreetViewService | StreetViewService() 매개변수: 없음 스트리트 뷰 서비스에 저장된 데이터에 대한 인터페이스를 제공하는 StreetViewService를 만듭니다. |
| 메서드 |
| --- |
| getPanorama | getPanorama(request[, callback]) 매개변수:  request:  StreetViewLocationRequest\|StreetViewPanoRequest callback:  function(StreetViewPanoramaData optional, StreetViewStatus): void optional 반환 값:  Promise<StreetViewResponse> 제공된 스트리트 뷰 쿼리 요청과 일치하는 파노라마의 StreetViewPanoramaData를 가져옵니다. StreetViewPanoramaData는 제공된 콜백에 전달됩니다. |

StreetViewStatus [https://developers.google.com/maps/documentation/javascript/reference/3.58/street-view-service?hl=ko#StreetViewStatus] 상수

google.maps.StreetViewStatus 상수

스트리트 뷰 요청이 완료될 때 StreetViewService에서 반환한 상태입니다. 값으로 지정하거나 상수의 이름을 사용하여 지정할 수 있습니다. 예를 들어 'OK' 또는 google.maps.StreetViewStatus.OK입니다.
const {StreetViewStatus} = await google.maps.importLibrary("streetView")를 호출하여 액세스합니다. Maps JavaScript API의 라이브러리 [https://developers.google.com/maps/documentation/javascript/libraries?hl=ko]를 참고하세요.
| 상수 |
| --- |
| OK | 요청이 성공했습니다. |
| UNKNOWN_ERROR | 요청을 처리하지 못했으며 실패에 대한 정확한 원인을 알 수 없습니다. |
| ZERO_RESULTS | 검색 기준과 일치하는 파노라마를 찾을 수 없습니다. |

StreetViewLocationRequest [https://developers.google.com/maps/documentation/javascript/reference/3.58/street-view-service?hl=ko#StreetViewLocationRequest] 인터페이스

google.maps.StreetViewLocationRequest 인터페이스

getPanorama와 함께 전송되는 스트리트 뷰 요청입니다. StreetViewLocationRequest를 사용하면 지정된 위치에서 스트리트 뷰 파노라마를 검색할 수 있습니다.
| 속성 |
| --- |
| location optional | 유형:  LatLng\|LatLngLiteral optional 스트리트 뷰 파노라마를 검색할 위치를 지정합니다. |
| preference optional | 유형:  StreetViewPreference optional 반경 내에서 파노라마를 찾아야 하는 환경설정(제공된 위치에 가장 가까운 파노라마 또는 반경 내에서 가장 좋은 파노라마)을 지정합니다. |
| radius optional | 유형:  number optional 기본값: 50 파노라마를 검색할 반경을 미터 단위로 설정합니다. |
| source optional | 지원 중단됨: 대신 sources를 사용하세요. 유형:  StreetViewSource optional 기본값: StreetViewSource.DEFAULT 검색할 파노라마의 출처를 지정합니다. 이렇게 하면 예를 들어 야외 파노라마만 검색하도록 제한할 수 있습니다. |
| sources optional | 유형:  Iterable<StreetViewSource> optional 기본값: [StreetViewSource.DEFAULT] 검색할 파노라마의 소스를 지정합니다. 이렇게 하면 예를 들어 야외 파노라마만 검색하도록 제한할 수 있습니다. 소스를 여러 개 설정하면 이러한 소스의 교집합으로 평가됩니다. |

StreetViewPreference [https://developers.google.com/maps/documentation/javascript/reference/3.58/street-view-service?hl=ko#StreetViewPreference] 상수

google.maps.StreetViewPreference 상수

요청 위치에 가장 가까운 스트리트 뷰 파노라마 또는 사용자가 보고 싶어 할 가능성이 가장 높은 파노라마를 반환하도록 검색 결과에 편향이 적용되는 옵션입니다. 값으로 지정하거나 상수의 이름을 사용하여 지정합니다. 예를 들면 'best' 또는 google.maps.StreetViewPreference.BEST입니다.
const {StreetViewPreference} = await google.maps.importLibrary("streetView")를 호출하여 액세스합니다. Maps JavaScript API의 라이브러리 [https://developers.google.com/maps/documentation/javascript/libraries?hl=ko]를 참고하세요.
| 상수 |
| --- |
| BEST | 사용자가 보려는 스트리트 뷰 파노라마일 가능성이 가장 높은 스트리트 뷰 파노라마를 반환합니다. 최적의 결과는 사용자 조사 및 인식된 관심 장소, 이미지 품질, 지정된 위치와의 거리와 같은 매개변수를 기반으로 알고리즘에 의해 결정됩니다. |
| NEAREST | 제공된 위치에서 가장 가까운 거리에 있는 스트리트 뷰 파노라마를 반환합니다. 이 방법은 제한된 반경 내에서만 잘 작동합니다. 권장 반경은 1km 이하입니다. |

StreetViewSource [https://developers.google.com/maps/documentation/javascript/reference/3.58/street-view-service?hl=ko#StreetViewSource] 상수

google.maps.StreetViewSource 상수

스트리트 뷰 검색을 선택한 출처로 제한하는 식별자입니다. 이러한 값은 문자열로 지정됩니다. 예를 들면 'outdoor'입니다.
const {StreetViewSource} = await google.maps.importLibrary("streetView")를 호출하여 액세스합니다. Maps JavaScript API의 라이브러리 [https://developers.google.com/maps/documentation/javascript/libraries?hl=ko]를 참고하세요.
| 상수 |
| --- |
| DEFAULT | 스트리트 뷰의 기본 출처를 사용합니다. 검색은 특정 출처로 제한되지 않습니다. |
| GOOGLE | 스트리트 뷰 검색을 공식 Google 컬렉션으로 제한합니다. |
| OUTDOOR | 스트리트 뷰 검색을 야외 컬렉션으로 제한합니다. 실내 컬렉션은 검색 결과에 포함되지 않습니다. 또한 실내 또는 실외인지 확인할 수 있는 파노라마만 검색 결과로 반환됩니다. 예를 들어 PhotoSphere은 실내인지 실외인지 알 수 없으므로 반환되지 않습니다. |
