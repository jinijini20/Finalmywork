Maps Static API
다른 코드 없이 간단한 HTTP 요청을 사용하여 웹페이지에 Google 지도 이미지를 삽입합니다.
시작하기
Maps Static API를 사용해 지도 작성을 시작합니다.
explore
Google Maps Platform 시작하기
Google Maps Platform 시작 가이드에 따라 계정을 만들고, API 키를 생성하고, 빌드를 시작합니다.
map
Maps Static API 이미지 삽입
<img> 태그를 사용하여 지도에 Maps Static API 이미지를 추가합니다.
palette
지도 스타일 맞춤설정
도로, 지형 지물, 관심 장소 등 지도의 모든 측면을 맞춤설정합니다.
location_on
지도에 마커 및 아이콘 추가
지도에 마커를 추가하고, 마커의 크기와 색상을 맞춤설정하고, 나만의 맞춤 이미지를 사용할 수 있습니다.
기능
Maps Static API의 핵심 기능에 대한 문서를 찾아봅니다.
지도 위치
지도의 중심을 맞추기 위해 위도/경도 좌표를 지정합니다.
마커
지도에 하나 이상의 마커 집합을 표시합니다.
맞춤 스타일 지정
지도에 자체 맞춤 스타일을 적용합니다.
지도 유형
도로, 위성, 하이브리드, 지형, 맞춤 지도를 표시합니다.
경로
지도에서 지점 사이에 맞춤설정 가능한 경로를 그립니다.
맞춤 표시 영역
표시되는 위치를 지정하여 표시 영역을 정의합니다.

Stack Overflow

Issue Tracker
