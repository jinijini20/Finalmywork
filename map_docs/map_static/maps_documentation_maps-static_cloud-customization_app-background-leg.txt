앱 배경 색상 변경
플랫폼 선택: Android iOS JavaScript 웹 서비스
지원 대상: 이 기능은 Android, iOS, Maps JavaScript API, Maps Static에서 볼 수 있습니다.
지도 컨테이너의 배경 색상을 변경할 수 있습니다. 이 색상은 타일이 로드되기 전에 표시됩니다.
앱 배경 색상 설정
지도 지형지물 창에서 톱니바퀴 아이콘 을 선택하여 설정 창을 엽니다.
색상 선택 도구를 사용하거나 16진수 문자열을 입력하여 앱 배경색을 설정합니다. 색상을 설정하면 선택한 배경색을 표시하기 위해 타일이 새로고침됩니다.
