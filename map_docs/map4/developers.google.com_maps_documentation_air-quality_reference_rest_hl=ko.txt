서비스: airquality.googleapis.com
이 서비스를 호출하려면 Google에서 제공하는 클라이언트 라이브러리 [https://cloud.google.com/apis/docs/client-libraries-explained?hl=ko]를 사용하는 것이 좋습니다. 애플리케이션에서 자체 라이브러리를 사용하여 이 서비스를 호출해야 한다면 API를 요청할 때 다음 정보를 사용하세요.

서비스 엔드포인트
서비스 엔드포인트 [https://cloud.google.com/apis/design/glossary?hl=ko#api_service_endpoint]는 API 서비스의 네트워크 주소를 지정하는 기본 URL입니다. 하나의 서비스에 여러 서비스 엔드포인트가 있을 수 있습니다. 이 서비스에는 다음 서비스 엔드포인트가 포함되고, 그 아래의 모든 URI가 이 서비스 엔드포인트와 관련됩니다.

https://airquality.googleapis.com

REST 리소스: v1.currentConditions [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/currentConditions?hl=ko]

| 메서드 |
| --- |
| lookup | POST /v1/currentConditions:lookup 현재 조건 엔드포인트는 100개가 넘는 국가의 시간별 공기질 정보를 최대 500x500m 해상도로 제공합니다. |

REST 리소스: v1.forecast [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/forecast?hl=ko]

| 메서드 |
| --- |
| lookup | POST /v1/forecast:lookup 지정된 기간 동안 특정 위치의 공기질 예보를 반환합니다. |

REST 리소스: v1.history [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/history?hl=ko]

| 메서드 |
| --- |
| lookup | POST /v1/history:lookup 지정된 기간 동안 특정 위치의 공기질 기록을 반환합니다. |

REST 리소스: v1.mapTypes.heatmapTiles [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/mapTypes.heatmapTiles?hl=ko]

| 메서드 |
| --- |
| lookupHeatmapTile | GET /v1/mapTypes/{mapType}/heatmapTiles/{zoom}/{x}/{y} 타일 PNG 이미지의 데이터가 포함된 바이트 배열을 반환합니다. |
