타일 PNG 이미지의 데이터를 포함하는 바이트 배열을 반환합니다.

HTTP 요청
GET https://pollen.googleapis.com/v1/mapTypes/{mapType}/heatmapTiles/{zoom}/{x}/{y}URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.

경로 매개변수
| 매개변수 |
| --- |
| mapType | enum (MapType) 필수 항목입니다. 꽃가루 히트맵의 유형입니다. 지도에서 그래픽으로 나타낼 꽃가루 유형 및 색인의 조합을 정의합니다. |
| zoom | integer 필수 항목입니다. 지도의 확대/축소 수준입니다. 지도뷰에 표시되는 지도 콘텐츠의 크기 또는 크기를 정의합니다. 확대/축소 수준 0은 전 세계를 단일 타일로 표시합니다. 확대/축소 수준 1은 4개의 타일로 전 세계를 나타냅니다. 확대/축소 수준 2는 16개의 타일로 전 세계를 나타냅니다. 확대/축소 수준 16은 전 세계 65,536개의 타일로 표현됩니다. 허용되는 값: 0~16 |
| x | integer 필수 항목입니다. 요청된 타일의 동서 지점을 정의합니다. |
| y | integer 필수 항목입니다. 요청된 타일의 남북 지점을 정의합니다. |

요청 본문
요청 본문은 비어 있어야 합니다.
