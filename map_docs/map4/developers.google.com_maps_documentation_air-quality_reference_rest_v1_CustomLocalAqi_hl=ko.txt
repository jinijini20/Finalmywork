내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

JSON 표현 [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/CustomLocalAqi?hl=ko#SCHEMA_REPRESENTATION]

'국가/지역을 AQI로' 표현 관계입니다. 해당 국가/지역에 필요한 공기질 데이터가 선택한 AQI에 따라 표시되도록 국가/지역을 원하는 AQI와 페어링합니다.

| JSON 표현 |
| --- |
| {   "regionCode": string,   "aqi": string } |

| 필드 |
| --- |
| regionCode | string 맞춤 AQI가 필요한 국가/지역입니다. ISO 3166-1 alpha-2 코드를 사용하여 값을 제공해야 합니다. |
| aqi | string 국가/지역을 연결할 AQI입니다. 값은 유효한 색인 코드여야 합니다. |
