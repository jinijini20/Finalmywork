forecast [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/forecast/lookup?hl=ko] 엔드포인트는 특정 위치의 시간별 공기질 예측 정보를 최대 96시간 (4일) 동안 제공합니다. 각 예측에는 시간당 공기질 지수와 추가 정보가 포함됩니다.

예측의 특정 시간 및 날짜 또는 기간
번역된 필드의 언어 코드
지역 또는 맞춤 공기질 지수 (AQI)
건강 권장사항
주요 오염 물질
오염 물질 농도
오염 물질에 대한 추가 정보
응답 페이지 크기
색상 팔레트

API 탐색기를 사용하면 API 및 API 옵션을 익히기 위해 실시간으로 요청할 수 있습니다.

사용해 보기 [https://developers.google.com/maps/documentation/air-quality/forecast?hl=ko#try_it]

공기질 예측을 가져오려면 아래 기본 요청 예 [https://developers.google.com/maps/documentation/air-quality/forecast?hl=ko#example_of_basic_request]와 같이 dateTime 매개변수를 향후 날짜 및 시간으로 설정하세요.

예측 요청 정보

다음 주소로 HTTP POST 요청을 전송하여 예측 엔드포인트를 사용하여 시간별 공기질 예측 정보를 요청할 수 있습니다.

https://airquality.googleapis.com/v1/forecast:lookup?key=YOUR_API_KEY

요청 본문에 매개변수 [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/forecast/lookup?hl=ko#request-body]를 포함합니다. 요청 본문에는 공기질 시간당 예측 정보를 가져올 위치와 응답에 포함할 추가 정보를 제어하는 다양한 옵션이 포함됩니다.

예측 응답 정보

응답 본문은 대기질 예측 정보, 대기질 지수, 맞춤설정된 추가 정보를 표시할 UTC 날짜 및 시간을 제공합니다. 응답 본문에는 정보가 요청된 페이지 크기에 맞지 않는 경우 다음 페이지를 검색하는 토큰도 포함됩니다.
