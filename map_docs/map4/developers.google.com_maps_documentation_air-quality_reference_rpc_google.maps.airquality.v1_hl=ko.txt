AirQualityIndex

다양한 공기질 측정항목을 나타내는 기본 객체입니다. 이러한 측정항목을 종합하면 현재 공기질 상태에 대한 개요를 제공합니다. 전 세계에는 공기질의 다양한 측면을 측정하는 데 관심이 있는 다양한 목적과 그룹을 제공하는 여러 지수가 있습니다.

| 필드 |
| --- |
| code | string 색인의 코드입니다. 이 필드는 공백 대신 스네이크 케이스를 사용하여 프로그래밍 목적으로 색인을 나타냅니다. 예: 'uaqi', 'fra_atmo'. |
| display_name | string 인간이 읽을 수 있는 색인 이름 표현입니다. 예: 'AQI(미국)' |
| aqi_display | string 색인 숫자 점수의 텍스트 표현으로, 접두사 또는 접미사 기호가 포함될 수 있으며 일반적으로 최악의 색인 점수를 나타냅니다. 예: >100 또는 10+ 참고: 이 필드는 색인 점수를 표시할 때 사용해야 합니다. 숫자가 아닌 색인의 경우 이 필드는 비어 있습니다. |
| color | Color AQI 숫자 점수를 나타내는 데 사용되는 색상입니다. |
| category | string 색인 숫자 점수 해석의 텍스트 분류입니다. 예: '공기질 매우 우수' |
| dominant_pollutant | string 주요 오염 물질의 화학 기호입니다. 예: 'CO' |
| aqi | int32 색인의 숫자 점수입니다. 예: 10, 100 이 값은 정규화되지 않으며 관련 공기질 지수의 맥락에서만 해석해야 합니다. 숫자가 아닌 색인의 경우 이 필드가 반환되지 않습니다. 참고: 이 필드는 계산, 그래프 표시 등에 사용해야 합니다. 지수 점수를 표시하려면 AQI 표시 필드를 사용해야 합니다. |

ColorPalette

'세계 공기질 지수'에서 제공하는 데이터에 사용되는 색상 팔레트를 결정합니다. (UAQI). 이 색상 팔레트는 UAQI에만 해당하며, 다른 AQI에는 제어할 수 없는 미리 정해진 색상 팔레트가 있습니다.

| 열거형 |
| --- |
| COLOR_PALETTE_UNSPECIFIED | 기본값입니다. 매개변수로 전달되면 무시됩니다. |
| RED_GREEN | 빨간색/녹색 팔레트를 사용할지 여부를 결정합니다. |
| INDIGO_PERSIAN_DARK | 인디고/페르시아 팔레트(어두운 테마)를 사용할지 결정합니다. |
| INDIGO_PERSIAN_LIGHT | 인디고/페르시아 팔레트(밝은 테마)를 사용할지 결정합니다. |

CustomLocalAqi

'국가/지역 - AQI' 관계를 표현합니다. 해당 국가/지역에 필요한 공기질 데이터가 선택한 AQI에 따라 표시되도록 국가/지역을 원하는 AQI와 페어링합니다.

| 필드 |
| --- |
| region_code | string 맞춤 AQI가 필요한 국가/지역입니다. ISO 3166-1 alpha-2 코드를 사용하여 값을 제공해야 합니다. |
| aqi | string 국가/지역을 연결할 AQI입니다. 값은 유효한 색인 코드여야 합니다. |

ExtraComputation

선택적으로 사용 설정할 수 있는 추가 기능 추가 계산을 지정하면 관련 요소와 필드가 응답에 반환됩니다.

| 열거형 |
| --- |
| EXTRA_COMPUTATION_UNSPECIFIED | 기본값입니다. 이 매개변수가 매개변수로 전달되면 서버는 이를 무시합니다. |
| LOCAL_AQI | 응답에 요청된 위치 (국가)의 로컬 (국가) AQI를 포함할지 여부를 결정합니다. 지정하면 응답에 'air_quality_index'가 포함됩니다. 해당 위치의 로컬 AQI에 대한 모든 관련 데이터가 포함된 데이터 구조와 일치해야 합니다. |
| HEALTH_RECOMMENDATIONS | 응답에 현재 AQI 상태에 대한 건강상의 조언과 권장 조치가 포함될지 여부를 결정합니다. 권장 사항은 일반 인구와 일반 인구보다 오염 물질에 더 민감한 위험 그룹에 있는 6개 모집단에 맞게 조정됩니다. 지정된 경우 관련 데이터를 사용할 수 있을 때 응답에 health_recommendations 필드가 채워집니다. |
| POLLUTANT_ADDITIONAL_INFO | 응답에 각 오염 물질의 추가 정보를 포함할지 여부를 결정합니다. 지정된 경우 '색인'에 포함된 각 공기질 지수 객체입니다. 데이터를 사용할 수 있는 경우 필드 응답에 additional_info 필드가 포함됩니다. |
| DOMINANT_POLLUTANT_CONCENTRATION | 응답에 글로벌 또는 지역 색인에 따라 측정된 주요 오염 물질의 농도가 포함되는지 여부를 결정합니다. 요청에 글로벌 AQI와 로컬 AQI가 모두 지정된 경우 최대 2개의 오염 물질 코드가 반환될 수 있습니다. 지정된 경우 'pollutants' 목록에 포함된 주요 오염 물질 객체에 데이터를 사용할 수 있는 경우 concentration 필드가 포함됩니다. |
| POLLUTANT_CONCENTRATION | 응답에 전 세계 또는 지역 색인에 따라 사용 가능한 측정값이 포함된 모든 오염 물질의 농도가 포함되는지 확인합니다. 지정된 경우 데이터를 사용할 수 있는 경우 응답의 'pollutants' 필드에 포함된 각 오염 물질 객체에 concentration 필드가 포함됩니다. |

HealthRecommendations

다양한 인구 집단에 대한 건강 권장사항으로, 일반 텍스트 형식입니다. 추천은 관련된 공기질 상태를 기반으로 도출되었습니다.

| 필드 |
| --- |
| general_population | string 민감한 사항이 없습니다. |
| elderly | string 은퇴자 및 일반인보다 연령이 높은 사용자 |
| lung_disease_population | string 호흡기 관련 문제 및 천식에 시달리게 됩니다. |
| heart_disease_population | string 심장 및 순환계 질환입니다. |
| athletes | string 스포츠 및 기타 격렬한 야외 활동을 할 수 있습니다. |
| pregnant_women | string 임신 초기부터 말기까지 모든 임산부 |
| children | string 아동, 유아, 아기를 포함한 젊은 인구. |

LookupCurrentConditionsRequest

공기질 현재 상태의 요청 정의입니다.

| 필드 |
| --- |
| location | LatLng 필수 항목입니다. API에서 공기질 현재 상태 데이터를 찾는 경도 및 위도입니다. |
| extra_computations[] | ExtraComputation 선택사항입니다. 선택적으로 사용 설정할 수 있는 추가 기능입니다. 추가 계산을 지정하면 관련 요소와 필드가 응답에 반환됩니다. |
| uaqi_color_palette | ColorPalette 선택사항입니다. '세계 공기질 지수'에서 제공하는 데이터에 사용되는 색상 팔레트를 결정합니다. (UAQI). 이 색상 팔레트는 UAQI에만 관련이 있으며, 다른 AQI에는 제어할 수 없는 사전 지정된 색상 팔레트가 있습니다. |
| custom_local_aqis[] | CustomLocalAqi 선택사항입니다. '국가/지역을 AQI로' 표현 관계입니다. 해당 국가/지역에 필요한 공기질 데이터가 선택한 AQI에 따라 표시되도록 국가/지역을 원하는 AQI와 페어링합니다. 이 매개변수를 사용하면 특정 국가의 기본값이 아닌 AQI를 지정할 수 있습니다. 예를 들어 캐나다의 기본 지수가 아닌 미국 EPA 지수를 가져올 수 있습니다. |
| universal_aqi | bool 선택사항입니다. true로 설정하면 범용 AQI가 응답의 'indexes' 필드에 포함됩니다. 기본값은 true입니다. |
| language_code | string 선택사항입니다. 클라이언트가 응답 언어를 선택할 수 있습니다. 해당 언어로 데이터를 제공할 수 없는 경우 API는 가장 유사한 일치 항목을 사용합니다. 허용되는 값은 IETF 표준을 따릅니다. 기본값은 en입니다. |

LookupCurrentConditionsResponse

| 필드 |
| --- |
| date_time | Timestamp RFC3339 UTC 'Zulu'의 내림된 타임스탬프 형식으로, 나노초 해상도 및 최대 9자리 소수 자릿수로 구성됩니다. 예: '2014-10-02T15:00:00Z' |
| region_code | string 요청에 제공된 위치에 해당하는 국가/지역의 ISO_3166-1 alpha-2 코드입니다. 요청에 제공된 위치가 분쟁 지역인 경우 이 필드는 응답에서 생략될 수 있습니다. |

농도

공기 중 특정 오염 물질의 농도입니다.

| 필드 |
| --- |
| units | Unit 오염 농도를 측정하는 단위입니다. |
| value | float 오염 물질 농도의 값입니다. |

단위

오염 물질의 농도를 측정하는 데 사용되는 단위를 정의합니다. 특정 오염 물질의 기본 농도 단위를 확인하려면 보고된 오염 물질 [https://developers.google.com/maps/documentation/air-quality/pollutants?hl=ko#reported_pollutants]의 표를 참고하세요.

| 열거형 |
| --- |
| UNIT_UNSPECIFIED | 농도 단위가 지정되지 않았습니다. |
| PARTS_PER_BILLION | ppb (10억 명당 파트) 농도 단위 |
| MICROGRAMS_PER_CUBIC_METER | 'µg/m^3'(미터당 마이크로그램) 농도 단위입니다. |
