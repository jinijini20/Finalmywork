HTTP 요청
GET https://airquality.googleapis.com/v1/mapTypes/{mapType}/heatmapTiles/{zoom}/{x}/{y}URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.

경로 매개변수
| 매개변수 |
| --- |
| mapType | enum (MapType) 필수 항목입니다. 공기질 히트맵의 유형입니다. 지도에 그래픽으로 표시할 오염 물질을 정의합니다. Allowed values: - UAQI_RED_GREEN (UAQI, red-green palette) - UAQI_INDIGO_PERSIAN (UAQI, indigo-persian palette) - PM25_INDIGO_PERSIAN - GBR_DEFRA - DEU_UBA - CAN_EC - FRA_ATMO - US_AQI |
| zoom | integer 필수 항목입니다. 지도의 확대/축소 수준입니다. 지도 뷰에 지도 콘텐츠가 얼마나 크게 표시되는지 정의합니다. 확대/축소 수준 0은 전 세계를 단일 타일로 표시합니다. 확대/축소 수준 1은 4개의 타일로 전 세계를 나타냅니다. 확대/축소 수준 2는 16개 타일로 구성된 전 세계입니다. 확대/축소 수준 16은 65,536개의 타일로 구성된 전 세계입니다. 허용되는 값: 0~16 |
| x | integer 필수 항목입니다. 요청된 타일의 동서 지점을 정의합니다. |
| y | integer 필수 항목입니다. 요청된 타일의 남북 지점을 정의합니다. |

요청 본문
요청 본문은 비어 있어야 합니다.

응답 본문
PNG 형식의 히트맵 타일 이미지입니다.

승인 범위
다음 OAuth 범위가 필요합니다.

https://www.googleapis.com/auth/cloud-platform
자세한 내용은 OAuth 2.0 개요 [https://developers.google.com/identity/protocols/OAuth2?hl=ko]를 참고하세요.

지도 유형

공기질 히트맵의 유형입니다. 지도에 그래픽으로 나타낼 오염 물질을 정의합니다.

| 열거형 |
| --- |
| MAP_TYPE_UNSPECIFIED | 기본값입니다. 이 매개변수가 매개변수로 전달되면 서버는 이를 무시합니다. |
| UAQI_RED_GREEN | 범용 공기질 지수 빨간색-녹색 팔레트 |
| UAQI_INDIGO_PERSIAN | 세계 공기질 지수의 남색 팔레트입니다. |
| PM25_INDIGO_PERSIAN | PM2.5 색인 인디고-페르시아 팔레트 |
| GBR_DEFRA | 일일 공기질 지수 (영국) 색상 팔레트. |
| DEU_UBA | 독일 지역 공기질 지수 색상 팔레트 |
| CAN_EC | 캐나다 공기질 건강 지수 색상 팔레트 |
| FRA_ATMO | 프랑스 공기질 지수 색상 팔레트 |
| US_AQI | 미국 공기질 지수 색상 팔레트 |
