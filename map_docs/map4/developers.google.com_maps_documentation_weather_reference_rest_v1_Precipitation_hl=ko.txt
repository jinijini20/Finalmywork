지정된 위치의 강수량 값 집합을 나타냅니다.

| JSON 표현 |
| --- |
| {   "probability": {     object (PrecipitationProbability)   },   "qpf": {     object (QuantitativePrecipitationForecast)   } } |

| 필드 |
| --- |
| probability | object (PrecipitationProbability) 강수 확률 (0~100) |
| qpf | object (QuantitativePrecipitationForecast) 일정 기간 동안 누적된 강수량 (비 또는 눈)으로, 액체 물 등가량으로 측정됩니다. 참고: QPF는 양적 강수 예보의 약어입니다 (자세한 내용은 QuantitativePrecipitationForecast 정의를 참고하세요). |

PrecipitationProbability

특정 위치의 강수 확률을 나타냅니다.

| JSON 표현 |
| --- |
| {   "type": enum (PrecipitationType),   "percent": integer } |

| 필드 |
| --- |
| type | enum (PrecipitationType) 강수 유형을 나타내는 코드입니다. |
| percent | integer 강수 확률을 나타내는 0~100의 백분율입니다. |

PrecipitationType

특정 위치의 강수 유형을 나타냅니다.

| 열거형 |
| --- |
| PRECIPITATION_TYPE_UNSPECIFIED | 지정되지 않은 강수 유형입니다. |
| NONE | 강수 없음 |
| SNOW | 눈 강수량 |
| RAIN | 비가 내림 |
| LIGHT_RAIN | 약한 비가 내리고 있습니다. |
| HEAVY_RAIN | 폭우가 내립니다. |
| RAIN_AND_SNOW | 비와 눈의 강수량입니다. |
| SLEET | 진눈깨비 강수 |
| FREEZING_RAIN | 우빙 강우 |
