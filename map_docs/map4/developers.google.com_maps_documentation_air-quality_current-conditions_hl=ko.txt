currentConditions [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/currentConditions/lookup?hl=ko] 엔드포인트는 현재 시간당 공기질 정보를 제공합니다. 반환되는 AQI 및 관련 정보 (예: 오염 물질 농도 수준, 건강 권장사항)를 제어할 수 있습니다.

다음 주소로 HTTP POST 요청을 전송하여 currentConditions [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/currentConditions/lookup?hl=ko] 엔드포인트를 사용하여 현재 시간별 공기질을 요청할 수 있습니다.

https://airquality.googleapis.com/v1/currentConditions:lookup?key=YOUR_API_KEY

JSON 요청 본문 [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/currentConditions/lookup?hl=ko#request-body]에 요청 옵션을 포함합니다.
요청 본문에는 공기질 현재 상태를 가져올 위치와 응답에 포함할 공기질 정보를 제어하는 다양한 옵션이 포함됩니다.

API 탐색기를 사용하면 API 및 API 옵션을 익히기 위해 실시간으로 요청할 수 있습니다.
