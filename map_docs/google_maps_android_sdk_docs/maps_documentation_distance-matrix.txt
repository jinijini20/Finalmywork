Postman 컬렉션으로도 제공되는 Distance Matrix API의 OpenAPI 사양을 가져옵니다.
기능
Distance Matrix API의 핵심 기능에 대해 알아보세요.
교통정보 확인 [https://developers.google.com/maps/documentation/distance-matrix/distance-matrix?hl=ko#distance-matrix-advanced]
현재 교통상황을 기반으로 이동 시간을 계산하는 거리 행렬 요청을 생성합니다.
도로의 측면 지정 [https://developers.google.com/maps/documentation/distance-matrix/distance-matrix?hl=ko#location-modifiers]
계산된 경로가 도로의 특정 면을 통과해야 하는지 여부를 지정합니다.
앱 예
Maps JavaScript API를 사용하여 로컬 컴퓨터와 즐겨 사용하는 코드 플레이그라운드에서 라이브 코드 샘플을 실행합니다.
code [https://developers.google.com/maps/documentation/javascript/examples/distance-matrix?hl=ko]
거리 행렬 서비스 [https://developers.google.com/maps/documentation/javascript/examples/distance-matrix?hl=ko]
DistanceMatrixService 객체를 사용하여 위치 집합 간의 거리를 가져옵니다.
