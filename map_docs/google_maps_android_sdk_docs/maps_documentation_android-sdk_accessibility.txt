접근성 향상
Android용 Maps SDK에는 접근성 지원 기능이 기본적으로 포함되어 있습니다. 이 섹션에는 API를 사용하는 모든 애플리케이션에서 자동으로 사용 설정되는 접근성 기능의 개괄적 요약이 포함되어 있습니다.
사용자가 휴대기기에서 Talkback 접근성 기능을 사용 설정한 경우, 화면을 한 번 스와이프할 때마다 한 UI 요소에서 다른 UI 요소로 초점이 이동합니다. (화면을 한 번 스와이프하는 대신 인터페이스 위에서 손가락으로 드래그하여 UI 요소를 탐색할 수도 있습니다.) UI 요소에 초점이 맞춰지면 Talkback 기능이 요소의 이름을 읽습니다. 사용자가 화면의 아무 곳이나 두 번 탭하면 초점이 맞춰진 작업이 실행됩니다.
Android 앱의 접근성 향상에 대한 안내는 Android 접근성 문서 [https://developer.android.com/guide/topics/ui/accessibility/?hl=ko]를 참고하세요. 특히 지도를 설명하는 공지사항을 추가하는 것이 좋습니다. 공지사항의 텍스트를 지정하려면 뷰에서 setContentDescription() [https://developer.android.com/reference/android/view/View.html?hl=ko#setContentDescription(java.lang.CharSequence)]을 호출하세요.
