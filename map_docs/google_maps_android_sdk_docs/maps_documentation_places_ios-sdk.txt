시작하기
iOS용 Places SDK의 일반적으로 사용되는 기능을 사용하여 지도 작성을 시작합니다.
explore [https://developers.google.com/maps/documentation/places/ios-sdk/cloud-setup?hl=ko]
Google Maps Platform 시작하기 [https://developers.google.com/maps/documentation/places/ios-sdk/cloud-setup?hl=ko]
계정을 만들고, API 키를 생성하고, 지도 작성을 시작합니다.
settings [https://developers.google.com/maps/documentation/places/ios-sdk/start?hl=ko]
앱 구성 [https://developers.google.com/maps/documentation/places/ios-sdk/start?hl=ko]
iOS용 Places SDK를 사용하도록 앱을 구성합니다.
pin_drop [https://developers.google.com/maps/documentation/places/ios-sdk/place-id?hl=ko]
장소 ID에 대해 알아보기 [https://developers.google.com/maps/documentation/places/ios-sdk/place-id?hl=ko]
Google 지역 정보 데이터베이스 및 Google 지도에 있는 장소를 고유하게 식별하는 장소 ID에 대해 알아보세요.
storefront [https://developers.google.com/maps/documentation/places/ios-sdk/icons?hl=ko]
장소 아이콘에 대해 알아보기 [https://developers.google.com/maps/documentation/places/ios-sdk/icons?hl=ko]
다양한 유형의 장소에 대한 아이콘을 요청하고 표시합니다.
iOS용 Places SDK 기능 (신규)
iOS용 Places SDK (신규)의 핵심 기능에 대해 알아보세요.
장소 세부정보 (신규) [https://developers.google.com/maps/documentation/places/ios-sdk/details-place?hl=ko]
장소 ID를 사용하여 장소에 대한 세부정보를 가져옵니다.
장소 사진 (신규) [https://developers.google.com/maps/documentation/places/ios-sdk/place-photos?hl=ko]
애플리케이션에 고화질 장소 사진을 추가합니다.
텍스트 검색(신규) [https://developers.google.com/maps/documentation/places/ios-sdk/text-search?hl=ko]
텍스트 검색을 사용하여 장소에 대한 정보를 얻을 수 있습니다.
주변 검색 (신규) [https://developers.google.com/maps/documentation/places/ios-sdk/nearby-search?hl=ko]
위치를 검색 영역으로 지정하여 장소에 대한 정보를 가져옵니다.
Place Autocomplete (신규) [https://developers.google.com/maps/documentation/places/ios-sdk/place-autocomplete?hl=ko]
애플리케이션에 자동 완성 기능을 추가합니다(장소 이름, 주소 또는 플러스 코드로 검색).
장소 유형 (신규) [https://developers.google.com/maps/documentation/places/ios-sdk/place-types?hl=ko]
새 버전의 장소 유형에서는 새로운 여러 장소 유형을 추가하여 장소 검색 및 장소 자동완성 요청의 결과를 더욱 세분화할 수 있습니다.
