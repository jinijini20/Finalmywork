Route Optimization API
bookmark_border
Route Optimization API는 여러 차량에 작업과 경로를 할당하여 사용자가 교통 목표를 위해 제공하는 목표 및 제약 조건을 기준으로 최적화합니다.
시작하기
Route Optimization API를 사용해 빌드를 시작합니다.
explore [https://developers.google.com/maps/documentation/route-optimization/cloud-setup?hl=ko]
Google Maps Platform 시작하기 [https://developers.google.com/maps/documentation/route-optimization/cloud-setup?hl=ko]
계정을 만들고, Cloud 프로젝트를 설정하고, 빌드를 시작합니다.
done_all [https://developers.google.com/maps/documentation/route-optimization/construct-request?hl=ko]
첫 번째 경로 최적화 요청 [https://developers.google.com/maps/documentation/route-optimization/construct-request?hl=ko]
최적 경로를 계산합니다.
code [https://developers.google.com/maps/documentation/route-optimization/client-libraries?hl=ko]
클라이언트 라이브러리 [https://developers.google.com/maps/documentation/route-optimization/client-libraries?hl=ko]
C#, Java, Python, Go 또는 Node.js 클라이언트 라이브러리를 사용하여 서버에서 Google 지도 서비스를 사용하세요.
code [https://developers.google.com/maps/documentation/route-optimization/reference?hl=ko]
API 참조 보기 [https://developers.google.com/maps/documentation/route-optimization/reference?hl=ko]
경로 최적화 REST 및 gRPC API 참조를 확인합니다.
실험적 기능
Route Optimization API에서 사용할 수 있는 실험용 기능입니다.
sync_problem [https://developers.google.com/maps/tt/route-optimization/experimental/u-turn-avoidance/make-request?hl=ko]
U턴 회피 [https://developers.google.com/maps/tt/route-optimization/experimental/u-turn-avoidance/make-request?hl=ko]
경로 최적화 U턴 회피 실험용 REST 및 gRPC API 참조를 확인합니다.
settings_suggest [https://developers.google.com/maps/tt/route-optimization/experimental/objectives/make-request?hl=ko]
목표 [https://developers.google.com/maps/tt/route-optimization/experimental/objectives/make-request?hl=ko]
경로 최적화 목표 실험용 REST 및 gRPC API 참조를 확인하세요.
more_time [https://developers.google.com/maps/tt/route-optimization/experimental/otlr/make-request?hl=ko]
단일 비 차단 최적화 [https://developers.google.com/maps/tt/route-optimization/experimental/otlr/make-request?hl=ko]
경로 최적화 OptimizeToursLongRunning 및 OptimizeToursUri 실험용 REST 및 gRPC API 참조를 확인하세요.
energy_savings_leaf [https://developers.google.com/maps/tt/route-optimization/experimental/load-cost/make-request?hl=ko]
로드 비용 [https://developers.google.com/maps/tt/route-optimization/experimental/load-cost/make-request?hl=ko]
경로 최적화 로드 비용 실험용 REST 및 gRPC API 참조를 확인하세요.
