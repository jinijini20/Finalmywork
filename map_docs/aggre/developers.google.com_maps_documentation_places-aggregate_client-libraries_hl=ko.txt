이 페이지의 내용클라이언트 라이브러리 설치 [https://developers.google.com/maps/documentation/places-aggregate/client-libraries?hl=ko#install_the_client_library]인증 설정 [https://developers.google.com/maps/documentation/places-aggregate/client-libraries?hl=ko#setting_up_authentication]클라이언트 라이브러리 사용 [https://developers.google.com/maps/documentation/places-aggregate/client-libraries?hl=ko#use_the_client_library]추가 리소스 [https://developers.google.com/maps/documentation/places-aggregate/client-libraries?hl=ko#additional_resources]

이 페이지에서는 Places Aggregate API용 클라이언트 라이브러리를 시작하는 방법을 보여줍니다.

클라이언트 라이브러리에 대한 자세한 내용은 클라이언트 라이브러리 설명 [https://cloud.google.com/apis/docs/client-libraries-explained?hl=ko]을 참고하세요.

클라이언트 라이브러리 설치
--- 탭: 자바 [https://developers.google.com/maps/documentation/places-aggregate/client-libraries?hl=ko#%EC%9E%90%EB%B0%94] ---
자세한 내용은 자바 개발 환경 설정 [https://cloud.google.com/java/docs/setup?hl=ko]을 참고하세요.

설치 안내는 Google Places Aggregate API Client for Java [https://github.com/googleapis/google-cloud-java/tree/main/java-maps-area-insights]를 참고하세요.

--- 탭: Go [https://developers.google.com/maps/documentation/places-aggregate/client-libraries?hl=ko#go] ---
go get cloud.google.com/go/maps
자세한 내용은 Go 개발 환경 설정 [https://cloud.google.com/go/docs/setup?hl=ko]을 참고하세요.

설치 안내는 Go용 Google Places Aggregate API 클라이언트 [https://pkg.go.dev/cloud.google.com/go/maps/areainsights/apiv1]를 참고하세요.

--- 탭: Node.js [https://developers.google.com/maps/documentation/places-aggregate/client-libraries?hl=ko#node.js] ---
npm install @googlemaps/areainsights
자세한 내용은 Node.js 개발 환경 설정 [https://cloud.google.com/nodejs/docs/setup?hl=ko]을 참고하세요.

전체 설치 안내는 Node.js용 Google Places Aggregate API 클라이언트 [https://github.com/googleapis/google-cloud-node/tree/main/packages/google-maps-areainsights]를 참고하세요.

--- 탭: Python [https://developers.google.com/maps/documentation/places-aggregate/client-libraries?hl=ko#python] ---
자세한 내용은 Python 개발 환경 설정 [https://cloud.google.com/python/docs/setup?hl=ko]을 참고하세요.

전체 설치 안내는 Python용 Google Places Aggregate API 클라이언트 [https://github.com/googleapis/google-cloud-python/tree/main/packages/google-maps-areainsights]를 참고하세요.

--- 탭: UNNAMED ---
자세한 내용은 .NET 개발 환경 설정 [https://cloud.google.com/dotnet/docs/setup?hl=ko]을 참고하세요.

전체 설치 안내는 .Net용 Google Places Aggregate API 클라이언트 [https://github.com/googleapis/google-cloud-dotnet/tree/main/apis/Google.Maps.AreaInsights.V1]를 참고하세요.
인증 설정

클라이언트 라이브러리를 사용할 때는 애플리케이션 기본 사용자 인증 정보(ADC) [https://cloud.google.com/docs/authentication/application-default-credentials?hl=ko]를 사용하여 인증합니다. ADC 설정은 애플리케이션 기본 사용자 인증 정보에 대한 사용자 인증 정보 제공 [https://cloud.google.com/docs/authentication/provide-credentials-adc?hl=ko]을 참조하세요.
클라이언트 라이브러리에서 ADC를 사용하는 방법은 클라이언트 라이브러리를 사용하여 인증 [https://cloud.google.com/docs/authentication/client-libraries?hl=ko]을 참조하세요.

클라이언트 라이브러리 사용

사용 정보는 클라이언트 라이브러리 예 [https://developers.google.com/maps/documentation/places-aggregate/client-library-examples?hl=ko]를 참고하세요.
