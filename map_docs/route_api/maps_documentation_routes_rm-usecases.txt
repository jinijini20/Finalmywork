경로 매트릭스로 수행할 수 있는 작업
유럽 경제 지역 (EEA) 개발자
경로 행렬에는 거리 및 예상 도착 시간을 비롯하여 여러 출발지와 도착지 간의 추천 경로에 관한 정보 행렬이 포함됩니다.
경로 매트릭스를 사용하면 다음과 같은 사용 사례와 같이 비즈니스 요구사항에 따라 출발지와 도착지의 최적 조합을 선택할 수 있습니다.
배송할 최적의 창고를 찾습니다.
가장 가까운 적합한 차량을 찾아 픽업 또는 하차 요청을 합니다.
Routes API는 경로 유형과 경로 매트릭스에 제공된 정보를 맞춤설정하는 여러 옵션을 제공합니다. 다음 가이드 주제를 참고하여 시작해 보세요.
주제 설명
경로 매트릭스 가져오기 [https://developers.google.com/maps/documentation/routes/compute_route_matrix?hl=ko] 경로 매트릭스를 만드는 방법과 포함해야 할 정보를 알아보세요.
사용 가능한 경로 매트릭스 차량 유형 [https://developers.google.com/maps/documentation/routes/vehicles-rm?hl=ko] 경로를 확인할 차량 유형을 선택합니다.
사용 가능한 경로 매트릭스 옵션 [https://developers.google.com/maps/documentation/routes/options-rm?hl=ko] 경로 매트릭스에 포함할 정보 유형을 선택합니다.
경로 매트릭스 응답 검토 [https://developers.google.com/maps/documentation/routes/understand-rm-response?hl=ko] 애플리케이션에서 응답을 처리하는 방법을 알 수 있도록 Routes API에서 경로 매트릭스가 반환되는 방식을 검토합니다.
