경로에 사용 가능한 차량 유형
유럽 경제 지역 (EEA) 개발자
다양한 차량 유형에 적합한 경로를 확인할 수 있습니다. 차량 유형을 지정하지 않으면 Routes API는 기본적으로 휘발유 자동차를 사용합니다.
travelMode [https://developers.google.com/maps/documentation/routes/reference/rest/v2/RouteTravelMode?hl=ko] 매개변수를 사용하여 대중교통, 운전, 이륜차, 도보, 자전거 중에서 선택하여 이동 수단을 설정합니다.
참고: 도보, 자전거, 이륜차 경로는 베타 버전이며 명확한 인도, 보행자 전용 도로 또는 자전거 전용 도로가 누락될 수 있습니다. 앱에 표시하는 모든 도보, 자전거, 이륜차 경로에 이 경고를 사용자에게 표시해야 합니다.
차량 유형 설명
자전거 자전거 및 기타 사람이 직접 이동하는 경로를 계산합니다. 동력식 이륜차의 경우 이륜차를 참고하세요.
Drive [https://developers.google.com/maps/documentation/routes/compute_route_directions?hl=ko] 운전 경로를 계산합니다. 차량 유형을 지정하지 않으면 기본값은 Drive입니다.
대중교통 [https://developers.google.com/maps/documentation/routes/transit-route?hl=ko] 해당 지역에서 이용 가능한 대중교통 옵션을 사용하여 내비게이션 안내를 위한 대중교통 경로를 가져옵니다. 대중교통 옵션에는 버스, 지하철, 기차 등이 포함될 수 있습니다. 대중교통 경로에는 일반적으로 대중교통 정거장까지, 정거장 사이, 정거장에서의 도보 경로에 관한 안내도 포함됩니다.
이륜차 [https://developers.google.com/maps/documentation/routes/route_two_wheel?hl=ko] 오토바이는 2륜 전동차 (예: 오토바이)를 의미합니다. 오토바이 이동 모드는 사람이 직접 힘을 가해 이동하는 자전거 이동 모드와 다릅니다.
도보 도보 경로를 계산합니다.
