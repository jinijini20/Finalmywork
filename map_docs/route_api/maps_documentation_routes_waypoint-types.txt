경유지 유형 및 옵션
유럽 경제 지역 (EEA) 개발자
경유지란 경로의 시작 (출발) 위치와 끝(도착) 위치, 경로 행렬의 출발지와 목적지를 포함하여 경로의 한 지점입니다.
위치 지정 [https://developers.google.com/maps/documentation/routes/specify_location?hl=ko]에 설명된 대로 웨이포인트의 위치를 지정하는 방법에는 여러 가지가 있습니다. 시작 위치와 종료 위치를 설정하는 것 외에도 경로가 통과할 중간 경로를 추가하고 경로 동작에 관한 몇 가지 옵션을 사용할 수 있습니다.
경유지 옵션 설명
경유지 위치 지정 [https://developers.google.com/maps/documentation/routes/specify_location?hl=ko] 경로의 시작점과 끝점, 경로 중간 지점을 지정합니다.
중간 경유지 설정 [https://developers.google.com/maps/documentation/routes/intermed_waypoints?hl=ko] 중간 웨이포인트를 설정하여 경로가 통과할 출발지와 목적지 사이의 위치를 지정합니다. 중간 웨이포인트는 경유 또는 통과 웨이포인트일 수 있습니다 (아래 섹션 참고).
경로에 정류장 설정하기 [https://developers.google.com/maps/documentation/routes/stop_over?hl=ko] 중간 경유지를 경로의 정류장(예: 승하차)으로 지정합니다.
경로가 통과할 지점 설정 [https://developers.google.com/maps/documentation/routes/pass-through?hl=ko] 경로가 통과할 중간 웨이포인트를 지정합니다.
경유지의 차량 방향 지정 [https://developers.google.com/maps/documentation/routes/location_modifiers?hl=ko#specify_the_vehicle_heading] 차량이 도착했을 때의 방향을 지정합니다. 도로변에서는 사용할 수 없습니다.
경로의 도로 측면 지정 [https://developers.google.com/maps/documentation/routes/location_modifiers?hl=ko#specify_a_side_of_the_road_preference] 차량이 도착할 도로의 측면을 지정합니다. 제목과 함께 사용할 수 없습니다.
경로의 정류장 순서 최적화 [https://developers.google.com/maps/documentation/routes/opt-way?hl=ko] 차량이 각 웨이포인트에 도착할 때 가고자 하는 방향을 지정합니다.
