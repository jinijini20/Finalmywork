경로 환경
경로 환경은 탐색 환경에서 사용되는 경로를 맞춤설정할 수 있는 Navigation SDK 기능을 의미합니다. 경로 환경 기능은 Google 탐색 환경 [https://developers.google.com/maps/documentation/navigation/ios-sdk/intro-google-nav?hl=ko]과 맞춤 탐색 환경 [https://developers.google.com/maps/documentation/navigation/ios-sdk/intro-custom-nav?hl=ko] 모두에 적용됩니다.
