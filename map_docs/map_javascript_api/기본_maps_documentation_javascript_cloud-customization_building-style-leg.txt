Change the style of buildings
Select platform: Android [https://developers.google.com/maps/documentation/android-sdk/cloud-customization/building-style] iOS [https://developers.google.com/maps/documentation/ios-sdk/cloud-customization/building-style] JavaScript [https://developers.google.com/maps/documentation/javascript/cloud-customization/building-style] Web Service [https://developers.google.com/maps/documentation/maps-static/cloud-customization/building-style]
Availability: This feature is visible on Android, iOS, Maps JavaScript API, and Maps Static.
When you use a vector map in Android, iOS, or JavaScript, buildings use the 3D style beginning at zoom level 17 and higher by default. If you want to display only the 2D outlines of buildings for a cleaner appearance, select the Footprints style.
Experimental: This feature can only be set for light map styles. When you link a light map style that has custom building styles to a map ID, these styles also apply to the dark map style.
