오픈소스 라이브러리
여러 오픈소스 라이브러리를 사용하여 Maps JavaScript API를 위한 추가 기능을 제공할 수 있습니다. 이러한 라이브러리는 GitHub [https://github.com/googlemaps]에서 버전이 관리되고 npm [https://www.npmjs.com/]에 게시됩니다.
웹 구성요소
Maps JavaScript API 웹 구성요소 [https://developers.google.com/maps/documentation/web-components?hl=ko]를 사용하여 웹사이트에 지도 기능을 빠르게 추가할 수 있습니다. 웹 구성요소는 상용구 코드, 권장사항, 반응형 디자인을 캡슐화하여 복잡한 지도 UI를 단일 HTML 요소로 바꿉니다.
확장 구성요소 라이브러리(베타) [https://github.com/googlemaps/extended-component-library]
Google Maps Platform 확장 구성요소 라이브러리는 개발자가 더 나은 지도를 더 쉽고 빠르게 빌드하는 데 도움이 되는 웹 구성요소 모음입니다.
npm i @googlemaps/extended-component-library
[https://www.npmjs.com/package/@googlemaps/extended-component-library] [https://github.com/googlemaps/extended-component-library] [https://discord.gg/jRteCzP]
API 로드
최신 웹 애플리케이션에서는 다음과 같은 라이브러리를 사용하여 Maps JavaScript API를 로드할 수 있습니다.
JS API 로더 [https://github.com/googlemaps/js-api-loader]
프로미스를 사용하여 Maps JavaScript API를 동적으로 로드합니다.
npm i @googlemaps/js-api-loader
[https://www.npmjs.com/package/@googlemaps/js-api-loader] [https://github.com/googlemaps/js-api-loader] [https://discord.gg/jRteCzP]
Using TypeScript
TypeScript 사용자는 다음 패키지를 사용하여 시작할 수 있습니다. 자세한 내용은 TypeScript 사용 가이드 [https://developers.google.com/maps/documentation/javascript/using-typescript?hl=ko]를 참고하세요.
TypeScript 유형 [https://github.com/definitelytyped/definitelytyped]
Maps JavaScript API용 TypeScript 유형
npm i -D @types/google.maps
[https://www.npmjs.com/package/@types/google.maps] [https://github.com/definitelytyped/definitelytyped] [https://discord.gg/jRteCzP]
TypeScript 가드 [https://github.com/googlemaps/js-typescript-guards]
합집합 유형의 범위를 좁히기 위한 TypeScript 가드입니다.
npm i -D @googlemaps/typescript-guards
[https://www.npmjs.com/package/@googlemaps/typescript-guards] [https://github.com/googlemaps/js-typescript-guards] [https://discord.gg/jRteCzP]

마커 및 오버레이 확장

많은 양의 마커를 위해 확대/축소 수준별로 클러스터를 만들고 관리합니다.
npm i @googlemaps/markerclusterer
