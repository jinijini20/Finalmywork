Change the app background color
Select platform: Android [https://developers.google.com/maps/documentation/android-sdk/cloud-customization/app-background] iOS [https://developers.google.com/maps/documentation/ios-sdk/cloud-customization/app-background] JavaScript [https://developers.google.com/maps/documentation/javascript/cloud-customization/app-background] Web Service [https://developers.google.com/maps/documentation/maps-static/cloud-customization/app-background]
Availability: This feature is visible on Android, iOS, Maps JavaScript API, and Maps Static.
You can change the background color of the Map container. This color shows before the tiles load.
Set the app background color
In the Map feature pane, select the gear icon to open the Settings pane.
Set the app background color by using the color picker or entering a Hex string. When you set the color, the tiles reload to show the chosen background color.
