장소 클래스
플랫폼 선택: Android [https://developers.google.com/maps/documentation/places/android-sdk/overview?hl=ko] iOS [https://developers.google.com/maps/documentation/places/ios-sdk/overview?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/place?hl=ko] 웹 서비스 [https://developers.google.com/maps/documentation/places/web-service/overview?hl=ko]
유럽 경제 지역 (EEA) 개발자
개요
장소 클래스는 장소 라이브러리를 사용하기 위한 API를 제공하며 프로미스 [https://developers.google.com/maps/documentation/javascript/promises?hl=ko]와 같은 최신 사용 패턴을 지원합니다. Place 클래스를 사용하면 앱에서 위치 인식 기능을 만들고, 필요에 따라 맞춤설정할 수 있는 지능형 검색 기능을 통해 사용자에게 자세한 위치 데이터를 제공할 수 있습니다. Places 서비스의 데이터는 실제 세계에서 가장 정확하고 최신이며 포괄적인 장소 모델 중 하나입니다.
Place 클래스로 할 수 있는 작업
장소 클래스에는 다음과 같은 기능이 있습니다.
장소 검색
텍스트 검색 (신규) [https://developers.google.com/maps/documentation/javascript/place-search?hl=ko]을 사용하면 텍스트 쿼리를 사용하여 장소 데이터를 검색할 수 있습니다.
주변 지역 검색 (신규) [https://developers.google.com/maps/documentation/javascript/nearby-search?hl=ko]을 사용하면 장소 유형별로 주변 장소를 검색할 수 있습니다.
장소 자동완성 [https://developers.google.com/maps/documentation/javascript/place-autocomplete-overview?hl=ko]은 앱에 Google 지도 검색창의 검색 전 입력 기능을 제공합니다.
장소 데이터 가져오기
장소 세부정보 (신규) [https://developers.google.com/maps/documentation/javascript/place-details?hl=ko]는 텍스트 요약, 영업시간, 주소, 사용자 리뷰, 사진 등 장소에 관한 자세한 정보를 제공합니다.
장소 사진 [https://developers.google.com/maps/documentation/javascript/place-photos?hl=ko]은 장소 데이터베이스의 고품질 사진 이미지에 대한 액세스를 제공합니다.
장소 리뷰 [https://developers.google.com/maps/documentation/javascript/place-reviews?hl=ko]는 장소 리뷰 및 평점에 대한 액세스를 제공합니다.
