Places API
2억 곳 이상의 장소에 대한 위치 데이터를 가져와 앱에 장소 세부정보, 검색, 자동 완성 기능을 추가하세요.
시작하기
Places API를 사용하여 빌드를 시작합니다.
explore [https://developers.google.com/maps/documentation/places/web-service/cloud-setup?hl=ko]
Google Maps Platform 시작하기 [https://developers.google.com/maps/documentation/places/web-service/cloud-setup?hl=ko]
계정을 만들고, API 키를 생성하고, 지도 작성을 시작합니다.
quiz [https://developers.google.com/maps/documentation/places/web-service/choose-api?hl=ko]
API 버전 선택 [https://developers.google.com/maps/documentation/places/web-service/choose-api?hl=ko]
Places는 두 가지 API 버전인 Places API와 차세대 API인 Places API (신규)를 지원합니다. 나에게 적합한 버전을 선택하는 방법을 알아보세요.
pin_drop [https://developers.google.com/maps/documentation/places/web-service/place-id?hl=ko]
장소 ID에 대해 알아보기 [https://developers.google.com/maps/documentation/places/web-service/place-id?hl=ko]
장소 ID는 Google Places 데이터베이스 및 Google Maps에 있는 장소를 고유하게 식별합니다.
travel_explore [https://developers.google.com/maps/documentation/places/web-service/text-search?hl=ko]
첫 번째 텍스트 검색 (신규) 요청 [https://developers.google.com/maps/documentation/places/web-service/text-search?hl=ko]
이름, 주소 또는 전화번호와 같은 텍스트 입력을 기반으로 장소 정보를 반환합니다.
code [https://developers.google.com/maps/documentation/javascript/places?hl=ko]
Maps JavaScript API의 장소 라이브러리 [https://developers.google.com/maps/documentation/javascript/places?hl=ko]
장소 검색 및 자동완성 기능을 애플리케이션에 추가하여 사용자가 텍스트 쿼리나 전화번호를 기반으로 장소를 찾을 수 있도록 합니다.
storefront [https://developers.google.com/maps/documentation/places/web-service/icons?hl=ko]
장소 아이콘에 대해 알아보기 [https://developers.google.com/maps/documentation/places/web-service/icons?hl=ko]
다양한 유형의 장소에 대한 아이콘을 요청하고 표시합니다.
Places API의 기능 (신규)
Places API의 핵심 기능 (신규)에 대해 알아보세요.
주변 검색 (신규) [https://developers.google.com/maps/documentation/places/web-service/nearby-search?hl=ko]
지정된 지역 내에서 일치하는 장소 유형을 검색합니다.
텍스트 검색(신규) [https://developers.google.com/maps/documentation/places/web-service/text-search?hl=ko]
텍스트 검색 문자열을 기반으로 장소 정보를 검색합니다.
장소 세부정보 (신규) [https://developers.google.com/maps/documentation/places/web-service/place-details?hl=ko]
특정 시설 또는 관심 장소에 대한 세부정보를 가져옵니다.
장소 사진 (신규) [https://developers.google.com/maps/documentation/places/web-service/place-photos?hl=ko]
애플리케이션에 고화질 장소 사진을 추가합니다.
자동 완성 (신규) [https://developers.google.com/maps/documentation/places/web-service/place-autocomplete?hl=ko]
입력 문자열에 대한 응답으로 장소 예상 검색어 및 쿼리 예상 검색어를 가져옵니다.
예시 앱
Maps JavaScript API를 사용하여 로컬 컴퓨터와 즐겨 사용하는 코드 플레이그라운드에서 라이브 코드 샘플을 실행합니다.
travel_explore [https://developers.google.com/maps/documentation/javascript/examples/place-search?hl=ko]
장소 검색 [https://developers.google.com/maps/documentation/javascript/examples/place-search?hl=ko]
findPlaceFromQuery() 메서드를 사용하여 장소를 찾고 지도에 마커를 추가합니다.
business [https://developers.google.com/maps/documentation/javascript/examples/place-details?hl=ko]
장소 세부정보 [https://developers.google.com/maps/documentation/javascript/examples/place-details?hl=ko]
getDetails() 메서드를 사용하여 특정 장소의 장소 세부정보를 검색합니다.
description [https://developers.google.com/maps/documentation/javascript/examples/place-search-pagination?hl=ko]
장소 검색 페이지로 나누기 [https://developers.google.com/maps/documentation/javascript/examples/place-search-pagination?hl=ko]
장소 검색결과의 추가 페이지를 가져옵니다.
input [https://developers.google.com/maps/documentation/javascript/examples/places-autocomplete?hl=ko]
장소 자동완성 [https://developers.google.com/maps/documentation/javascript/examples/places-autocomplete?hl=ko]
텍스트 검색에 대한 응답으로 장소 예상 검색어를 반환합니다.
dynamic_form [https://developers.google.com/maps/documentation/javascript/examples/places-autocomplete-addressform?hl=ko]
장소 자동 완성 주소 양식 [https://developers.google.com/maps/documentation/javascript/examples/places-autocomplete-addressform?hl=ko]
장소의 선택된 주소 구성요소를 캡처하여 주소 양식을 채우는 데 사용합니다.
hotel [https://developers.google.com/maps/documentation/javascript/examples/places-autocomplete-hotelsearch?hl=ko]
장소 자동 완성 호텔 검색 [https://developers.google.com/maps/documentation/javascript/examples/places-autocomplete-hotelsearch?hl=ko]
클릭 시 각 호텔의 세부정보와 함께 지정된 위치에 있는 모든 호텔을 지도에 표시합니다.
