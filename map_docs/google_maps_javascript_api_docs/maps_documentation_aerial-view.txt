Aerial View API
bookmark_border
Google의 3D 지리 공간 이미지를 사용하여 렌더링된 항공 뷰 동영상을 만들고 표시할 수 있습니다. 항공 뷰 동영상은 상공에서 선회하는 드론이 촬영한 동영상을 시뮬레이션하여 관심 장소의 실사 3D 항공 뷰입니다.
