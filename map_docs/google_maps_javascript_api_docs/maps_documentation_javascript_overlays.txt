지도에 그리기
지도에 객체를 추가하여 점, 선, 영역 또는 객체의 모음을 지정할 수 있습니다. Maps JavaScript API에서는 이러한 객체를 오버레이라고 합니다. 오버레이는 위도/경도 좌표에 연결되어 지도를 드래그하거나 확대/축소하면 움직입니다.
그리기 라이브러리: 사용자가 지도에 그릴 수 있도록 하려면 그리기 라이브러리 [https://developers.google.com/maps/documentation/javascript/drawinglayer?hl=ko] 문서를 참고하세요. 그리기 라이브러리에서 제공되는 도구를 사용하여 지도에 다중선, 다각형, 마커를 그릴 수 있는 그래픽 인터페이스를 제공할 수 있습니다.
지오코딩: 지오코딩 [https://developers.google.com/maps/documentation/javascript/geocoding?hl=ko] 요청을 사용하여 장소나 상세 주소의 위도 및 경도를 쉽게 가져올 수 있습니다. 요청 시 각각 LatLng가 포함된 응답의 목록이 반환됩니다.
오버레이 유형
Maps JavaScript API에는 프로그래매틱 방식으로 추가할 수 있는 여러 유형의 오버레이가 있습니다.
정보 창은 지도의 지정된 위치에서 팝업 풍선 안에 콘텐츠(일반적으로 텍스트 또는 이미지)를 표시하는 특별한 종류의 오버레이입니다. 정보 창 [https://developers.google.com/maps/documentation/javascript/infowindows?hl=ko]을 참고하세요.
지도에서 선은 위치의 정해진 순서를 나타내는 다중선을 사용하여 표시됩니다. 도형 및 선 [https://developers.google.com/maps/documentation/javascript/shapes?hl=ko#polylines]을 참고하세요.
지도에서 임의의 도형으로 된 영역은 다각형을 사용하여 표시됩니다. 다중선과 마찬가지로 다각형은 위치의 정해진 순서를 나타내지만 다중선과 달리 다각형은 닫힌 영역을 정의합니다. 도형 및 선 [https://developers.google.com/maps/documentation/javascript/shapes?hl=ko#polygons]을 참고하세요.
지도에서 원 [https://developers.google.com/maps/documentation/javascript/shapes?hl=ko#circles]과 직사각형 [https://developers.google.com/maps/documentation/javascript/shapes?hl=ko#circles]도 정의할 수 있습니다.
기호를 사용하여 마커의 아이콘을 맞춤설정하거나 다중선에 이미지를 추가할 수 있습니다. 기호는 SVG 경로 표기법을 사용하여 경로로 정의되는 벡터 기반 이미지입니다. API는 기호를 표시하는 방법을 관리할 수 있는 옵션도 제공합니다. 기호 [https://developers.google.com/maps/documentation/javascript/symbols?hl=ko]를 참고하세요.
지도에 이미지를 배치하려는 경우 지면 오버레이를 사용할 수 있습니다. 지면 오버레이 [https://developers.google.com/maps/documentation/javascript/groundoverlays?hl=ko]를 참고하세요.
또한 OverlayView 인터페이스를 구현하여 자신만의 맞춤 오버레이를 구현할 수도 있습니다. 맞춤 오버레이 [https://developers.google.com/maps/documentation/javascript/customoverlays?hl=ko]를 참고하세요.
지도 레이어는 오버레이 지도 유형을 사용하여 표시할 수도 있습니다. 기본 지도 타일 집합을 대체하거나 기존 기본 지도 타일 집합 위에 오버레이로 표시되는 맞춤 지도 유형을 만들어 자신만의 타일 집합을 만들 수 있습니다. 맞춤 지도 유형 [https://developers.google.com/maps/documentation/javascript/maptypes?hl=ko#CustomMapTypes]을 참고하세요.
