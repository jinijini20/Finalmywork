Android용 Maps SDK
bookmark_border
Android 앱을 위한 양방향의 동적 맞춤 지도, 위치, 지리정보 환경을 구축하세요.
시작하기
Android용 Maps SDK의 일반적인 기능을 사용하여 지도 작성을 시작합니다.
explore [https://developers.google.com/maps/documentation/android-sdk/cloud-setup?hl=ko]
Google Maps Platform 시작하기 [https://developers.google.com/maps/documentation/android-sdk/cloud-setup?hl=ko]
계정을 만들고, API 키를 생성하고, 지도 작성을 시작합니다.
map [https://developers.google.com/maps/documentation/android-sdk/map-with-marker?hl=ko]
마커가 포함된 첫 지도 만들기 [https://developers.google.com/maps/documentation/android-sdk/map-with-marker?hl=ko]
Android용 Maps SDK를 로드하고 마커가 있는 지도를 웹 앱에 추가하는 방법을 알아봅니다.
palette [https://developers.google.com/maps/documentation/android-sdk/maps-customization-overview?hl=ko]
지도 스타일 맞춤설정 [https://developers.google.com/maps/documentation/android-sdk/maps-customization-overview?hl=ko]
도로, 지형 지물, 관심 장소 등 지도의 모든 측면을 맞춤설정합니다.
chat [https://developers.google.com/maps/documentation/android-sdk/current-place-tutorial?hl=ko]
지도에서 현재 장소를 선택하고 세부정보를 표시합니다. [https://developers.google.com/maps/documentation/android-sdk/current-place-tutorial?hl=ko]
Android 기기의 현재 위치를 찾고 해당 위치에 있는 장소 또는 기타 관심 장소의 세부정보를 표시합니다.
기능
Android용 Maps SDK의 핵심 기능에 대한 문서를 찾아봅니다.
동적 지도 [https://developers.google.com/maps/documentation/android-sdk/map?hl=ko]
Android 앱에 기본 지도를 추가합니다.
마커 [https://developers.google.com/maps/documentation/android-sdk/marker?hl=ko]
지도에 마커를 추가하고, 클릭 이벤트와 상호작용하도록 설정하며, 색상, 이미지 등을 맞춤설정합니다.
정보 창 [https://developers.google.com/maps/documentation/android-sdk/infowindows?hl=ko]
사용자가 마커를 탭하면 추가 정보와 컨텍스트가 포함된 정보 창을 표시합니다.
컨트롤 및 동작 [https://developers.google.com/maps/documentation/android-sdk/controls?hl=ko]
기본으로 제공되는 UI 구성요소 및 동작을 구성하여 사용자가 상호작용하는 방식을 맞춤설정합니다.
일정 [https://developers.google.com/maps/documentation/android-sdk/events?hl=ko]
지도 클릭, 마커 클릭, 카메라 변경, 오버레이 이벤트 등을 비롯한 지도상의 이벤트를 수신합니다.
도형 [https://developers.google.com/maps/documentation/android-sdk/shapes?hl=ko]
지도에 다각형, 다중선, 원을 추가하여 지도 모양을 맞춤설정합니다.
지면 오버레이 [https://developers.google.com/maps/documentation/android-sdk/groundoverlay?hl=ko]
위도와 경도에 연결된 이미지 오버레이를 추가하고 이를 움직여 지도를 드래그하거나 확대/축소합니다.
Wear OS 지원 [https://developers.google.com/maps/documentation/android-sdk/wear?hl=ko]
Android용 Maps SDK를 사용하여 웨어러블 앱을 만듭니다.
카메라 및 뷰 [https://developers.google.com/maps/documentation/android-sdk/views?hl=ko]
위치, 확대/축소 수준, 방위 등 카메라의 다양한 측면을 설정합니다.
타일 오버레이 [https://developers.google.com/maps/documentation/android-sdk/tileoverlay?hl=ko]
Google의 기본 지도 타일 위에 이미지를 겹쳐서 데이터와 로컬 이미지를 앱에 추가합니다.
유틸리티 및 도우미 라이브러리
Android 앱용 오픈소스 유틸리티 및 도우미 라이브러리를 통합합니다.
code [https://developers.google.com/maps/documentation/android-sdk/ktx?hl=ko]
Maps Android KTX [https://developers.google.com/maps/documentation/android-sdk/ktx?hl=ko]
Kotlin 확장 프로그램을 사용하여 간결하고 자연스러운 Kotlin을 사용 설정합니다.
code [https://developers.google.com/maps/documentation/android-sdk/utility?hl=ko]
Android용 Maps SDK 유틸리티 라이브러리 [https://developers.google.com/maps/documentation/android-sdk/utility?hl=ko]
다양한 애플리케이션에 사용할 수 있는 유틸리티로 앱을 향상합니다.
code [https://github.com/googlemaps/android-maps-rx]
Maps Android Rx [https://github.com/googlemaps/android-maps-rx]
Android용 Maps SDK 및 Android용 Places SDK에 RxJava 바인딩을 사용합니다.
Jetpack Compose 구성요소 [https://github.com/googlemaps/android-maps-compose]
Android용 Maps SDK에 Jetpack Compose를 사용합니다.
