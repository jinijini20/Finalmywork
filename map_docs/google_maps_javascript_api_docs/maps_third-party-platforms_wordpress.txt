WordPress 사이트에서 Google Maps Platform 사용하기
Google Maps Platform은 지도를 표시하고, 장소 데이터를 검색하고, 경로를 생성하는 등 Google 지도의 다양한 기능을 앱에서 사용할 수 있게 해 줍니다.
많은 WordPress 플러그인이 Google Maps Platform을 사용하여 위치 기반 기능을 제공합니다. 매장 검색 기능을 비롯하여 여러 위치 간의 경로를 제공하고 주변 비즈니스와 명소를 표시하는 등의 기능을 예로 들 수 있습니다.
사용 가능한 WordPress 플러그인
다양한 WordPress 플러그인을 통해 지도, 매장 검색 기능, 장소 자동 완성 검색창 등 Google Maps Platform을 활용하는 기능을 WordPress 사이트에 추가할 수 있습니다.
전체 목록을 보려면 wordpress.org에서 사용 가능한 플러그인을 찾아보세요.
플러그인 찾아보기 [https://wordpress.org/plugins/search/Google+Maps/]
Google Maps Platform API 키 생성하기
API 키는 적절한 Google Cloud Platform 계정에 대한 보안 액세스 및 사용료 청구를 위해 Google Maps Platform으로 전송되는 요청을 인증하는 데 사용되는 고유한 사용자 인증 정보입니다.
Google Maps Platform 기능을 사용하려는 경우 WordPress 플러그인에서 사용자를 대신하여 요청을 처리할 수 있도록 API 키를 제공해야 합니다.
API 키를 만들려면 다음 단계를 따르세요.
Google Cloud Platform 계정이 없으면 계정을 만듭니다.
새 Google Cloud Platform 프로젝트를 만듭니다.
Google Maps Platform API를 사용 설정합니다.
API 키를 생성합니다.
Google Maps Platform 퀵스타트 위젯 [https://developers.google.com/maps/third-party-platforms/quick-start-widget-users?hl=ko]을 사용하면 계정 설정을 처리하고 WordPress 플러그인에 제공할 API 키를 생성하는 과정을 간편하게 완료할 수 있습니다.
요금 정보
Google Maps Platform에서는 Google Cloud 프로젝트에 연결된 결제 계정에 매월 200달러 상당의 무료 사용량을 자동으로 제공합니다.
지도 로드, 경로, 장소 정보 요청과 같이 Google Maps Platform API에 대해 이루어지는 요청별로 사용량에 대한 요금이 부과됩니다. 요청당 비용은 요청 유형과 월간 총 요청 수에 따라 다릅니다. 자세한 내용은 가격 계산기를 참고하세요.
요금 계산기로 이동 [https://mapsplatform.google.com/pricing/?hl=ko]
