Routes API
Compute Routes 및 Compute Route Matrix
Routes API 서비스에는 두 가지 메서드가 있습니다. Compute Routes는 두 위치 간의 이상적인 경로를 반환하고 Compute Route Matrix는 여러 출발지와 목적지 위치 간의 경로 행렬에 대한 거리와 이동 시간을 반환합니다.
시작하기
Routes API Compute Routes 및 Compute Route Matrix로 빌드를 시작합니다.
[https://developers.google.com/maps/documentation/routes/compute-route-over?hl=ko]
컴퓨팅 경로 [https://developers.google.com/maps/documentation/routes/compute-route-over?hl=ko]
여러 위치 간의 대중교통, 자전거, 운전, 2륜 모터 차량 또는 도보 경로를 실시간 교통정보와 함께 제공합니다.
자세히 알아보기 [https://developers.google.com/maps/documentation/routes/compute-route-over?hl=ko]

기능
Routes API의 핵심 기능을 알아봅니다.
오토바이 경주 [https://developers.google.com/maps/documentation/routes/route_two_wheel?hl=ko]
오토바이란 이륜차를 말합니다. 이 모드는 사람이 움직이는 이동 수단인 자전거 이동 모드와 다릅니다.
통행료 계산 [https://developers.google.com/maps/documentation/routes/calculate_toll_fees?hl=ko]
일부 도시의 경우 노선에 대한 예상 통행료를 적절한 통화로 계산할 수 있습니다.
친환경 경로 구성하기 [https://developers.google.com/maps/documentation/routes/eco-routes?hl=ko]
차량 엔진 유형에 따라 연비 또는 에너지 효율이 가장 우수한 경로를 보여주는 친환경 경로를 추정합니다.
다중선에 대한 교통정보 요청 [https://developers.google.com/maps/documentation/routes/traffic_on_polylines?hl=ko]
교통정보 인식 다중선을 따라 교통상황에 대한 정보를 요청합니다.
품질 대 지연 시간 구성 [https://developers.google.com/maps/documentation/routes/config_trade_offs?hl=ko]
경로 품질 및 지연 시간을 제어합니다.
Compute Route Matrix의 결과 스트리밍 [https://developers.google.com/maps/documentation/routes/compute_route_matrix?hl=ko#stream_results]
지연 시간을 단축할 수 있는 대로 결과를 반환합니다.
차량 스톱오버 한정자 사용 [https://developers.google.com/maps/documentation/routes/stop_over?hl=ko]
경유지가 승차 또는 하차를 위한 정류장에 사용되는지 여부를 지정합니다.
