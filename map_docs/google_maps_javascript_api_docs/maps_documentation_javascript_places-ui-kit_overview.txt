장소 UI 키트
플랫폼 선택: Android [https://developers.google.com/maps/documentation/places/android-sdk/places-ui-kit-overview?hl=ko] iOS [https://developers.google.com/maps/documentation/places/ios-sdk/places-ui-kit-overview?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/places-ui-kit/overview?hl=ko]
장소 UI 키트 구성요소 라이브러리를 사용하면 Places API를 지원하는 동일한 데이터를 사용하여 앱과 웹페이지에 장소에 대한 익숙한 Google 지도 사용자 경험을 구현할 수 있습니다. 독립적으로, 함께 또는 다른 Google Maps Platform API와 함께 사용하여 최소한의 비용과 코드로 장소 관련 풍부한 환경을 제공할 수 있는 개별 UI 구성요소 집합이 포함되어 있습니다.
Places UI Kit에는 장소 데이터를 렌더링하기 위한 다음 HTML 요소가 포함되어 있습니다.
장소 세부정보 요소 [https://developers.google.com/maps/documentation/javascript/places-ui-kit/place-details?hl=ko] (장소 세부정보 및 장소 세부정보 Compact)는 선택한 장소의 영업시간, 웹사이트, 리뷰와 같은 세부정보를 렌더링합니다.
장소 검색 요소 [https://developers.google.com/maps/documentation/javascript/places-ui-kit/place-list?hl=ko](장소 주변 검색 및 장소 텍스트 검색)는 주변 검색 또는 텍스트 검색 쿼리에 대한 응답으로 장소 목록을 렌더링합니다.
기본 장소 자동 완성 요소 [https://developers.google.com/maps/documentation/javascript/places-ui-kit/basic-autocomplete?hl=ko]는 텍스트 입력란을 렌더링하고 UI 선택 목록에 장소 예상 검색어를 제공하며 선택한 장소의 장소 ID를 반환합니다.
고도 [https://developers.google.com/maps/documentation/javascript/places-ui-kit/elevation?hl=ko](실험적)는 고도 데이터를 단일 지점의 숫자 값 또는 경로의 시각적 플롯으로 렌더링합니다.
도움말: 장소 UI 키트는 추가 비용 없이 다양한 시각적 맞춤설정 옵션을 제공합니다. 자세한 내용은 Places UI 키트 맞춤 스타일 지정 [https://developers.google.com/maps/documentation/javascript/places-ui-kit/custom-styling?hl=ko]을 참고하세요.
핵심 기능 및 역량
최소한의 코드로 시작하여 Google의 신뢰할 수 있는 환경을 통합하세요.
Places API보다 저렴한 비용으로 장소용 Google 지도 UI를 앱에 가져올 수 있습니다.
필요에 가장 적합한 데이터 및 표시 옵션을 선택합니다.
