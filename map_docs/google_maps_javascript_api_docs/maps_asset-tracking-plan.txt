Google Maps Platform 애셋 추적 요금제
bookmark_border
이 페이지의 내용
결제 [https://developers.google.com/maps/asset-tracking-plan?hl=ko#billing]
할당량 한도 [https://developers.google.com/maps/asset-tracking-plan?hl=ko#quota_limits]
2019년 6월 3일, Google Maps Platform에서 새로운 애셋 추적 요금제가 출시되었습니다. 이 요금제는 대규모 애셋(수천 개의 애셋)과 특정한 사용 사례가 있는 고객에게만 제공됩니다. 이 요금제에 관심이 있다면 영업팀에 문의 [https://cloud.google.com/contact-maps?hl=ko]하시기 바랍니다.
참고: 애셋 추적에는 매달 제공되는 200달러의 지도 크레딧이 적용되지 않습니다.
다음 API에는 Google Maps Platform 애셋 추적 요금제가 적용됩니다.
Geocoding API
Directions API
Geolocation API
Elevation API
Roads API(속도 제한 포함)
Time Zone API
Distance Matrix API
Maps Static API
Maps JavaScript API
Street View API
결제
이 요금제에 대한 결제는 결제 계정에 연결된 고정 월간 요금으로 설정됩니다. 그러면 개발자 콘솔 사용 보고서 [https://console.cloud.google.com/project/_/billing?hl=ko]: 지도 월별 수수료에 새 SKU가 표시됩니다. 이 수수료는 일할 계산되어 사용량 그래프에 표시됩니다. 월별 결제 수수료는 특정 프로젝트가 아닌 결제 계정에 연결되므로 사용량 보고서에서 두 항목을 '프로젝트' 또는 'SKU'로 그룹화할 때 표시됩니다.
API당 사용량은 사용량 보고서에 계속 표시됩니다. 위에서 설명한 API는 가격이 0이며 매월 고정 수수료가 적용됩니다. 적용되지 않는 API(Places API)는 사용한 만큼만 요금이 청구됩니다. 자세한 내용은 계약에 포함된 '장소 서비스 가격' 섹션을 참고하세요.
할당량 한도
가입한 요금제에 따라 적용되는 할당량 한도가 결정됩니다. 프로젝트에 적용되는 한도를 확인하려면 계약의 '지도 애셋 추적 구독 요금제 등급' 섹션을 살펴보세요. 표에 해당하는 QPS 증가분이 포함된 것을 확인할 수 있습니다.
프로젝트를 최초 설정할 때 포함된 QPS를 Directions API와 Geocoding API 모두에 프로비저닝했습니다. 이 QPS 증가분을 Google Maps Platform 애셋 추적 요금제가 적용되는 다른 API에 추가해야 하는 경우 기술 지원팀 [https://developers.google.com/maps/support?hl=ko]에 문의하세요.
포함된 QPS 이상이 필요하다면 계정 관리자에게 문의하여 상위 구독 요금제로 전환하세요.
긴급한 임시 할당량 증가 요청을 해야 한다면 기술 지원팀 [https://developers.google.com/maps/support?hl=ko]에 문의하세요.
