건물의 스타일 변경
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/cloud-customization/building-style-leg?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/cloud-customization/building-style-leg?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/cloud-customization/building-style-leg?hl=ko] 웹 서비스 [https://developers.google.com/maps/documentation/maps-static/cloud-customization/building-style-leg?hl=ko]
지원 대상: 이 기능은 Android, iOS, Maps JavaScript API, Maps Static에서 볼 수 있습니다.
Android, iOS 또는 JavaScript에서 벡터 지도를 사용하는 경우 확대/축소 수준 17 이상부터 기본적으로 건물에서 3D 스타일을 사용합니다. 더 깔끔하게 보이도록 건물을 2D 윤곽선으로만 표시하려면 풋프린트 스타일을 선택하세요.
실험용: 이 기능은 밝은 지도 스타일에만 설정할 수 있습니다. 맞춤 건물 스타일이 있는 밝은 지도 스타일을 지도 ID에 연결하면 이러한 스타일이 어두운 지도 스타일에도 적용됩니다.
건물 스타일 설정
표시된 건물을 볼 수 있을 정도로 지도를 확대합니다.
지도 지형지물 창에서 톱니바퀴 아이콘 을 선택하여 설정 창을 엽니다.
건물을 3D 윤곽선으로 표시하려면 3D를 선택합니다.
참고: 3D 스타일 건물에는 획 색상 및 획 너비 맞춤설정을 사용할 수 없습니다.
건물을 2D 풋프린트로 표시하려면 풋프린트를 선택합니다.
