시작하기
시작하기 전에: Elevation API를 사용하려면 결제 계정이 있고 Elevation API가 사용 설정된 프로젝트가 필요합니다. 프로젝트 소유자 및 결제 관리자를 여러 명 만들어 이러한 역할을 맡은 담당자가 팀에 상주하도록 하는 것이 좋습니다. 자세한 내용은 Cloud 콘솔에서 설정하기 [https://developers.google.com/maps/documentation/elevation/cloud-setup?hl=ko]를 참고하세요.
Elevation API는 해저 지형 (음수 값 반환)을 포함하는 지표면 위치의 고도 데이터를 제공합니다.
이 서비스는 클라이언트 측 지도 JavaScript API [https://developers.google.com/maps/documentation/javascript/elevation?hl=ko]의 일부로 사용하거나 Google 지도 서비스용 Java 클라이언트, Python 클라이언트, Go 클라이언트, Node.js 클라이언트 [https://developers.google.com/maps/documentation/elevation/client-library?hl=ko]와 함께 서버 측에서 사용할 수도 있습니다.
샘플 요청 및 응답
HTTP 인터페이스를 통해 Elevation API에 액세스합니다. 요청은 URL 문자열로 구성되며, 위도/경도 좌표를 사용하여 위치 또는 경로 정점을 식별합니다. 요청에는 API 키가 포함되어야 합니다.
다음 예에서는 '마일 하이 시티'인 콜로라도주 덴버의 고도를 JSON 형식으로 요청합니다.
웹브라우저에 URL을 입력하여 테스트할 수 있습니다 (YOUR_API_KEY를 실제 API 키 [https://developers.google.com/maps/documentation/elevation/start?hl=ko#get-a-key]로 바꾸어야 함). 응답에는 해당 지점의 고도(미터)와 해상도 값(고도가 보간된 데이터 점 사이의 최대 거리, 미터)이 포함됩니다.
요청 URL 및 사용 가능한 매개변수 빌드 [https://developers.google.com/maps/documentation/elevation/overview?hl=ko#ElevationRequests] 및 응답 이해 [https://developers.google.com/maps/documentation/elevation/overview?hl=ko#ElevationResponses]에 관한 자세한 내용은 개발자 가이드 [https://developers.google.com/maps/documentation/elevation/overview?hl=ko]를 참고하세요.
다음은 샘플 응답입니다.
클라이언트 라이브러리로 코딩 시작하기
클라이언트 라이브러리는 인증, 요청 제한, 자동 재시도와 같은 일반적인 작업의 간단한 네이티브 구현을 제공하여 Google 지도 웹 서비스 API로 개발하는 것을 더 쉽게 만듭니다. Elevation API는 Google 지도 서비스용 Java 클라이언트, Python 클라이언트, Go 클라이언트, Node.js 클라이언트 [https://developers.google.com/maps/documentation/elevation/client-library?hl=ko]에서 사용할 수 있습니다.
인증, 할당량, 가격 책정, 정책
인증
Elevation API를 사용하려면 먼저 API를 사용 설정하고 적절한 인증 사용자 인증 정보를 가져와야 합니다. 자세한 내용은 Google Maps Platform 시작하기 [https://developers.google.com/maps/get-started?hl=ko]를 참고하세요.
할당량 및 가격 책정
Elevation API에 설정된 할당량 및 가격에 관한 자세한 내용은 사용량 및 결제 [https://developers.google.com/maps/documentation/elevation/usage-and-billing?hl=ko] 페이지를 참고하세요.
정책
Elevation API는 API 정책 [https://developers.google.com/maps/documentation/elevation/policies?hl=ko]에 따라 사용해야 합니다.
자세히 알아보기
Elevation API를 사용하면 여러 위치에서 고도 데이터를 요청 [https://developers.google.com/maps/documentation/elevation/overview?hl=ko#PointElevation]하는 등 다양한 작업을 할 수 있습니다. 자세한 예시와 기타 세부정보는 Elevation API 개발자 가이드 [https://developers.google.com/maps/documentation/elevation/overview?hl=ko]를 참고하세요.
Elevation API 개발자 가이드 [https://developers.google.com/maps/documentation/elevation/overview?hl=ko]는 Google Maps Platform API 중 하나에서 제공하는 지도 내에서 고도 데이터를 사용하려는 웹사이트 및 모바일 개발자를 위해 작성되었습니다. API 사용에 관한 소개와 사용 가능한 매개변수에 관한 참조 자료를 제공합니다.
