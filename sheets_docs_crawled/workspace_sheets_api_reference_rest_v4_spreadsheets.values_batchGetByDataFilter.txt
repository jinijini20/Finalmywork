승인 범위
MatchedValueRange
지정된 데이터 필터와 일치하는 값 범위를 하나 이상 반환합니다. 호출자는 스프레드시트 ID와 하나 이상의 DataFilters [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko]를 지정해야 합니다. 요청의 데이터 필터와 일치하는 범위가 반환됩니다.
HTTP 요청
POST https://sheets.googleapis.com/v4/spreadsheets/{spreadsheetId}/values:batchGetByDataFilter
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
경로 매개변수
매개변수
spreadsheetId
string
데이터를 가져올 스프레드시트의 ID입니다.
요청 본문
요청 본문에는 다음과 같은 구조의 데이터가 포함됩니다.
JSON 표현
{
  "dataFilters": [
    {
      object (DataFilter [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko])
    }
  ],
  "majorDimension": enum (Dimension [https://developers.google.com/workspace/sheets/api/reference/rest/v4/Dimension?hl=ko]),
  "valueRenderOption": enum (ValueRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko]),
  "dateTimeRenderOption": enum (DateTimeRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DateTimeRenderOption?hl=ko])
}
필드
dataFilters[]
object (DataFilter [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko])
검색할 값의 범위를 일치시키는 데 사용되는 데이터 필터입니다. 지정된 데이터 필터와 일치하는 범위가 응답에 포함됩니다.
majorDimension
enum (Dimension [https://developers.google.com/workspace/sheets/api/reference/rest/v4/Dimension?hl=ko])
결과에서 사용해야 하는 주요 측정기준입니다.
예를 들어 스프레드시트 데이터가 A1=1,B1=2,A2=3,B2=4이면 해당 범위를 선택하고 majorDimension=ROWS를 설정하는 요청은 [[1,2],[3,4]]를 반환하지만 majorDimension=COLUMNS를 설정하는 요청은 [[1,3],[2,4]]를 반환합니다.
valueRenderOption
enum (ValueRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko])
값이 출력에 표시되는 방식입니다. 기본 렌더링 옵션은 FORMATTED_VALUE [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko]입니다.
dateTimeRenderOption
enum (DateTimeRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DateTimeRenderOption?hl=ko])
날짜, 시간, 기간이 출력에 표시되는 방식 valueRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values/batchGetByDataFilter?hl=ko#body.request_body.FIELDS.value_render_option]가 FORMATTED_VALUE [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko]이면 무시됩니다. 기본 dateTime 렌더링 옵션은 SERIAL_NUMBER [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DateTimeRenderOption?hl=ko]입니다.
응답 본문
DataFilters [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko]로 선택한 스프레드시트에서 두 개 이상의 값 범위를 검색할 때의 응답입니다.
성공한 경우 응답 본문은 다음과 같은 구조의 데이터를 포함합니다.
JSON 표현
{
  "spreadsheetId": string,
  "valueRanges": [
    {
      object (MatchedValueRange [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values/batchGetByDataFilter?hl=ko#MatchedValueRange])
    }
  ]
}
필드
spreadsheetId
string
데이터를 가져온 스프레드시트의 ID입니다.
valueRanges[]
object (MatchedValueRange [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values/batchGetByDataFilter?hl=ko#MatchedValueRange])
요청된 값과 일치하는 데이터 필터 목록
승인 범위
다음 OAuth 범위 중 하나가 필요합니다.
https://www.googleapis.com/auth/drive
https://www.googleapis.com/auth/drive.file
https://www.googleapis.com/auth/spreadsheets
자세한 내용은 승인 가이드 [https://developers.google.com/workspace/guides/configure-oauth-consent?hl=ko]를 참고하세요.
MatchedValueRange
하나 이상의 데이터 필터에 의해 일치된 값 범위입니다.
JSON 표현
{
  "valueRange": {
    object (ValueRange [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values?hl=ko#ValueRange])
  },
  "dataFilters": [
    {
      object (DataFilter [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko])
    }
  ]
}
필드
valueRange
object (ValueRange [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values?hl=ko#ValueRange])
DataFilter [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko]와 일치하는 값입니다.
dataFilters[]
object (DataFilter [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko])
값 범위와 일치하는 요청의 DataFilters [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko]입니다.
