스프레드시트의 범위에 값을 설정합니다. 호출자는 스프레드시트 ID, 범위, valueInputOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueInputOption?hl=ko]를 지정해야 합니다.
HTTP 요청
PUT https://sheets.googleapis.com/v4/spreadsheets/{spreadsheetId}/values/{range}
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
경로 매개변수
매개변수
spreadsheetId
string
업데이트할 스프레드시트의 ID입니다.
range
string
업데이트할 값의 A1 표기법 [https://developers.google.com/workspace/sheets/api/guides/concepts?hl=ko#cell]입니다.
쿼리 매개변수
매개변수
valueInputOption
enum (ValueInputOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueInputOption?hl=ko])
입력 데이터가 해석되어야 하는 방식입니다.
includeValuesInResponse
boolean
업데이트 응답에 업데이트된 셀의 값을 포함할지 여부를 결정합니다. 기본적으로 응답에는 업데이트된 값이 포함되지 않습니다. 작성할 범위가 실제로 작성된 범위보다 큰 경우 응답에는 요청된 범위의 모든 값 (마지막 빈 행 및 열 제외)이 포함됩니다.
responseValueRenderOption
enum (ValueRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko])
응답의 값을 렌더링하는 방법을 결정합니다. 기본 렌더링 옵션은 FORMATTED_VALUE [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko]입니다.
responseDateTimeRenderOption
enum (DateTimeRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DateTimeRenderOption?hl=ko])
응답의 날짜, 시간, 기간을 렌더링하는 방법을 결정합니다. responseValueRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values/update?hl=ko#body.QUERY_PARAMETERS.response_value_render_option]가 FORMATTED_VALUE [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko]이면 무시됩니다. 기본 dateTime 렌더링 옵션은 SERIAL_NUMBER [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DateTimeRenderOption?hl=ko]입니다.
요청 본문
요청 본문에 ValueRange [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values?hl=ko#ValueRange]의 인스턴스가 포함됩니다.
응답 본문
성공한 경우 응답 본문에 UpdateValuesResponse [https://developers.google.com/workspace/sheets/api/reference/rest/v4/UpdateValuesResponse?hl=ko]의 인스턴스가 포함됩니다.
승인 범위
다음 OAuth 범위 중 하나가 필요합니다.
https://www.googleapis.com/auth/drive
https://www.googleapis.com/auth/drive.file
https://www.googleapis.com/auth/spreadsheets
자세한 내용은 승인 가이드 [https://developers.google.com/workspace/guides/configure-oauth-consent?hl=ko]를 참고하세요.
