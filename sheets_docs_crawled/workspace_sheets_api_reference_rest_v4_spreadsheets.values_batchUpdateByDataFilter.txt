스프레드시트의 하나 이상의 범위에 값을 설정합니다. 호출자는 스프레드시트 ID, valueInputOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueInputOption?hl=ko], 하나 이상의 DataFilterValueRanges [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values/batchUpdateByDataFilter?hl=ko#DataFilterValueRange]를 지정해야 합니다.
HTTP 요청
POST https://sheets.googleapis.com/v4/spreadsheets/{spreadsheetId}/values:batchUpdateByDataFilter
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.

string
업데이트할 스프레드시트의 ID입니다.

필드
valueInputOption
enum (ValueInputOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueInputOption?hl=ko])
입력 데이터가 해석되어야 하는 방식입니다.
data[]
object (DataFilterValueRange [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values/batchUpdateByDataFilter?hl=ko#DataFilterValueRange])
스프레드시트에 적용할 새 값입니다. 지정된 DataFilter [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko]에 두 개 이상의 범위가 일치하는 경우 지정된 값이 이러한 모든 범위에 적용됩니다.
includeValuesInResponse
boolean
업데이트 응답에 업데이트된 셀의 값을 포함할지 여부를 결정합니다. 기본적으로 응답에는 업데이트된 값이 포함되지 않습니다. 각 BatchUpdateValuesResponse.responses [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values/batchUpdate?hl=ko#body.BatchUpdateValuesResponse.FIELDS.responses] 내의 updatedData 필드에는 업데이트된 값이 포함됩니다. 작성할 범위가 실제로 작성된 범위보다 큰 경우 응답에는 요청된 범위의 모든 값 (마지막 비어 있는 행 및 열 제외)이 포함됩니다.
responseValueRenderOption
enum (ValueRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko])
응답의 값을 렌더링하는 방법을 결정합니다. 기본 렌더링 옵션은 FORMATTED_VALUE [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko]입니다.
responseDateTimeRenderOption
enum (DateTimeRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DateTimeRenderOption?hl=ko])
응답의 날짜, 시간, 기간을 렌더링하는 방법을 결정합니다. responseValueRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values/batchUpdateByDataFilter?hl=ko#body.request_body.FIELDS.response_value_render_option]가 FORMATTED_VALUE [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko]이면 무시됩니다. 기본 dateTime 렌더링 옵션은 SERIAL_NUMBER [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DateTimeRenderOption?hl=ko]입니다.
