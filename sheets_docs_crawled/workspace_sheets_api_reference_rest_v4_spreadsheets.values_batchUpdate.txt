스프레드시트의 하나 이상의 범위에 값을 설정합니다. 호출자는 스프레드시트 ID, valueInputOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueInputOption?hl=ko], 하나 이상의 ValueRanges [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values?hl=ko#ValueRange]를 지정해야 합니다.
HTTP 요청
POST https://sheets.googleapis.com/v4/spreadsheets/{spreadsheetId}/values:batchUpdate
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
경로 매개변수
매개변수
spreadsheetId
string
업데이트할 스프레드시트의 ID입니다.
요청 본문
요청 본문에는 다음과 같은 구조의 데이터가 포함됩니다.
JSON 표현
{
  "valueInputOption": enum (ValueInputOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueInputOption?hl=ko]),
  "data": [
    {
      object (ValueRange [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values?hl=ko#ValueRange])
    }
  ],
  "includeValuesInResponse": boolean,
  "responseValueRenderOption": enum (ValueRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko]),
  "responseDateTimeRenderOption": enum (DateTimeRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DateTimeRenderOption?hl=ko])
}
필드
valueInputOption
enum (ValueInputOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueInputOption?hl=ko])
입력 데이터가 해석되어야 하는 방식입니다.
data[]
object (ValueRange [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values?hl=ko#ValueRange])
스프레드시트에 적용할 새 값입니다.
includeValuesInResponse
boolean
업데이트 응답에 업데이트된 셀의 값을 포함할지 여부를 결정합니다. 기본적으로 응답에는 업데이트된 값이 포함되지 않습니다. 각 BatchUpdateValuesResponse.responses [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values/batchUpdate?hl=ko#body.BatchUpdateValuesResponse.FIELDS.responses] 내의 updatedData 필드에는 업데이트된 값이 포함됩니다. 작성할 범위가 실제로 작성된 범위보다 큰 경우 응답에는 요청된 범위의 모든 값 (마지막 빈 행 및 열 제외)이 포함됩니다.
responseValueRenderOption
enum (ValueRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko])
응답의 값을 렌더링하는 방법을 결정합니다. 기본 렌더링 옵션은 FORMATTED_VALUE [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko]입니다.
responseDateTimeRenderOption
enum (DateTimeRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DateTimeRenderOption?hl=ko])
응답의 날짜, 시간, 기간을 렌더링하는 방법을 결정합니다. responseValueRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values/batchUpdate?hl=ko#body.request_body.FIELDS.response_value_render_option]가 FORMATTED_VALUE [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko]이면 무시됩니다. 기본 dateTime 렌더링 옵션은 SERIAL_NUMBER [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DateTimeRenderOption?hl=ko]입니다.

스프레드시트에서 값 범위를 업데이트할 때의 응답입니다.
