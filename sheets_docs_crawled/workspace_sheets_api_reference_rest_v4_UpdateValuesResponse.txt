UpdateValuesResponse
bookmark_border
스프레드시트에서 값 범위를 업데이트할 때의 응답입니다.
JSON 표현
{
  "spreadsheetId": string,
  "updatedRange": string,
  "updatedRows": integer,
  "updatedColumns": integer,
  "updatedCells": integer,
  "updatedData": {
    object (ValueRange [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values?hl=ko#ValueRange])
  }
}
필드
spreadsheetId
string
업데이트가 적용된 스프레드시트입니다.
updatedRange
string
업데이트가 적용된 범위 (A1 표기법)입니다.
updatedRows
integer
행의 셀이 하나 이상 업데이트된 행의 수입니다.
updatedColumns
integer
열의 셀이 하나 이상 업데이트된 열의 개수입니다.
updatedCells
integer
업데이트된 셀 수입니다.
updatedData
object (ValueRange [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values?hl=ko#ValueRange])
업데이트가 적용된 후 셀의 값입니다. 요청의 includeValuesInResponse 필드가 true인 경우에만 포함됩니다.
