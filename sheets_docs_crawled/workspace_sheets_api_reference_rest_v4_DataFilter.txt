필드
통합 필드 filter. 선택되는 데이터를 제한할 수 있는 필터의 종류입니다. filter은 다음 중 하나여야 합니다.
developerMetadataLookup
object (DeveloperMetadataLookup [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko#DeveloperMetadataLookup])
이 DeveloperMetadataLookup [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko#DeveloperMetadataLookup]에 설명된 기준과 일치하는 개발자 메타데이터와 연결된 데이터를 선택합니다.
a1Range
string
지정된 A1 범위와 일치하는 데이터를 선택합니다.
gridRange
object (GridRange [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/other?hl=ko#GridRange])
GridRange [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/other?hl=ko#GridRange]로 설명된 범위와 일치하는 데이터를 선택합니다.
DeveloperMetadataLookup
지정된 모든 필드와 일치하는 DeveloperMetadata [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata?hl=ko#DeveloperMetadata]를 선택합니다. 예를 들어 메타데이터 ID만 지정된 경우 이 특정 고유 ID가 있는 DeveloperMetadata [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata?hl=ko#DeveloperMetadata]가 고려됩니다. 메타데이터 키가 지정되면 해당 키가 있는 모든 개발자 메타데이터가 고려됩니다. 키, 공개 상태, 위치 유형이 모두 지정된 경우 해당 유형의 위치와 연결된 키와 공개 상태가 있는 모든 개발자 메타데이터가 고려됩니다. 일반적으로 지정된 모든 필드의 교집합과 일치하는 모든 DeveloperMetadata가 선택됩니다. 어떤 필드나 필드 조합이든 지정할 수 있습니다.

필드
locationType
enum (DeveloperMetadataLocationType [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata?hl=ko#DeveloperMetadata.DeveloperMetadataLocationType])
선택한 개발자 메타데이터를 지정된 유형의 위치와 연결된 항목으로 제한합니다. 예를 들어 이 필드가 ROW [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata?hl=ko#DeveloperMetadata.DeveloperMetadataLocationType]로 지정된 경우 이 조회는 행에 연결된 개발자 메타데이터만 고려합니다. 이 필드를 지정하지 않으면 모든 위치 유형이 고려됩니다. locationMatchingStrategy [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko#DeveloperMetadataLocationMatchingStrategy]가 INTERSECTING으로 지정되었거나 metadataLocation [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata?hl=ko#DeveloperMetadata.DeveloperMetadataLocation]가 스프레드시트가 아닌 위치로 지정된 경우 이 필드는 SPREADSHEET [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata?hl=ko#DeveloperMetadata.DeveloperMetadataLocationType]로 지정할 수 없습니다. 스프레드시트 메타데이터는 다른 개발자 메타데이터 위치와 교차할 수 없습니다. 또한 locationMatchingStrategy [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko#DeveloperMetadataLocationMatchingStrategy]가 EXACT로 지정된 경우 이 필드는 지정되지 않아야 합니다.
metadataLocation
object (DeveloperMetadataLocation [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata?hl=ko#DeveloperMetadata.DeveloperMetadataLocation])
선택한 개발자 메타데이터를 지정된 위치와 연결된 항목으로 제한합니다. 이 필드는 지정된 locationMatchingStrategy [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko#DeveloperMetadataLocationMatchingStrategy]에 따라 정확한 위치 또는 교차하는 모든 위치와 일치합니다.
locationMatchingStrategy
enum (DeveloperMetadataLocationMatchingStrategy [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko#DeveloperMetadataLocationMatchingStrategy])
이 조회가 위치와 일치하는 방식을 결정합니다. 이 필드가 EXACT로 지정된 경우 지정된 정확한 위치에 연결된 개발자 메타데이터만 일치합니다. 이 필드가 INTERSECTING으로 지정된 경우 교차하는 위치에 연결된 개발자 메타데이터도 일치합니다. 이 필드를 지정하지 않으면 기본값은 INTERSECTING [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko#DeveloperMetadataLocationMatchingStrategy]입니다. 이 필드를 지정하는 경우 metadataLocation [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata?hl=ko#DeveloperMetadata.DeveloperMetadataLocation]도 지정해야 합니다.
metadataId
integer
선택한 개발자 메타데이터를 일치하는 DeveloperMetadata.metadata_id [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata?hl=ko#DeveloperMetadata.FIELDS.metadata_id]가 있는 메타데이터로 제한합니다.
metadataKey
string
선택한 개발자 메타데이터를 일치하는 DeveloperMetadata.metadata_key [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata?hl=ko#DeveloperMetadata.FIELDS.metadata_key]가 있는 메타데이터로 제한합니다.
metadataValue
string
선택한 개발자 메타데이터를 일치하는 DeveloperMetadata.metadata_value [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata?hl=ko#DeveloperMetadata.FIELDS.metadata_value]가 있는 메타데이터로 제한합니다.
visibility
enum (DeveloperMetadataVisibility [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata?hl=ko#DeveloperMetadata.DeveloperMetadataVisibility])
선택한 개발자 메타데이터를 일치하는 DeveloperMetadata.visibility [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata?hl=ko#DeveloperMetadata.FIELDS.visibility]가 있는 메타데이터로 제한합니다. 지정하지 않으면 요청하는 프로젝트에 표시되는 모든 개발자 메타데이터가 고려됩니다.
DeveloperMetadataLocationMatchingStrategy
개발자 메타데이터 위치를 일치시키기 위한 전략의 열거입니다.
열거형
DEVELOPER_METADATA_LOCATION_MATCHING_STRATEGY_UNSPECIFIED 기본값 이 값은 사용하면 안 됩니다.
EXACT_LOCATION 지정된 위치와 정확하게 일치해야 함을 나타냅니다. 예를 들어 세 번째 행이 위치로 지정된 경우 이 일치 전략은 세 번째 행에 연결된 개발자 메타데이터와만 일치합니다. 다른 위치에 연결된 메타데이터는 고려되지 않습니다.
INTERSECTING_LOCATION 지정된 위치가 해당 정확한 위치와 교차하는 위치와 일치해야 함을 나타냅니다. 예를 들어 세 번째 행이 위치로 지정된 경우 이 일치 전략은 세 번째 행에 연결된 개발자 메타데이터와 세 번째 행과 교차하는 위치에 연결된 메타데이터를 일치시킵니다. 예를 들어 B열에 개발자 메타데이터가 연결된 경우 B열이 3번 행과 교차하므로 이 일치 전략은 해당 위치와도 일치합니다.
