승인 범위
스프레드시트에서 하나 이상의 값 범위를 지웁니다. 호출자는 스프레드시트 ID와 하나 이상의 DataFilters [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko]를 지정해야 합니다. 지정된 데이터 필터와 일치하는 범위가 지워집니다. 값만 삭제되고 셀의 다른 모든 속성 (예: 서식, 데이터 유효성 검사 등)은 유지됩니다.
HTTP 요청
POST https://sheets.googleapis.com/v4/spreadsheets/{spreadsheetId}/values:batchClearByDataFilter
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
경로 매개변수
매개변수
spreadsheetId
string
업데이트할 스프레드시트의 ID입니다.
요청 본문
요청 본문에는 다음과 같은 구조의 데이터가 포함됩니다.
JSON 표현
{
  "dataFilters": [
    {
      object (DataFilter [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko])
    }
  ]
}
필드
dataFilters[]
object (DataFilter [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko])
지울 범위를 결정하는 데 사용되는 DataFilters [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko]입니다.
응답 본문
스프레드시트에서 DataFilters [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko]로 선택한 값 범위를 지울 때의 응답입니다.
성공한 경우 응답 본문은 다음과 같은 구조의 데이터를 포함합니다.
JSON 표현
{
  "spreadsheetId": string,
  "clearedRanges": [
    string
  ]
}
필드
spreadsheetId
string
업데이트가 적용된 스프레드시트입니다.
clearedRanges[]
string
삭제된 범위(A1 표기법 [https://developers.google.com/workspace/sheets/api/guides/concepts?hl=ko#cell]) 요청이 경계가 없는 범위 또는 시트의 경계보다 큰 범위에 관한 경우, 시트의 한도에 경계가 지정된 삭제된 실제 범위입니다.
