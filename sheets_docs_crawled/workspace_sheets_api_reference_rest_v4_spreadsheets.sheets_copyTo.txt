스프레드시트에서 다른 스프레드시트로 단일 시트를 복사합니다. 새로 만든 시트의 속성을 반환합니다.
HTTP 요청
POST https://sheets.googleapis.com/v4/spreadsheets/{spreadsheetId}/sheets/{sheetId}:copyTo
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
경로 매개변수
매개변수
spreadsheetId
string
복사할 시트가 포함된 스프레드시트의 ID입니다.
sheetId
integer
복사할 시트의 ID입니다.
요청 본문
요청 본문에는 다음과 같은 구조의 데이터가 포함됩니다.
JSON 표현
{
  "destinationSpreadsheetId": string
}
필드
destinationSpreadsheetId
string
시트를 복사할 스프레드시트의 ID입니다.
응답 본문
성공한 경우 응답 본문에 SheetProperties [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/sheets?hl=ko#SheetProperties]의 인스턴스가 포함됩니다.
승인 범위
다음 OAuth 범위 중 하나가 필요합니다.
https://www.googleapis.com/auth/drive
https://www.googleapis.com/auth/drive.file
https://www.googleapis.com/auth/spreadsheets
자세한 내용은 승인 가이드 [https://developers.google.com/workspace/guides/configure-oauth-consent?hl=ko]를 참고하세요.
