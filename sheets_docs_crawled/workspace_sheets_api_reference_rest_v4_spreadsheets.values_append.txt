InsertDataOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values/append?hl=ko#insertdataoption]
스프레드시트에 값을 추가합니다. 입력 범위는 기존 데이터를 검색하고 해당 범위 내에서 '표'를 찾는 데 사용됩니다. 값은 테이블의 첫 번째 열부터 테이블의 다음 행에 추가됩니다. 테이블이 감지되고 데이터가 추가되는 방법에 관한 자세한 내용은 가이드 [https://developers.google.com/workspace/sheets/api/guides/values?hl=ko#appending_values] 및 샘플 코드 [https://developers.google.com/workspace/sheets/api/samples/writing?hl=ko#append_values]를 참고하세요.
호출자는 스프레드시트 ID, 범위, valueInputOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueInputOption?hl=ko]를 지정해야 합니다. valueInputOption는 입력 데이터가 시트에 추가되는 방식 (열 단위 또는 행 단위)만 제어하며 데이터가 쓰기 시작하는 셀에는 영향을 미치지 않습니다.
HTTP 요청
POST https://sheets.googleapis.com/v4/spreadsheets/{spreadsheetId}/values/{range}:append
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.

string

string
데이터의 논리적 테이블을 검색할 범위의 A1 표기법 [https://developers.google.com/workspace/sheets/api/guides/concepts?hl=ko#cell] 값은 테이블의 마지막 행 뒤에 추가됩니다.

enum (InsertDataOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values/append?hl=ko#InsertDataOption])

enum (ValueRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko])
응답의 값을 렌더링하는 방법을 결정합니다. 기본 렌더링 옵션은 FORMATTED_VALUE [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko]입니다.
responseDateTimeRenderOption
enum (DateTimeRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DateTimeRenderOption?hl=ko])
응답의 날짜, 시간, 기간을 렌더링하는 방법을 결정합니다. responseValueRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values/append?hl=ko#body.QUERY_PARAMETERS.response_value_render_option]가 FORMATTED_VALUE [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko]이면 무시됩니다. 기본 dateTime 렌더링 옵션은 SERIAL_NUMBER [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DateTimeRenderOption?hl=ko]입니다.
요청 본문
요청 본문에 ValueRange [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values?hl=ko#ValueRange]의 인스턴스가 포함됩니다.
응답 본문
스프레드시트에서 값 범위를 업데이트할 때의 응답입니다.
성공한 경우 응답 본문은 다음과 같은 구조의 데이터를 포함합니다.

  }

필드
spreadsheetId
string
업데이트가 적용된 스프레드시트입니다.
tableRange
string
값이 추가되기 전 (값이 추가되기 전)에 값이 추가되는 테이블의 범위 (A1 표기법)입니다. 테이블을 찾을 수 없는 경우 비어 있습니다.
updates
object (UpdateValuesResponse [https://developers.google.com/workspace/sheets/api/reference/rest/v4/UpdateValuesResponse?hl=ko])
적용된 업데이트에 관한 정보입니다.
승인 범위
다음 OAuth 범위 중 하나가 필요합니다.
https://www.googleapis.com/auth/drive
https://www.googleapis.com/auth/drive.file
https://www.googleapis.com/auth/spreadsheets
자세한 내용은 승인 가이드 [https://developers.google.com/workspace/guides/configure-oauth-consent?hl=ko]를 참고하세요.
InsertDataOption
새 데이터가 입력될 때 기존 데이터가 변경되는 방식을 결정합니다.
열거형
OVERWRITE 새 데이터는 쓰여지는 영역의 기존 데이터를 덮어씁니다. 참고: 시트 끝에 데이터를 추가해도 데이터를 쓸 수 있도록 새 행이나 열이 삽입됩니다.
INSERT_ROWS 새 데이터의 행이 삽입됩니다.
