승인 범위
MatchedDeveloperMetadata
지정된 DataFilter [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko]와 일치하는 모든 개발자 메타데이터를 반환합니다. 제공된 DataFilter가 DeveloperMetadataLookup [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko#DeveloperMetadataLookup] 객체를 나타내는 경우 선택한 모든 DeveloperMetadata 항목이 반환됩니다. DataFilter가 스프레드시트의 위치를 나타내는 경우 이 지역과 교차하는 위치와 연결된 모든 개발자 메타데이터가 반환됩니다.
HTTP 요청
POST https://sheets.googleapis.com/v4/spreadsheets/{spreadsheetId}/developerMetadata:search
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
경로 매개변수
매개변수
spreadsheetId
string
메타데이터를 가져올 스프레드시트의 ID입니다.
요청 본문
요청 본문에는 다음과 같은 구조의 데이터가 포함됩니다.
JSON 표현
{
  "dataFilters": [
    {
      object (DataFilter [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko])
    }
  ]
}
필드
dataFilters[]
object (DataFilter [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DataFilter?hl=ko])
반환할 DeveloperMetadata 항목을 결정하는 데 사용되는 기준을 설명하는 데이터 필터입니다. 지정된 필터와 일치하는 DeveloperMetadata가 응답에 포함됩니다.
응답 본문
개발자 메타데이터 검색 요청에 대한 답장입니다.
성공한 경우 응답 본문은 다음과 같은 구조의 데이터를 포함합니다.
JSON 표현
{
  "matchedDeveloperMetadata": [
    {
      object (MatchedDeveloperMetadata [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata/search?hl=ko#MatchedDeveloperMetadata])
    }
  ]
}
필드
matchedDeveloperMetadata[]
object (MatchedDeveloperMetadata [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.developerMetadata/search?hl=ko#MatchedDeveloperMetadata])
검색 요청의 기준과 일치하는 메타데이터입니다.
승인 범위
다음 OAuth 범위 중 하나가 필요합니다.
https://www.googleapis.com/auth/drive
https://www.googleapis.com/auth/drive.file
https://www.googleapis.com/auth/spreadsheets
자세한 내용은 승인 가이드 [https://developers.google.com/workspace/guides/configure-oauth-consent?hl=ko]를 참고하세요.
