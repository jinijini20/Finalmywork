DimensionRange
bookmark_border
시트의 단일 측정기준에 따른 범위입니다. 모든 색인은 0부터 시작합니다. 색인은 반쯤 열려 있습니다. 시작 색인은 포함하고 종료 색인은 제외합니다. 누락된 색인은 해당 측면에서 범위가 무제한임을 나타냅니다.
JSON 표현
{
  "sheetId": integer,
  "dimension": enum (Dimension [https://developers.google.com/workspace/sheets/api/reference/rest/v4/Dimension?hl=ko]),
  "startIndex": integer,
  "endIndex": integer
}
필드
sheetId
integer
이 범위가 있는 시트입니다.
dimension
enum (Dimension [https://developers.google.com/workspace/sheets/api/reference/rest/v4/Dimension?hl=ko])
스팬의 크기입니다.
startIndex
integer
범위의 시작 (양 끝값 포함) 또는 범위가 무제한인 경우 설정되지 않습니다.
endIndex
integer
범위의 끝 (해당 값 제외) 또는 제한되지 않은 경우 설정되지 않습니다.
