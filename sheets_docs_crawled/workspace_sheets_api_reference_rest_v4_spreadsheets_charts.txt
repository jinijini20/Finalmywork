object (DataLabel [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#DataLabel])
누적 차트에 도메인 축을 따라 각 값의 모든 누적 값의 총합계를 더한 데이터 라벨을 추가로 표시할지 여부를 제어합니다. 이러한 데이터 라벨은 chartType [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#BasicChartSpec.FIELDS.chart_type]이 AREA [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#BasicChartType], BAR [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#BasicChartType], COLUMN [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#BasicChartType], COMBO [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#BasicChartType] 또는 STEPPED_AREA [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#BasicChartType] 중 하나이고 stackedType [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#BasicChartSpec.FIELDS.stacked_type]이 STACKED [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#BasicChartStackedType] 또는 PERCENT_STACKED [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#BasicChartStackedType]인 경우에만 설정할 수 있습니다. 또한 COMBO [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#BasicChartType]의 경우 쌓을 수 있는 계열 type [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#BasicChartSeries.FIELDS.type] 유형이 하나뿐이거나 한 유형에 다른 유형보다 계열이 더 많고 다른 유형에는 계열이 하나만 있는 경우에만 지원됩니다. 예를 들어 차트에 누적 막대 계열이 2개 있고 영역 계열이 1개 있는 경우 총 데이터 라벨이 지원됩니다. 막대 계열이 3개이고 영역 계열이 2개인 경우 총 데이터 라벨은 허용되지 않습니다. totalDataLabel [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#BasicChartSpec.FIELDS.total_data_label]에서는 CUSTOM [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#DataLabelType] 또는 placement [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#DataLabel.FIELDS.placement]를 설정할 수 없습니다.

기준 값이 키 값의 음수 변화를 나타내는 경우 사용되는 색상입니다. 이 필드는 선택사항입니다. negativeColor [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#BaselineValueFormat.FIELDS.negative_color]도 설정된 경우 이 필드가 우선 적용됩니다.
ComparisonType
키 값과 기준 값의 비교 유형입니다.
열거형
COMPARISON_TYPE_UNDEFINED 기본값이며 사용할 수 없습니다.
ABSOLUTE_DIFFERENCE 키와 기준 값의 절대차를 사용합니다.
PERCENTAGE_DIFFERENCE 키와 기준 값 간의 비율 차이를 사용합니다.
ChartNumberFormatSource
차트 속성의 숫자 형식 지정 소스 옵션입니다.
열거형
CHART_NUMBER_FORMAT_SOURCE_UNDEFINED 기본값이며 사용할 수 없습니다.
FROM_DATA 데이터에서 숫자 형식을 상속합니다.
CUSTOM ChartCustomNumberFormatOptions [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets/charts?hl=ko#ChartCustomNumberFormatOptions]에 지정된 대로 맞춤 서식을 적용합니다.
ChartCustomNumberFormatOptions
차트 속성의 맞춤 숫자 형식 지정 옵션입니다.
JSON 표현
{
  "prefix": string,
  "suffix": string
}
필드
prefix
string
차트 속성에 추가할 맞춤 접두어입니다. 이 필드는 선택사항입니다.
suffix
string
차트 속성에 추가할 맞춤 접미사입니다. 이 필드는 선택사항입니다.
ChartHiddenDimensionStrategy
차트에서 숨겨진 소스 행을 처리하는 방법을 결정합니다. 숨겨진 행에는 수동으로 숨겨진 행과 필터로 숨겨진 행이 모두 포함됩니다.
열거형
CHART_HIDDEN_DIMENSION_STRATEGY_UNSPECIFIED 기본값이며 사용할 수 없습니다.
SKIP_HIDDEN_ROWS_AND_COLUMNS 차트는 숨겨진 행과 열을 건너뜁니다.
SKIP_HIDDEN_ROWS 차트는 숨겨진 행만 건너뜁니다.
SKIP_HIDDEN_COLUMNS 차트는 숨겨진 열만 건너뜁니다.
SHOW_ALL 차트는 숨겨진 행이나 열을 건너뛰지 않습니다.
