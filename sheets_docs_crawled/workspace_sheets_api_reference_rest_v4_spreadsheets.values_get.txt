승인 범위
스프레드시트에서 값 범위를 반환합니다. 호출자는 스프레드시트 ID와 범위를 지정해야 합니다.
HTTP 요청
GET https://sheets.googleapis.com/v4/spreadsheets/{spreadsheetId}/values/{range}
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
경로 매개변수
매개변수
spreadsheetId
string
데이터를 가져올 스프레드시트의 ID입니다.
range
string
값을 검색할 범위의 A1 표기법 또는 R1C1 표기법 [https://developers.google.com/workspace/sheets/api/guides/concepts?hl=ko#cell]입니다.
쿼리 매개변수
매개변수
majorDimension
enum (Dimension [https://developers.google.com/workspace/sheets/api/reference/rest/v4/Dimension?hl=ko])
결과에서 사용해야 하는 주요 측정기준입니다.
예를 들어 Sheet1의 스프레드시트 데이터가 A1=1,B1=2,A2=3,B2=4이면 range=Sheet1!A1:B2?majorDimension=ROWS를 요청하면 [[1,2],[3,4]]가 반환되고 range=Sheet1!A1:B2?majorDimension=COLUMNS를 요청하면 [[1,3],[2,4]]가 반환됩니다.
valueRenderOption
enum (ValueRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko])
값이 출력에 표시되는 방식입니다. 기본 렌더링 옵션은 FORMATTED_VALUE [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko]입니다.
dateTimeRenderOption
enum (DateTimeRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DateTimeRenderOption?hl=ko])
날짜, 시간, 기간이 출력에 표시되는 방식 valueRenderOption [https://developers.google.com/workspace/sheets/api/reference/rest/v4/spreadsheets.values/get?hl=ko#body.QUERY_PARAMETERS.value_render_option]가 FORMATTED_VALUE [https://developers.google.com/workspace/sheets/api/reference/rest/v4/ValueRenderOption?hl=ko]이면 무시됩니다. 기본 dateTime 렌더링 옵션은 SERIAL_NUMBER [https://developers.google.com/workspace/sheets/api/reference/rest/v4/DateTimeRenderOption?hl=ko]입니다.
